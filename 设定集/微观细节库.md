## 添加新词条的说明

添加一条新的衣物时，需要在右侧加入该服装对应的文化背景内的原生语言译文。
建议一并补全：`材质/版型`、`季节版本`、`常见搭配`、`适用场景`。

## 标签规范

- 标签模板：`文化=...` `风格=...` `时代=...` `功能=...` `气质=...`
- 最低要求：每条至少填写 `文化` + `风格` 两项。
- 建议写法：
  - `文化`：日式/中式/欧式/古罗马/古希腊 + 细分（如现代校园、唐风、维多利亚）。
  - `风格`：学院/甜系/哥特/礼仪/军务/通勤/仪式感等。
  - `时代`：古典/近代/现代/架空。
  - `功能`：常服/制服/礼服/工装/战斗/仪式。
  - `气质`：清冷/华丽/克制/张扬/庄严/机能。


## 服装类

### 服装组件字段（新增）

- 每条服装词条建议至少包含：
  - `组成成分`：拆成3-6个部件，每个部件写形态 + 功能 + 可见细节。
  - `常见材质`：棉/麻/毛/丝/皮革/化纤/混纺等。
  - `微观观察点`：缝线、褶量、磨损、起球、反光、声响、活动时形变。

### 宅系服装

- 水手服（セーラー服）
    - 标签：`文化=日式-现代校园` `风格=学院` `时代=现代` `功能=制服` `气质=清爽`
    - 二次元常见萌属性之一。
    - 组成成分：
      - `水手领`：后领片较大，覆盖肩背，边缘常有对比色滚条。
      - `前胸挡片`：与领口连成视觉三角区，决定制服“脸部框线”。
      - `前胸系带`：リボン/ネクタイ两类，结体大小决定幼态或成熟感。
      - `上衣主体`：短身或常规长度，袖口常见双线条呼应领边。
      - `百褶裙`：褶宽与褶密决定动态摆幅，走动时有规律开合。
    - 常见材质：棉涤混纺、聚酯纤维、呢料（冬装）。
    - 季节版型：春夏短袖、秋冬长袖。
    - 常见变体：不良系水手服、ブレザー混搭、セパレート改造款。
    - 微观观察点：领边压线、褶线锋利度、系带松紧、裙摆回弹、鞋跟回声。

- JK制服（女子高生制服）
    - 标签：`文化=日式-现代校园` `风格=学院/都市` `时代=现代` `功能=制服` `气质=利落`
    - 组成成分：
      - `衬衫`：标准翻领，白色最常见，领角硬挺度影响整体干净感。
      - `西装外套`：肩线、驳头、扣位构成“校规感”；胸章通常在左胸。
      - `针织层`：背心或开衫，增加层次并用于季节保温。
      - `下装`：百褶裙或西裤，长度与褶量体现校规与个人风格。
      - `领部件`：领带/领结颜色常用于年级或校别区分。
      - `鞋袜`：乐福鞋 + 中筒袜/过膝袜，决定收束感与节奏感。
    - 常见材质：棉、羊毛混纺、聚酯纤维、针织面料。
    - 校规变化点：裙长、袜长、外套扣法、领结颜色。
    - 常见穿搭分层：衬衫 + 针织背心（ニットベスト）+ 开衫（カーディガン）。
    - 微观观察点：外套肩线是否塌陷、胸章磨损、领带结点偏移、袜口压痕。

- Lolita（ロリータ・ファッション）
    - 标签：`文化=日式-次文化` `风格=甜系/古典/哥特` `时代=现代` `功能=常服/活动` `气质=华丽`
    - 组成成分：
      - `主裙体`：JSK/OP为核心，腰线与裙摆体积共同决定轮廓。
      - `衬裙`：提供体积支撑，硬挺度影响钟形或A字形态。
      - `内搭上衣`：高领、泡泡袖或蕾丝领常用于增强层次。
      - `头饰`：KC、bonnet、发带决定风格识别度。
      - `腿部与鞋履`：花纹袜 + 玛丽珍/厚底鞋，延续整体主题。
    - 常见材质：棉、提花、雪纺、蕾丝、天鹅绒。
    - 轮廓核心：通过衬裙（petticoat/crinoline）做出钟形或A字裙摆。
    - 常见单品：JSK（ジャンパースカート）/ OP（ワンピース），裙长多在膝部附近。
    - 主流子风格：Sweet、Classic、Gothic。
    - 细节关键词：蕾丝层次、胸前蝴蝶结、头饰（ボンネット/リボン）。
    - 微观观察点：蕾丝层级、裙摆波浪边、胸前蝴蝶结对称性、鞋面擦痕。

- 女仆装（メイド服 / maid outfit）
    - 标签：`文化=欧式-维多利亚衍生` `风格=服务角色扮演` `时代=近代/现代再演绎` `功能=制服/表演` `气质=端正或戏剧化`
    - 组成成分：
      - `连衣裙主体`：深色底布为主，领口与袖口常有荷叶边或花边。
      - `围裙`：白色高对比前片，腰带系法可传达角色习惯。
      - `头饰`：荷叶边发箍或头巾，强化职业角色识别。
      - `袖部与下摆`：泡泡袖或长袖版本决定甜系/端正走向。
      - `袜鞋组合`：常配长袜与低跟鞋，强调礼仪与整洁。
    - 常见材质：棉、涤纶、府绸、蕾丝辅料。
    - 常见结构：深色连衣裙 + 白围裙 + 头饰（カチューシャ/フリルヘッドドレス）。
    - 语义分化：维多利亚风偏端正，法式女仆风偏戏剧化。
    - 场景标签：女仆咖啡店（メイド喫茶）通常搭配固定问候语与角色扮演服务。
    - 微观观察点：围裙褶皱、腰带结尾长短、袖口白边、鞋面抛光程度。

- 死库水（スクール水着）
    - 标签：`文化=日式-校园体育` `风格=运动/制服` `时代=现代` `功能=泳装` `气质=功能性`
    - 组成成分：
      - `连体主身`：一体成型包覆躯干，强调贴身和阻力控制。
      - `肩带`：宽肩带更稳定，细肩带更强调外观轻量化。
      - `胸前名牌区`：学校或姓名标识位，常见白色名牌布块。
      - `腿根开口`：开口高低决定活动自由度与视觉风格。
      - `内衬层`：防透与贴肤舒适层，湿水后表现明显。
    - 常见材质：尼龙、涤纶、弹性纤维（氨纶）。
    - 全称缩写来自 school mizugi（スクール水着）。
    - 常见为贴身连体款，颜色多为深蓝/黑，面料多见尼龙 + 涤纶。
    - 版本迭代可写：旧型/新型/竞泳型/スパッツ型。
    - 微观观察点：湿水后深浅变化、肩带压痕、名牌边缘卷曲、干湿交界线。

### 工作服装

- 作务衣（作務衣 / さむえ / samue）
    - 标签：`文化=日式-寺务/工务` `风格=朴素` `时代=传统延续` `功能=工装` `气质=克制`
    - 组成成分：
      - `上衣`：交叠前襟，以系带固定，活动时不易掀开。
      - `下装`：宽松长裤，腰部多为抽绳或松紧结构。
      - `袖口`：可卷起处理，便于劳动和清洗。
      - `口袋`：简洁实用，常位于上衣前片或裤侧。
    - 常见材质：棉、麻、竹纤维混纺。
    - 日本僧侣与寺务劳动服出身，后扩展到园艺、餐饮与旅馆工装。
    - 常见面料：棉、麻；版型宽松，便于长时活动。
    - 微观观察点：系带磨毛、膝部褶皱、袖口水渍、布面旧化程度。

- 法被（法被 / はっぴ / happi）
    - 标签：`文化=日式-祭典` `风格=民俗/应援` `时代=传统延续` `功能=活动服` `气质=热闹`
    - 组成成分：
      - `前襟`：直开襟，便于快速穿脱和叠穿。
      - `筒袖`：短而宽，动作幅度大时不束缚手臂。
      - `背纹`：大面积文字或家纹，是团队识别核心。
      - `腰带`：简化固定和调节松紧，便于行进或舞动。
    - 常见材质：棉布、斜纹布、涤棉混纺。
    - 传统筒袖短外衣，常在祭典、商铺活动、应援场景使用。
    - 识别点：前襟与背部印字（店名、组名、祭典名）。
    - 微观观察点：印字开裂、领口汗渍、腰带打结方向、袖口飞边。

- 白衣（白衣 / はくい / ラボコート）
    - 标签：`文化=现代通用` `风格=专业` `时代=现代` `功能=工装` `气质=理性`
    - 组成成分：
      - `翻领`：驳领或平领，决定职业观感（医疗/实验）。
      - `门襟`：纽扣或暗扣结构，开合速度影响工作效率。
      - `功能口袋`：胸袋与下摆袋用于放笔、手套、记录本。
      - `袖口`：直口或收口，影响防污染和操作便利。
      - `后开衩`：提升步幅和坐姿舒适度。
    - 常见材质：棉涤混纺、防泼液面料、防静电纤维（实验室）。
    - 医疗与实验场景高频工装，强调“防污、可高温清洗、身份识别”。
    - 微观观察点：胸袋笔渍、袖口消毒剂泛白、下摆摩擦皱、暗扣松动。

- 连体工装（つなぎ / tsunagi）
    - 标签：`文化=现代通用` `风格=机能` `时代=现代` `功能=工装` `气质=硬朗`
    - 组成成分：
      - `连体主身`：上衣与裤身一体，减少勾挂风险。
      - `前中拉链`：从领口延伸到裆部，便于快速穿脱。
      - `工具口袋`：胸、臀、腿部多口袋分工明确。
      - `收口结构`：袖口/裤脚可收紧，避免卷入设备。
      - `膝部加固`：双层布或补丁提高耐磨性。
    - 常见材质：帆布、斜纹棉、阻燃面料、耐油污涂层布。
    - 机械维修、仓储、片场布景常见；多口袋、耐磨、便于收纳工具。
    - 微观观察点：拉链齿磨损、膝部油渍层、口袋边缘起毛、反光条刮痕。

### 常服

- 连帽卫衣（フーディー / hoodie）
    - 标签：`文化=现代通用` `风格=街头/休闲` `时代=现代` `功能=常服` `气质=松弛`
    - 组成成分：
      - `帽体`：双层或单层帽，决定垂坠和保暖性。
      - `帽绳`：金属头或塑封头，体现做工等级。
      - `前袋`：kangaroo pocket常用于暖手和临时收纳。
      - `袖口下摆罗纹`：收束轮廓并控制松紧。
    - 常见材质：棉、抓绒、毛圈布、涤棉混纺。
    - 轮廓关键词：落肩、帽绳、前袋（kangaroo pocket）。
    - 微观观察点：罗纹回弹、帽绳磨损、前袋边线拉扯、起球密度。

- 开衫（カーディガン / cardigan）
    - 标签：`文化=欧式起源-全球化` `风格=学院/通勤` `时代=近代至现代` `功能=常服` `气质=温和`
    - 组成成分：
      - `门襟`：纽扣排决定开合方式和视觉纵线。
      - `领型`：V领/圆领影响脸部比例和内搭露出面积。
      - `袖型`：插肩袖或常规袖影响肩线柔和度。
      - `罗纹边`：领口、袖口、下摆统一收束。
    - 常见材质：羊毛、棉纱、腈纶、羊绒混纺。
    - 叠穿友好：衬衫外层、制服外层、吊带外层。
    - 微观观察点：扣眼松紧、肘部起鼓、袖口起球、肩线塌陷。

- 针织背心（ニットベスト / knit vest）
    - 标签：`文化=欧式起源-全球化` `风格=学院` `时代=现代` `功能=常服/制服内搭` `气质=文雅`
    - 组成成分：
      - `领口`：V领最常见，露出领带结和衬衫领尖。
      - `肩部`：无袖开口尺寸决定手臂活动和层次露出。
      - `下摆罗纹`：控制腰部收束，防止上翻。
      - `纹理`：平针、麻花、提花影响视觉密度。
    - 常见材质：棉纱、羊毛、腈纶混纺。
    - 常用于“学院感”造型，配衬衫与领带层次明确。
    - 微观观察点：领口卷边、肩口磨毛、织纹拉伸、内搭透影。

- 百褶裙（プリーツスカート / pleated skirt）
    - 标签：`文化=现代通用` `风格=学院/通勤` `时代=现代` `功能=常服/制服` `气质=轻盈`
    - 组成成分：
      - `腰头`：硬衬腰头决定贴合和稳定性。
      - `褶裥区`：刀褶/箱褶不同，动态表现差异明显。
      - `裙长`：膝上、及膝、过膝对应不同礼仪和气质。
      - `内衬`：提升舒适度并减少透感。
    - 常见材质：毛涤混纺、聚酯纤维、棉质斜纹。
    - 静态看褶线密度，动态看摆幅与回弹。
    - 微观观察点：褶线保型度、腰扣磨损、坐压折痕、下摆飞线。

- 直筒牛仔裤（ストレートデニム / straight denim）
    - 标签：`文化=现代通用` `风格=工装衍生/休闲` `时代=现代` `功能=常服` `气质=中性`
    - 组成成分：
      - `裤腰`：腰头 + 皮牌 + 纽扣门襟，决定品牌识别度。
      - `裆部结构`：前后片拼接影响活动舒适度。
      - `口袋系统`：前插袋、零钱袋、后贴袋兼具功能与造型。
      - `裤管`：直筒线条从大腿到脚口基本一致。
      - `铆钉和车线`：高受力部位的耐久关键。
    - 常见材质：丹宁布（棉为主）、弹力牛仔（加氨纶）。
    - 写作重点：洗水、磨白、膝部折痕、裤脚堆积。
    - 微观观察点：猫须纹、蜂窝褶、口袋边磨损、脚口毛边。

- 切斯特大衣（チェスターコート / chester coat）
    - 标签：`文化=欧式-近代男装衍生` `风格=都市/正式` `时代=近代至现代` `功能=外套` `气质=克制`
    - 组成成分：
      - `驳领`：缺角驳领最常见，决定正式度。
      - `门襟`：单排或双排扣，控制胸前体量。
      - `肩部结构`：垫肩强弱影响气场和轮廓硬度。
      - `口袋`：插袋或盖袋，兼顾线条与实用。
      - `后开衩`：单衩/双衩决定步幅与动态垂感。
    - 常见材质：羊毛、羊绒混纺、粗花呢。
    - 冬季通勤核心单品，肩线和翻领决定气场。
    - 微观观察点：呢面起毛、袖口抛光磨亮、纽扣松紧、雨雪吸水痕。

### 配饰

- 领结（リボン）
    - 标签：`文化=日式-制服` `风格=学院/甜系` `时代=现代` `功能=配饰` `气质=可爱`
    - 制服系核心识别件，结体大小影响“幼态/成熟”观感。

- 领带（ネクタイ）
    - 标签：`文化=欧式-正装` `风格=正式/学院` `时代=近代至现代` `功能=配饰` `气质=理性`
    - 常见打法：平结、温莎结；松紧度可映射角色状态。

- 过膝袜（ニーハイソックス）
    - 标签：`文化=现代通用` `风格=学院/二次元` `时代=现代` `功能=配饰` `气质=活泼`
    - 与裙装搭配时，露肤比例会显著改变角色气质。

- 松垮袜（ルーズソックス / loose socks）
    - 标签：`文化=日式-街头校园` `风格=复古校园` `时代=现代` `功能=配饰` `气质=张扬`
    - 90年代至00年代初街头校园风高频视觉符号。

- 乐福鞋（ローファー / penny loafers）
    - 标签：`文化=欧式起源-校园化` `风格=学院/通勤` `时代=近代至现代` `功能=鞋履` `气质=稳重`
    - 制服搭配中常见，皮面磨损和鞋跟声是重要听觉细节。

- 发箍（カチューシャ）
    - 标签：`文化=现代通用` `风格=甜系/古典` `时代=现代` `功能=配饰` `气质=精致`
    - 视觉重心上移的快捷配件，常用于“角色识别点固定”。

- 颈圈（チョーカー / choker）
    - 标签：`文化=欧式起源-现代次文化` `风格=哥特/街头` `时代=现代` `功能=配饰` `气质=锐利`
    - 贴颈配饰，金属件、蕾丝、皮革会传达不同角色气质。


### 中式服装

- 汉服（汉服 / Hanfu）
    - 标签：`文化=中式-汉文化` `风格=礼制/古典` `时代=古典与现代复兴` `功能=礼服/常服` `气质=端雅`
    - 组成成分：
      - `内层中衣`：贴身基础层，吸汗并保护外层衣料。
      - `上衣`：交领右衽为核心，系带隐藏在前襟内外层。
      - `下装`：裙或裤按朝代与场景变化。
      - `腰封/系带`：固定层次并调整视觉比例。
      - `披帛/外搭`：提升仪式感和动态飘带效果。
    - 常见材质：丝、纱、棉、麻、锦、绫罗。
    - 泛指汉族传统服饰体系，核心特征是交领右衽、系带结构、层次叠穿。
    - 微观观察点：领缘包边、系带收口、衣摆垂坠线、纹样织密度。

- 襦裙（襦裙 / Ruqun）
    - 标签：`文化=中式-汉文化` `风格=古典` `时代=先秦至明清多期演化` `功能=常服/礼服` `气质=柔和`
    - 组成成分：
      - `襦`：短上衣，交领或对襟，袖形决定时代感。
      - `裙`：高腰或齐腰长裙，重点在腰线位置与褶量。
      - `系带`：长带绕系，结位可在前胸、腰侧或背后。
      - `披帛`：可选配，增强动态和层次。
    - 常见材质：丝绸、棉麻、纱罗。
    - 上襦下裙的两件式结构，高腰/齐胸/齐腰等形制均可用于时代区分。
    - 微观观察点：胸前系带压痕、裙褶展开节奏、袖口压线、下摆拖地痕。

- 马面裙（马面裙 / Mamianqun）
    - 标签：`文化=中式-明清` `风格=礼制/华丽` `时代=宋元原型-明清成熟` `功能=礼服/常服` `气质=端庄`
    - 组成成分：
      - `裙门`：前后平面区（马面）形成视觉纵向中轴。
      - `侧褶`：两侧密褶提供步行动态和容量。
      - `腰头`：多层缝制承重，保证垂坠稳定。
      - `系结`：长带或扣件固定，决定穿着紧致度。
    - 常见材质：织金锦、妆花、缎、棉混纺日常料。
    - 识别点：前后“马面”平面区 + 两侧褶裥，行走时开合节奏明显。
    - 微观观察点：裙门纹样对称、侧褶回弹、腰头厚度、裙脚磨边。

- 圆领袍（圆领袍 / Yuanlingpao）
    - 标签：`文化=中式-官民通用` `风格=礼仪/官式` `时代=隋唐至明` `功能=礼服/常服` `气质=庄重`
    - 组成成分：
      - `圆领`：贴颈环形领口，强调中轴与秩序感。
      - `袍身`：直身长袍结构，前后片连贯。
      - `袖部`：宽袖或窄袖依时代与身份变化。
      - `腰带`：控制袍身松紧并挂佩饰。
      - `纹章区`：补子、纹样用于身份标识。
    - 常见材质：绫、罗、绸、纱、呢料（冬）。
    - 常见于官服和士人服饰系统，可通过补子、色阶、纹样体现阶序。
    - 微观观察点：领圈挺度、袖口翻折、腰带垂尾、补子边框针脚。

- 褙子（褙子 / Beizi）
    - 标签：`文化=中式-宋明` `风格=文人/日常` `时代=宋起` `功能=外搭` `气质=清雅`
    - 组成成分：
      - `前襟`：对襟直开，便于内搭展示。
      - `衣身`：长直轮廓，强调纵向线条。
      - `开衩`：两侧开衩提升步幅与垂坠流动。
      - `袖型`：直袖为主，利于文人日常动作。
    - 常见材质：轻绢、罗纱、棉麻。
    - 直身、对襟、侧开衩特征明显，适合写“行走时下摆分线”。
    - 微观观察点：开衩摆动、前襟平整度、下摆折边、轻纱透层。

- 旗袍（旗袍 / Qipao / Cheongsam）
    - 标签：`文化=中式-近现代都市` `风格=海派/礼仪` `时代=民国至现代` `功能=礼服/制服` `气质=利落`
    - 组成成分：
      - `立领`：包颈高度决定端庄度和限制感。
      - `门襟`：右衽开襟，盘扣是视觉焦点。
      - `胸腰省道`：塑造贴体曲线与身体线条。
      - `侧开衩`：控制步幅并形成动态“开合线”。
      - `下摆`：长度与开衩终点共同决定风格边界。
    - 常见材质：真丝、织锦、缎面、棉麻混纺日常款。
    - 识别点：立领、盘扣、右衽开襟、侧开衩；可用开衩高度区分保守与前卫。
    - 微观观察点：盘扣密度、领口压痕、臀腿处拉力纹、下摆转身轨迹。

### 欧式服装

- 束身衣（Corset）
    - 标签：`文化=欧式-近代` `风格=宫廷/复古` `时代=16-19世纪核心` `功能=塑形内衣` `气质=克制`
    - 组成成分：
      - `前中骨架`：金属或硬骨支撑正面平直。
      - `侧后骨条`：均匀分布以维持收腰曲线。
      - `后背系带`：松紧调节核心，决定塑形力度。
      - `胸口边缘`：低胸或高胸版本对应不同礼仪语境。
    - 常见材质：棉斜纹、缎面、皮革、钢骨/塑骨。
    - 核心机制：收腰塑形 + 支撑上身结构，常与裙撑体系联动。
    - 微观观察点：系带压痕、骨条顶点磨损、呼吸节奏变化、坐姿受限感。

- 燕尾服（Tailcoat）
    - 标签：`文化=欧式-正装体系` `风格=礼仪` `时代=近代至现代仪式场合` `功能=礼服` `气质=庄严`
    - 组成成分：
      - `前片`：前摆短切，露出马甲和礼服衬衣。
      - `后摆`：双尾分叉，是燕尾服核心识别结构。
      - `驳领`：常见丝缎面，形成礼服光泽对比。
      - `肩袖结构`：高挺肩线与窄袖强调仪式感。
    - 常见材质：羊毛精纺、缎面驳领、丝衬里。
    - 常见于白领结礼仪场景，后摆分叉是识别点。
    - 微观观察点：后摆甩动弧度、驳领反光、袖口露白长度、纽扣光泽。

- 马甲（Waistcoat）
    - 标签：`文化=欧式-三件套` `风格=绅士/通勤` `时代=近代至现代` `功能=中层` `气质=精致`
    - 组成成分：
      - `前身`：面料与西装一致或对比，用于构建层次中心。
      - `后背`：常用滑面衬里，减少与外套摩擦。
      - `门襟`：单排扣为主，扣数影响正式程度。
      - `后腰调节带`：微调腰围贴合度。
    - 常见材质：羊毛混纺、提花、棉、丝质后背。
    - 与衬衫、西装叠穿时可塑造“层次和秩序感”。
    - 微观观察点：门襟平整度、最后一颗扣法、后背调节带松紧、口袋怀表链。

- 斗篷（Cloak / Cape）
    - 标签：`文化=欧式-历史延续` `风格=戏剧化/旅行` `时代=古典至现代再演绎` `功能=外披` `气质=神秘`
    - 组成成分：
      - `披身`：半圆或全圆裁片，决定包覆范围与垂感。
      - `领部`：立领或翻领，构建脸部框线和气场。
      - `闭合件`：系带、搭扣、胸针用于前胸固定。
      - `兜帽`：可选结构，增强叙事功能性。
    - 常见材质：呢料、羊毛、天鹅绒、皮革。
    - 重点可写：披挂动作、领口固定件、行走时后摆拖曳。
    - 微观观察点：披摆扫地声、边缘毛绒磨损、扣件冷光、雨水流痕。

- 双排扣大衣（Double-breasted coat）
    - 标签：`文化=欧式-军政衍生` `风格=正式/军感` `时代=近代至现代` `功能=外套` `气质=硬朗`
    - 组成成分：
      - `双排门襟`：前片重叠提供防风与体量感。
      - `驳领`：宽驳领强化肩胸区域视觉权重。
      - `肩部`：肩章或硬挺肩线可加入军感信息。
      - `腰线`：收腰或直身影响力量感和机动性。
      - `袖口`：扣袢或装饰扣用于风格强调。
    - 常见材质：粗花呢、羊毛呢、防泼水混纺。
    - 特征是前襟重叠与双列纽扣，可通过扣位表现人物控制欲或松弛度。
    - 微观观察点：纽扣磨亮区、领口折痕、肩部压线、下摆吸水重量。

### 古罗马式服装

- 托加（Toga）
    - 标签：`文化=古罗马` `风格=公民礼仪` `时代=古典` `功能=仪式/公共场合` `气质=庄重`
    - 组成成分：
      - `主披布`：大幅半椭圆布料，披裹路径复杂。
      - `肩部挂点`：以身体重力和折叠维持稳定，无现代扣件。
      - `前胸褶层`：形成身份感强的层叠体积。
      - `下摆拖量`：长度决定行走节奏和仪式感。
    - 常见材质：羊毛（古典时期常见）。
    - 男性公民礼服属性强，披裹复杂，不适合高强度行动场景。
    - 微观观察点：褶层厚度、披法松紧、转身时摆尾滞后、肩部滑落风险。

- 图尼卡（Tunica）
    - 标签：`文化=古罗马` `风格=基础服装` `时代=古典` `功能=内外层皆可` `气质=实用`
    - 组成成分：
      - `衣身`：T形基础版型，前后片简洁。
      - `领口`：圆领或开口领，便于套穿。
      - `腰带`：收束衣身并调节长度。
      - `袖部`：无袖至短袖均有，依职业与场景调整。
    - 常见材质：羊毛、亚麻。
    - 男女皆可穿，长度和装饰常用于区分阶层或场景。
    - 微观观察点：腰带结位、下摆高低差、布面粗细、汗渍色差。

- 斯托拉（Stola）
    - 标签：`文化=古罗马` `风格=妇女礼服` `时代=古典` `功能=礼仪/身份` `气质=端庄`
    - 组成成分：
      - `长袍主身`：直筒长裙结构，长度多及足背。
      - `肩部固定`：肩部别针或缝位固定上沿。
      - `腰线收束`：腰带形成上提褶层。
      - `外披层`：常与帕拉组合增强外出礼仪性。
    - 常见材质：羊毛、亚麻、细织布。
    - 常作已婚女性身份象征，可与外披帕拉（palla）组合书写。
    - 微观观察点：肩针位置、腰部堆褶、脚边拖量、外披覆盖方式。

- 帕拉（Palla）
    - 标签：`文化=古罗马` `风格=外披` `时代=古典` `功能=出行披覆` `气质=含蓄`
    - 组成成分：
      - `矩形披布`：单片大布，依折法形成不同遮蔽程度。
      - `肩颈绕法`：可单肩披挂或双肩包覆。
      - `头部覆法`：公共场景可上提覆盖头部。
    - 常见材质：羊毛、亚麻。
    - 矩形披布，可罩肩、绕体、覆头，适合表现“公共空间中的自我保护”。
    - 微观观察点：布角下坠点、手臂掀披动作、风中边缘翻卷、遮面比例。

- 卡利加（Caligae）
    - 标签：`文化=古罗马` `风格=军务` `时代=古典` `功能=军靴` `气质=硬派`
    - 组成成分：
      - `鞋底`：厚底结构承重，适应长距离行军。
      - `绑带`：皮带交叉缠绕小腿，稳定脚踝。
      - `鞋面开孔`：增强透气并减轻重量。
      - `底钉`：提升抓地和耐磨。
    - 常见材质：皮革、金属钉。
    - 罗马军靴代表样式，绑带结构和鞋底钉可强化行军质感。
    - 微观观察点：绑带勒痕、底钉敲击声、泥尘堆积、鞋底磨平区。

### 古希腊式服装

- 希顿（Chiton）
    - 标签：`文化=古希腊` `风格=垂坠` `时代=古典` `功能=基础服装` `气质=流动`
    - 组成成分：
      - `矩形布体`：围体后在肩部固定形成筒状。
      - `肩部别针`：fibula固定开口并控制垂坠。
      - `腰带`：提高腰线并制造上部蓬褶（kolpos）。
      - `下摆`：长度依身份与场景变化。
    - 常见材质：亚麻、羊毛。
    - 长方形布料通过肩部别针固定，男女皆可穿，常见多样系带法。
    - 微观观察点：肩针金属光、腰带上提褶、步行摆线、风压贴体变化。

- 佩普洛斯（Peplos）
    - 标签：`文化=古希腊` `风格=古典礼制` `时代=古风至古典期` `功能=女装` `气质=沉静`
    - 组成成分：
      - `外翻上缘`：顶部翻折形成apoptygma层。
      - `肩部固定`：双肩别针形成柱状轮廓。
      - `腰部束带`：控制翻折层与下摆比例。
      - `侧边处理`：开放或缝合决定活动自由度。
    - 常见材质：羊毛、亚麻。
    - 顶部翻折形成上层垂褶，可开侧缝或胸针固定。
    - 微观观察点：翻折层厚薄、别针位置对称性、腰线高低、下摆拖曳。

- 希马申（Himation）
    - 标签：`文化=古希腊` `风格=外披` `时代=古典` `功能=外层` `气质=克制`
    - 组成成分：
      - `大幅披布`：覆盖肩背和躯干，提供体面与保暖。
      - `环绕路径`：从左肩或腋下绕行，形成身份化披法。
      - `前胸收束`：通过手臂压布保持结构稳定。
    - 常见材质：羊毛、亚麻。
    - 外披矩形布，常套在希顿外层，披法可体现身份与情绪。
    - 微观观察点：布角落点、前胸层次厚度、行走时后摆摆幅、手臂压布习惯。

- 克拉米斯（Chlamys）
    - 标签：`文化=古希腊` `风格=军旅/青年` `时代=古典至希腊化` `功能=短斗篷` `气质=利落`
    - 组成成分：
      - `短披身`：长度较短，便于骑乘和快速行动。
      - `肩部别针`：通常在右肩固定，形成斜向披挂。
      - `边缘包缝`：防止披布散边和撕裂。
    - 常见材质：羊毛、厚织布。
    - 右肩别针固定的短披风，机动性高，常用于军务或狩猎场景。
    - 微观观察点：别针受力点、转身甩摆、边缘磨损、风中贴背形态。

- 凉鞋（σανδάλια / sandalia）
    - 标签：`文化=古希腊` `风格=日常` `时代=古典` `功能=鞋履` `气质=朴素`
    - 组成成分：
      - `鞋底`：平底结构，厚度决定耐磨与抓地。
      - `前掌带`：固定前脚掌，控制滑动。
      - `踝带`：缠绕脚踝与小腿，提供稳定支撑。
      - `扣结`：打结或扣件影响穿脱效率。
    - 常见材质：皮革、植物纤维、木质底（特定场景）。
    - 绑带路径可用于强化角色身份（平民实用/贵族装饰）。
    - 微观观察点：绑带压痕、脚背晒痕、鞋底磨耗图案、行走落地声。


## 属性类

### 视觉属性

- 发色（髪色 / hair color）
    - 标签：`文化=通用` `风格=视觉锚点`
    - 关键词模板：底色、挑染、明度、冷暖、反光。

- 瞳色（瞳色 / eye color）
    - 标签：`文化=通用` `风格=角色识别`
    - 关键词模板：主色、外圈深度、虹膜纹理、光点位置。

- 体态（姿勢 / posture）
    - 标签：`文化=通用` `风格=行为叙事`
    - 站姿重心、步幅、手部安放位置可替代对白说明。

### 行为属性

- 说话速度（話速 / speech rate）
    - 标签：`文化=通用` `风格=节奏控制`
    - 快语速常伴信息压缩，慢语速常伴权威或试探。

- 口癖（口癖 / verbal tic）
    - 标签：`文化=通用` `风格=角色语气`
    - 建议限定在1-2个，并允许随剧情阶段变化。

### 角色原型标签（用于速记，不等于定型）

- 傲娇（ツンデレ / tsundere）
    - 标签：`文化=日式-ACGN` `风格=关系推进`
    - 外冷内热，关系推进后逐步暴露柔软面。

- 辣妹系（ギャル / gyaru）
    - 标签：`文化=日式-街头` `风格=自我表达`
    - 重点是风格张力，不等于固定性格。

## 日常生活类

### 日式

- 玄关（玄関 / genkan）
    - 标签：`文化=日式` `风格=居住礼仪`
    - 居住空间入口缓冲区，核心动作是“脱鞋换鞋”。

- 鞋柜（下駄箱 / getabako）
    - 标签：`文化=日式-校园/住宅` `风格=日常管理`
    - 常位于玄关或学校入口，用于收纳外出鞋与室内鞋。

- 被炉（炬燵 / こたつ / kotatsu）
    - 标签：`文化=日式-家居` `风格=冬季生活`
    - 低桌 + 被褥 + 下置热源；现代多为电加热款。

- 便当（弁当 / bentō）
    - 标签：`文化=日式-餐食` `风格=通勤/校园`
    - 单人份盒餐，常见构成：主食 + 蛋白 + 腌渍/熟蔬。

- 交通IC卡（Suica / スイカ）
    - 标签：`文化=日式-城市交通` `风格=现代基础设施`
    - 2001年上线，可用于轨道交通和小额消费。

- 钱汤（銭湯 / sentō）
    - 标签：`文化=日式-公共生活` `风格=社区`
    - 公共浴场礼仪核心是“先洗后泡”。

- 扭蛋（ガシャポン / gashapon）
    - 标签：`文化=日式-消费文化` `风格=收藏/随机`
    - 叙事钩子：重复抽同款、交换凑齐系列。

- 拍贴（プリクラ / purikura）
    - 标签：`文化=日式-青年文化` `风格=社交记录`
    - 源自“Print Club”，1995年起流行。

### 古中式生活细节

- 茶肆（茶肆 / Chasi）
    - 标签：`文化=中式` `风格=社交/议事`
    - 场景细节：堂倌唱单、盖碗开合、桌面留痕。

- 香囊（香囊 / Xiangnang）
    - 标签：`文化=中式` `风格=贴身物件/礼赠`
    - 可用于“人物关系暗线”与“气味记忆锚点”。

- 簪钗（簪 / 钗 / Zan / Chai）
    - 标签：`文化=中式` `风格=发饰礼制`
    - 材质（木/玉/金属）可体现阶层、喜好和处境。

### 欧式生活细节

- 壁炉（Fireplace）
    - 标签：`文化=欧式` `风格=室内核心场景`
    - 细节钩子：炉火噼啪、煤灰、靠火站位形成权力距离。

- 下午茶（Afternoon tea）
    - 标签：`文化=欧式-英式传统` `风格=礼仪社交`
    - 细节：三层点心架、先奶后茶争议、银器反光。

- 舞会（Ball）
    - 标签：`文化=欧式-宫廷/上流` `风格=仪式社交`
    - 细节：舞序、请舞礼、手套材质与触感。

### 古罗马生活细节

- 公共浴场（Thermae）
    - 标签：`文化=古罗马` `风格=公共空间`
    - 可写流程：更衣区 -> 温热池 -> 热池 -> 冷池。

- 广场（Forum）
    - 标签：`文化=古罗马` `风格=政治/商业中心`
    - 叙事作用：公告、辩论、交易在同一场域叠加。

### 古希腊生活细节

- 市集广场（Agora）
    - 标签：`文化=古希腊` `风格=公共议事/交易`
    - 叙事作用：政治讨论与日常买卖共场并行。

- 会饮（Symposium）
    - 标签：`文化=古希腊` `风格=精英社交`
    - 细节钩子：斜倚姿势、诗歌辩论、器皿与酒水比例。
