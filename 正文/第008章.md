# 第008章 考场日与六小时窗口

考试当天，菲娜四点五十五就醒了。

闹钟还没响，她已经坐起身，先摸到枕边证件袋，再摸到昨晚写的流程卡。她把流程卡从头到尾默了一遍：`进场前检查 -> 笔试稳分 -> 等候席降心率 -> 实操先稳后快`。确认无误后，她才下床洗漱。

楼下厨房灯是亮的。蓝月已经把早餐摆好，牛奶温度正好，吐司切成她习惯的一口大小。两人都没说“加油”，像在执行一次再普通不过的日程。吃到一半，蓝月把一个小型通讯器放到她手边。

“今天我在考场外。你进场前可以给我发一次消息，出场后再发一次。中间不联系。”

“怕我慌？”

“怕你忘记补糖。”

菲娜低头笑了一下，把通讯器塞进口袋。临出门时，蓝月替她把领口抚平，动作很轻：“记住，你去考试不是去证明你比谁强，是去把你该拿的分拿回来。”

“收到。”

她背包下楼，天色刚亮，路口车流已经起来。她一边走一边在脑子里过最后几道定义题，走到第二个斑马线时，视线还黏在屏幕里的笔记截图上。下一秒，一阵急刹车声在她耳边炸开。

轮胎擦地的尖响几乎贴着她膝侧掠过去。

有人从后面猛地拽住她书包带，把她整个人往后提了半步。菲娜脚下一空，差点坐地上，手机直接飞出去砸在路边护栏上。

“你疯啦？走路看题？！”

蕾雅的声音在她耳边爆开，气得发抖。菲娜抬头才看清，刚才拽她的人就是蕾雅。车在前方两米处停住，司机降窗骂了两句又开走，路口信号灯刚好跳绿。

菲娜后背瞬间全是冷汗。她张了张嘴，嗓子发干：“我……”

“你什么你。”蕾雅把她手机捡起来，拍掉灰，“今天考的是魔法，不是比谁先把自己送走。”

菲娜接过手机，低声说：“谢谢。”

“别谢，给我活着进考场。”

两人并肩往前走，谁都没再说话。到校门口时，菲娜先给蓝月发了条简讯：`已到，刚刚差点出事故，人没事。`几秒后蓝月回：`先稳呼吸。进场前吃一片糖。`

她照做。甜味在舌尖化开时，心跳才慢慢落回可控范围。

考试楼外已经排起队。广播反复播规则：禁止携带未授权魔杖、禁止外部符文纸、禁止调用外部模型辅助。监考老师逐个检查证件和装备，节奏严得像军检。

笔试先开，题量比平时模拟稍大，但题型基本在预期里。菲娜按“先稳分后拉分”的策略做，先拿定义和基础推导，再回头啃两道高难题。交卷铃响时，她没有满分把握，但知道自己没崩。

离开笔试考场去实操等候区的那段走廊很短，菲娜却走得像过隧道。墙上贴着历年优秀考生名单，名字一排排整齐排列，她以前每次路过都只会看一眼就走，今天却忍不住多看了两秒。她不是想“上榜”，她只是突然很明确地意识到：自己正在跨过那道她想了很多年的门槛。

等候区入口有二次检查。监考老师逐个核对杖体编号，确认没有改装；再检查袖口和口袋，确认没有私带符文纸或外接设备。轮到菲娜时，老师把她的铝杖在检测环里过了一遍，检测环亮绿，她才被放行。

她坐下后，先按蓝月教的流程做“降噪”：看手背、看鞋尖、数呼吸。左侧的考生在小声背定义，右侧有人低头默念咒文，前排还有人一直抖腿，椅子咔咔响。她没去管这些声音，只把它们当背景噪声，逼自己把注意力钉在胸口起伏上。

真正让她紧起来的是实操等候席。几十名考生按编号坐成两排，每人手里都握着那根统一配发的铝杖。有人闭眼默咒，有人手抖到连杖都拿不稳，还有人低头疯狂翻最后一页小抄，被巡场老师当场收走。

等候席第三排忽然传来一阵骚动。一个男生把杖尖贴着掌心，偷偷夹着一张极小热敏纸，纸上印着复杂符文回路。旁边同学刚想提醒，巡场老师已经走到他面前。

“请把手摊开。”

男生僵住，还是把纸摊出来。纸面图样清晰得过头，角落还有命定色盘水印残影。老师没多说，直接举手示意主监考。几秒后，海伦本人从前排走过来，接过那张纸看了一眼。

全场安静到能听见呼吸。

海伦把纸翻过来，对全体考生平静宣布：“未经授权生成符文，且原封不动用于考试，按作弊处理。该考生本场成绩记零，取消后续实操资格。”

男生脸色刷白，嘴唇动了两下，最后什么都没说，被引导员带离等候席。骚动迅速平息，空气却更紧了。

菲娜握杖的手又湿了。她脑子里突然闪过一个念头：`如果我刚才也想偷一个“更稳模板”呢？`她立刻把这个念头掐断，深呼吸两次，盯住自己鞋尖前那条白线，强迫注意力回到流程卡。

“第27号，菲娜·希格墨伊德，准备。”

她站起来，腿有一点软。走到实操区中央时，灯光正好打在她正上方，周围观摩席坐着几位老师和后续考生。她先行礼，再报术式序列：

“申请展示：火系余火、风系线性气流、光系聚焦照明。”

第一项余火，她按昨天森林里的节奏走：先立边界，再增亮度，不追极限。火星稳稳浮在指定区，持续时间超过基准线。第二项风系线性气流，她把纸屑轨迹控在细长椭圆内，没有越界。第三项光系聚焦是她最担心的，她起手前明显停了一拍，听见自己心跳在耳膜里敲。

她想起蓝月那句“先稳，不追三环”，把输出压到安全档，光束在靶面中心稳稳点亮，没有抖，也没有头痛。

靶面光点稳定三秒后，观摩席后排有人低低“嘶”了一声。不是因为她打得多炫，而是因为她整套动作太干净：没有多余修正，没有临场硬拉，像把一条练过很多遍的线平平直直地送到终点。菲娜听见了那声，但没有回头。她知道此刻最危险的不是失误，而是“以为自己稳了”之后的松劲。

主监考记录员抬眼看她，点了下头：“完成。可退场。”

菲娜收杖那一刻，膝盖差点软掉。她走回等候区边缘，直到确认自己真的做完，才后知后觉地开始发抖。抖不是怕，是过载后的回弹。她坐下时，刚好看见诺亚被叫号上场。

诺亚上场前朝她这边看了一眼，眼神里是很短的一瞬错愕，像在说：`你今天怎么这么稳？`

菲娜冲他点了点头，没说话。

诺亚的实操依旧干净，甚至更快。但菲娜能看出他今天比平时保守，明显压住了“想秀一下”的冲动。她忽然有点想笑：看来那张被判零的热敏纸不只吓到了作弊的人，也把全场的冒进欲望一起压了下去。

等所有场次结束，成绩不会当场全公布，只会先发“通过/待复核”。菲娜领到回执卡，看到“通过”两个字时，整个人像突然失重，站在原地半天没动。蕾雅从后面扑过来抱她，几乎把她撞出门框。

“你过啦！我也过啦！诺亚也过啦！今天我们都活下来了！”

菲娜被她晃得头晕，笑着推她：“先放手，快窒息了。”

出考场第一件事，她掏出通讯器给蓝月发消息：`通过。人完好。`

十秒不到，蓝月回：`收到。原地等我三分钟。`

她抬头望向校门外，蓝月果然就站在马路对面。哥哥今天没穿显眼衣服，只是一件普通黑外套，但她还是一眼就认出来。那一瞬间，菲娜胸口压了一周的石头终于彻底落地。

她跑过去，差点又忘了看车，被蓝月一把拦住手腕：“慢点。”

“我过了。”

“我看到了。”

“你不夸我？”

蓝月抬手揉她头发：“做得很好，考生。”

菲娜笑着把回执卡塞进他手里。蓝月接卡的同时，另一只手上的终端屏幕亮起新通知：`命定色盘 v1.2 发布成功。`紧接着第二条：`暗月会启动倒计时：06:00:00。`

通知弹窗还在往上叠：`缘平台正在灰度更新`、`API请求峰值提升`、`二创区流量分流开始`。菲娜只看懂一半术语，但她看得懂蓝月的表情：考试这边的门刚关，他那边的战斗已经开火。

“你今天会回来吃晚饭吗？”她问。

“不一定。”蓝月目光还在终端上，“前两小时最关键，错过就要花两天补。”

菲娜点头，没再追问。她突然明白这六小时窗口和自己刚结束的考试有某种相似：都要求你在有限时间内把能做对的事全部做对，任何一次犹豫都会被放大成代价。

菲娜看见倒计时，笑意慢慢收住：“你现在就开打？”

“嗯，六小时窗口。”蓝月把终端收回口袋，“这段时间里，`缘`会很难受。”

她听懂了这话里的冷度，却没再劝。考试刚结束，战场已经切换。她只是问：“那我们先回家吗？”

“先回家。你先睡一觉。”

“你呢？”

“我上前线。”

菲娜本想说“别熬太狠”，话到嘴边变成了：“我睡醒给你做汤。”

蓝月侧头看她，笑了笑：“成交。”

两人并肩走出校门。背后还有家长和学生的喧哗声，前方街屏已经开始推送命定色盘v1.2更新海报。菲娜把手插进外套口袋，指尖碰到那张“通过”回执卡，终于允许自己在心里说一句：

`第一关，过了。`

她也知道，哥哥那边的第一关，才刚刚开始。

走到路口时，蕾雅和诺亚从后面追上来。蕾雅还在兴奋地挥回执卡，诺亚则难得没抱电脑，只把双手插在口袋里，表情像刚跑完一场长距离赛。

“晚上开黑庆祝？”蕾雅问。

“我先睡。”菲娜说，“醒了看情况。”

诺亚点头：“合理。今天谁都别再做高风险测试。”

蕾雅看了他一眼，笑出声：“你居然会说这句话。”

诺亚推了推眼镜，没反驳。菲娜看着他们，忽然觉得这一天终于有了结束感。她冲两人挥手告别，跟着蓝月上车，任由疲惫像潮水一样慢慢把自己淹没。
车门合上那一刻，她终于把肩膀彻底放松下来。
接下来，她只想先睡一场完整的觉。
然后再看新的战场。
一步一步来。
不抢跑。
喵。
走。
