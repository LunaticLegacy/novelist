## 添加新词条的说明

添加一条新的衣物时，需要在右侧加入该服装对应的文化背景内的原生语言译文。
建议一并补全：`材质/版型`、`季节版本`、`常见搭配`、`适用场景`。

## 标签规范

- 标签模板：`文化=...` `风格=...` `时代=...` `功能=...` `气质=...`
- 最低要求：每条至少填写 `文化` + `风格` 两项。
- 建议写法：
  - `文化`：日式/中式/欧式/古罗马/古希腊/斯拉夫式 + 细分（如现代校园、唐风、维多利亚）。
  - `风格`：学院/甜系/哥特/礼仪/军务/通勤/仪式感等。
  - `时代`：古典/近代/现代/架空。
  - `功能`：常服/制服/礼服/工装/战斗/仪式。
  - `气质`：清冷/华丽/克制/张扬/庄严/机能。

## 日语标注规则（新增）

- 纯片假名词：后接原始语言词形与语种标注。  
  例：`リボン / ribbon, English`、`カヌレ / canelé, French`
- 含日语汉字词：汉字后补假名读音。  
  例：`玄関(げんかん)`、`作務衣(さむえ)`、`口癖(くちぐせ)`
- 和製語（日本本土造词）仍可保留日语说明，并标注“和製語”。


## 服装类

### 服装组件字段

- 每条服装词条建议至少包含：
  - `组成成分`：拆成3-6个部件，每个部件写形态 + 功能 + 可见细节。
  - `常见材质`：棉/麻/毛/丝/皮革/化纤/混纺等。
  - `微观观察点`：缝线、褶量、磨损、起球、反光、声响、活动时形变。

### 宅系服装

- 水手服（セーラー服(せーらーふく) / sailor suit, English）
    - 标签：`文化=日式-现代校园` `风格=学院` `时代=现代` `功能=制服` `气质=清爽`
    - 二次元常见萌属性之一。
    - 组成成分：
      - `水手领`：后领片较大，覆盖肩背，边缘常有对比色滚条。
      - `前胸挡片`：与领口连成视觉三角区，决定制服“脸部框线”。
      - `前胸系带`：リボン / ribbon, English 与 ネクタイ / necktie, English 两类，结体大小决定幼态或成熟感。
      - `上衣主体`：短身或常规长度，袖口常见双线条呼应领边。
      - `百褶裙`：褶宽与褶密决定动态摆幅，走动时有规律开合。
    - 常见材质：棉涤混纺、聚酯纤维、呢料（冬装）。
    - 季节版型：春夏短袖、秋冬长袖。
    - 常见变体：不良系水手服、ブレザー / blazer, English 混搭、セパレート / separate, English 改造款。
    - 微观观察点：领边压线、褶线锋利度、系带松紧、裙摆回弹、鞋跟回声。

- JK制服（女子高生制服）
    - 标签：`文化=日式-现代校园` `风格=学院/都市` `时代=现代` `功能=制服` `气质=利落`
    - 组成成分：
      - `衬衫`：标准翻领，白色最常见，领角硬挺度影响整体干净感。
      - `西装外套`：肩线、驳头、扣位构成“校规感”；胸章通常在左胸。
      - `针织层`：背心或开衫，增加层次并用于季节保温。
      - `下装`：百褶裙或西裤，长度与褶量体现校规与个人风格。
      - `领部件`：领带/领结颜色常用于年级或校别区分。
      - `鞋袜`：乐福鞋 + 中筒袜/过膝袜，决定收束感与节奏感。
    - 常见材质：棉、羊毛混纺、聚酯纤维、针织面料。
    - 校规变化点：裙长、袜长、外套扣法、领结颜色。
    - 常见穿搭分层：衬衫 + 针织背心（ニットベスト / knit vest, English）+ 开衫（カーディガン / cardigan, English）。
    - 微观观察点：外套肩线是否塌陷、胸章磨损、领带结点偏移、袜口压痕。

- Lolita（ロリータ・ファッション / Lolita fashion, English）
    - 标签：`文化=日式-次文化` `风格=甜系/古典/哥特` `时代=现代` `功能=常服/活动` `气质=华丽`
    - 组成成分：
      - `主裙体`：JSK/OP为核心，腰线与裙摆体积共同决定轮廓。
      - `衬裙`：提供体积支撑，硬挺度影响钟形或A字形态。
      - `内搭上衣`：高领、泡泡袖或蕾丝领常用于增强层次。
      - `头饰`：KC、bonnet、发带决定风格识别度。
      - `腿部与鞋履`：花纹袜 + 玛丽珍/厚底鞋，延续整体主题。
    - 常见材质：棉、提花、雪纺、蕾丝、天鹅绒。
    - 轮廓核心：通过衬裙（petticoat/crinoline）做出钟形或A字裙摆。
    - 常见单品：JSK（ジャンパースカート / jumper skirt, English）/ OP（ワンピース / one-piece dress, English），裙长多在膝部附近。
    - 主流子风格：Sweet、Classic、Gothic。
    - 细节关键词：蕾丝层次、胸前蝴蝶结、头饰（ボンネット / bonnet, French 与 リボン / ribbon, English）。
    - 微观观察点：蕾丝层级、裙摆波浪边、胸前蝴蝶结对称性、鞋面擦痕。

- 女仆装（メイド服(めいどふく) / maid outfit, English）
    - 标签：`文化=欧式-维多利亚衍生` `风格=服务角色扮演` `时代=近代/现代再演绎` `功能=制服/表演` `气质=端正或戏剧化`
    - 组成成分：
      - `连衣裙主体`：深色底布为主，领口与袖口常有荷叶边或花边。
      - `围裙`：白色高对比前片，腰带系法可传达角色习惯。
      - `头饰`：荷叶边发箍或头巾，强化职业角色识别。
      - `袖部与下摆`：泡泡袖或长袖版本决定甜系/端正走向。
      - `袜鞋组合`：常配长袜与低跟鞋，强调礼仪与整洁。
    - 常见材质：棉、涤纶、府绸、蕾丝辅料。
    - 常见结构：深色连衣裙 + 白围裙 + 头饰（カチューシャ / katyusha, Russian 与 フリルヘッドドレス / frill headdress, English）。
    - 语义分化：维多利亚风偏端正，法式女仆风偏戏剧化。
    - 场景标签：女仆咖啡店（メイド喫茶(めいどきっさ)）通常搭配固定问候语与角色扮演服务。
    - 微观观察点：围裙褶皱、腰带结尾长短、袖口白边、鞋面抛光程度。

- 死库水（スクール水着(すくーるみずぎ) / school swimsuit, English）
    - 标签：`文化=日式-校园体育` `风格=运动/制服` `时代=现代` `功能=泳装` `气质=功能性`
    - 组成成分：
      - `连体主身`：一体成型包覆躯干，强调贴身和阻力控制。
      - `肩带`：宽肩带更稳定，细肩带更强调外观轻量化。
      - `胸前名牌区`：学校或姓名标识位，常见白色名牌布块。
      - `腿根开口`：开口高低决定活动自由度与视觉风格。
      - `内衬层`：防透与贴肤舒适层，湿水后表现明显。
    - 常见材质：尼龙、涤纶、弹性纤维（氨纶）。
    - 全称缩写来自 school mizugi（スクール水着(すくーるみずぎ)）。
    - 常见为贴身连体款，颜色多为深蓝/黑，面料多见尼龙 + 涤纶。
    - 版本迭代可写：旧型/新型/竞泳型/スパッツ / spats, English 型。
    - 微观观察点：湿水后深浅变化、肩带压痕、名牌边缘卷曲、干湿交界线。

### 工作服装

- 作务衣（作務衣(さむえ) / samue）
    - 标签：`文化=日式-寺务/工务` `风格=朴素` `时代=传统延续` `功能=工装` `气质=克制`
    - 组成成分：
      - `上衣`：交叠前襟，以系带固定，活动时不易掀开。
      - `下装`：宽松长裤，腰部多为抽绳或松紧结构。
      - `袖口`：可卷起处理，便于劳动和清洗。
      - `口袋`：简洁实用，常位于上衣前片或裤侧。
    - 常见材质：棉、麻、竹纤维混纺。
    - 日本僧侣与寺务劳动服出身，后扩展到园艺、餐饮与旅馆工装。
    - 常见面料：棉、麻；版型宽松，便于长时活动。
    - 微观观察点：系带磨毛、膝部褶皱、袖口水渍、布面旧化程度。

- 法被（法被(はっぴ) / happi）
    - 标签：`文化=日式-祭典` `风格=民俗/应援` `时代=传统延续` `功能=活动服` `气质=热闹`
    - 组成成分：
      - `前襟`：直开襟，便于快速穿脱和叠穿。
      - `筒袖`：短而宽，动作幅度大时不束缚手臂。
      - `背纹`：大面积文字或家纹，是团队识别核心。
      - `腰带`：简化固定和调节松紧，便于行进或舞动。
    - 常见材质：棉布、斜纹布、涤棉混纺。
    - 传统筒袖短外衣，常在祭典、商铺活动、应援场景使用。
    - 识别点：前襟与背部印字（店名、组名、祭典名）。
    - 微观观察点：印字开裂、领口汗渍、腰带打结方向、袖口飞边。

- 白衣（白衣(はくい) / ラボコート / lab coat, English）
    - 标签：`文化=现代通用` `风格=专业` `时代=现代` `功能=工装` `气质=理性`
    - 组成成分：
      - `翻领`：驳领或平领，决定职业观感（医疗/实验）。
      - `门襟`：纽扣或暗扣结构，开合速度影响工作效率。
      - `功能口袋`：胸袋与下摆袋用于放笔、手套、记录本。
      - `袖口`：直口或收口，影响防污染和操作便利。
      - `后开衩`：提升步幅和坐姿舒适度。
    - 常见材质：棉涤混纺、防泼液面料、防静电纤维（实验室）。
    - 医疗与实验场景高频工装，强调“防污、可高温清洗、身份识别”。
    - 微观观察点：胸袋笔渍、袖口消毒剂泛白、下摆摩擦皱、暗扣松动。

- 连体工装（つなぎ / tsunagi）
    - 标签：`文化=现代通用` `风格=机能` `时代=现代` `功能=工装` `气质=硬朗`
    - 组成成分：
      - `连体主身`：上衣与裤身一体，减少勾挂风险。
      - `前中拉链`：从领口延伸到裆部，便于快速穿脱。
      - `工具口袋`：胸、臀、腿部多口袋分工明确。
      - `收口结构`：袖口/裤脚可收紧，避免卷入设备。
      - `膝部加固`：双层布或补丁提高耐磨性。
    - 常见材质：帆布、斜纹棉、阻燃面料、耐油污涂层布。
    - 机械维修、仓储、片场布景常见；多口袋、耐磨、便于收纳工具。
    - 微观观察点：拉链齿磨损、膝部油渍层、口袋边缘起毛、反光条刮痕。

### 常服

- 连帽卫衣（フーディー / hoodie, English）
    - 标签：`文化=现代通用` `风格=街头/休闲` `时代=现代` `功能=常服` `气质=松弛`
    - 组成成分：
      - `帽体`：双层或单层帽，决定垂坠和保暖性。
      - `帽绳`：金属头或塑封头，体现做工等级。
      - `前袋`：kangaroo pocket常用于暖手和临时收纳。
      - `袖口下摆罗纹`：收束轮廓并控制松紧。
    - 常见材质：棉、抓绒、毛圈布、涤棉混纺。
    - 轮廓关键词：落肩、帽绳、前袋（kangaroo pocket）。
    - 微观观察点：罗纹回弹、帽绳磨损、前袋边线拉扯、起球密度。

- 开衫（カーディガン / cardigan, English）
    - 标签：`文化=欧式起源-全球化` `风格=学院/通勤` `时代=近代至现代` `功能=常服` `气质=温和`
    - 组成成分：
      - `门襟`：纽扣排决定开合方式和视觉纵线。
      - `领型`：V领/圆领影响脸部比例和内搭露出面积。
      - `袖型`：插肩袖或常规袖影响肩线柔和度。
      - `罗纹边`：领口、袖口、下摆统一收束。
    - 常见材质：羊毛、棉纱、腈纶、羊绒混纺。
    - 叠穿友好：衬衫外层、制服外层、吊带外层。
    - 微观观察点：扣眼松紧、肘部起鼓、袖口起球、肩线塌陷。

- 针织背心（ニットベスト / knit vest, English）
    - 标签：`文化=欧式起源-全球化` `风格=学院` `时代=现代` `功能=常服/制服内搭` `气质=文雅`
    - 组成成分：
      - `领口`：V领最常见，露出领带结和衬衫领尖。
      - `肩部`：无袖开口尺寸决定手臂活动和层次露出。
      - `下摆罗纹`：控制腰部收束，防止上翻。
      - `纹理`：平针、麻花、提花影响视觉密度。
    - 常见材质：棉纱、羊毛、腈纶混纺。
    - 常用于“学院感”造型，配衬衫与领带层次明确。
    - 微观观察点：领口卷边、肩口磨毛、织纹拉伸、内搭透影。

- 百褶裙（プリーツスカート / pleated skirt, English）
    - 标签：`文化=现代通用` `风格=学院/通勤` `时代=现代` `功能=常服/制服` `气质=轻盈`
    - 组成成分：
      - `腰头`：硬衬腰头决定贴合和稳定性。
      - `褶裥区`：刀褶/箱褶不同，动态表现差异明显。
      - `裙长`：膝上、及膝、过膝对应不同礼仪和气质。
      - `内衬`：提升舒适度并减少透感。
    - 常见材质：毛涤混纺、聚酯纤维、棉质斜纹。
    - 静态看褶线密度，动态看摆幅与回弹。
    - 微观观察点：褶线保型度、腰扣磨损、坐压折痕、下摆飞线。

- 直筒牛仔裤（ストレートデニム / straight denim, English）
    - 标签：`文化=现代通用` `风格=工装衍生/休闲` `时代=现代` `功能=常服` `气质=中性`
    - 组成成分：
      - `裤腰`：腰头 + 皮牌 + 纽扣门襟，决定品牌识别度。
      - `裆部结构`：前后片拼接影响活动舒适度。
      - `口袋系统`：前插袋、零钱袋、后贴袋兼具功能与造型。
      - `裤管`：直筒线条从大腿到脚口基本一致。
      - `铆钉和车线`：高受力部位的耐久关键。
    - 常见材质：丹宁布（棉为主）、弹力牛仔（加氨纶）。
    - 写作重点：洗水、磨白、膝部折痕、裤脚堆积。
    - 微观观察点：猫须纹、蜂窝褶、口袋边磨损、脚口毛边。

- 切斯特大衣（チェスターコート / chester coat, English）
    - 标签：`文化=欧式-近代男装衍生` `风格=都市/正式` `时代=近代至现代` `功能=外套` `气质=克制`
    - 组成成分：
      - `驳领`：缺角驳领最常见，决定正式度。
      - `门襟`：单排或双排扣，控制胸前体量。
      - `肩部结构`：垫肩强弱影响气场和轮廓硬度。
      - `口袋`：插袋或盖袋，兼顾线条与实用。
      - `后开衩`：单衩/双衩决定步幅与动态垂感。
    - 常见材质：羊毛、羊绒混纺、粗花呢。
    - 冬季通勤核心单品，肩线和翻领决定气场。
    - 微观观察点：呢面起毛、袖口抛光磨亮、纽扣松紧、雨雪吸水痕。

### 配饰

- 领结（リボン / ribbon, English）
    - 标签：`文化=日式-制服` `风格=学院/甜系` `时代=现代` `功能=配饰` `气质=可爱`
    - 制服系核心识别件，结体大小影响“幼态/成熟”观感。

- 领带（ネクタイ / necktie, English）
    - 标签：`文化=欧式-正装` `风格=正式/学院` `时代=近代至现代` `功能=配饰` `气质=理性`
    - 常见打法：平结、温莎结；松紧度可映射角色状态。

- 过膝袜（ニーハイソックス / knee-high socks, English）
    - 标签：`文化=现代通用` `风格=学院/二次元` `时代=现代` `功能=配饰` `气质=活泼`
    - 与裙装搭配时，露肤比例会显著改变角色气质。

- 松垮袜（ルーズソックス / loose socks, English）
    - 标签：`文化=日式-街头校园` `风格=复古校园` `时代=现代` `功能=配饰` `气质=张扬`
    - 90年代至00年代初街头校园风高频视觉符号。

- 乐福鞋（ローファー / loafer, English）
    - 标签：`文化=欧式起源-校园化` `风格=学院/通勤` `时代=近代至现代` `功能=鞋履` `气质=稳重`

### 中式服装

- 汉服（汉服 / Hanfu）
    - 标签：`文化=中式-汉文化` `风格=礼制/古典` `时代=古典与现代复兴` `功能=礼服/常服` `气质=端雅`
    - 组成成分：
      - `内层中衣`：贴身基础层，吸汗并保护外层衣料。
      - `上衣`：交领右衽为核心，系带隐藏在前襟内外层。
      - `下装`：裙或裤按朝代与场景变化。
      - `腰封/系带`：固定层次并调整视觉比例。
      - `披帛/外搭`：提升仪式感和动态飘带效果。
    - 常见材质：丝、纱、棉、麻、锦、绫罗。
    - 泛指汉族传统服饰体系，核心特征是交领右衽、系带结构、层次叠穿。
    - 微观观察点：领缘包边、系带收口、衣摆垂坠线、纹样织密度。

- 襦裙（襦裙 / Ruqun）
    - 标签：`文化=中式-汉文化` `风格=古典` `时代=先秦至明清多期演化` `功能=常服/礼服` `气质=柔和`
    - 组成成分：
      - `襦`：短上衣，交领或对襟，袖形决定时代感。
      - `裙`：高腰或齐腰长裙，重点在腰线位置与褶量。
      - `系带`：长带绕系，结位可在前胸、腰侧或背后。
      - `披帛`：可选配，增强动态和层次。
    - 常见材质：丝绸、棉麻、纱罗。
    - 上襦下裙的两件式结构，高腰/齐胸/齐腰等形制均可用于时代区分。
    - 微观观察点：胸前系带压痕、裙褶展开节奏、袖口压线、下摆拖地痕。

- 马面裙（马面裙 / Mamianqun）
    - 标签：`文化=中式-明清` `风格=礼制/华丽` `时代=宋元原型-明清成熟` `功能=礼服/常服` `气质=端庄`
    - 组成成分：
      - `裙门`：前后平面区（马面）形成视觉纵向中轴。
      - `侧褶`：两侧密褶提供步行动态和容量。
      - `腰头`：多层缝制承重，保证垂坠稳定。
      - `系结`：长带或扣件固定，决定穿着紧致度。
    - 常见材质：织金锦、妆花、缎、棉混纺日常料。
    - 识别点：前后“马面”平面区 + 两侧褶裥，行走时开合节奏明显。
    - 微观观察点：裙门纹样对称、侧褶回弹、腰头厚度、裙脚磨边。

- 圆领袍（圆领袍 / Yuanlingpao）
    - 标签：`文化=中式-官民通用` `风格=礼仪/官式` `时代=隋唐至明` `功能=礼服/常服` `气质=庄重`
    - 组成成分：
      - `圆领`：贴颈环形领口，强调中轴与秩序感。
      - `袍身`：直身长袍结构，前后片连贯。
      - `袖部`：宽袖或窄袖依时代与身份变化。

    - 常见于官服和士人服饰系统，可通过补子、色阶、纹样体现阶序。

- 褙子（褙子 / Beizi）

    - 组成成分：
      - `前襟`：对襟直开，便于内搭展示。

- 旗袍（旗袍 / Qipao / Cheongsam）

    - 组成成分：
      - `立领`：包颈高度决定端庄度和限制感。
      - `门襟`：右衽开襟，盘扣是视觉焦点。
      - `胸腰省道`：塑造贴体曲线与身体线条。
      - `侧开衩`：控制步幅并形成动态“开合线”。
      - `下摆`：长度与开衩终点共同决定风格边界。
    - 常见材质：真丝、织锦、缎面、棉麻混纺日常款。
    - 识别点：立领、盘扣、右衽开襟、侧开衩；可用开衩高度区分保守与前卫。
    - 微观观察点：盘扣密度、领口压痕、臀腿处拉力纹、下摆转身轨迹。

### 欧式服装

- 束身衣（Corset）
    - 标签：`文化=欧式-近代` `风格=宫廷/复古` `时代=16-19世纪核心` `功能=塑形内衣` `气质=克制`
    - 组成成分：
      - `前中骨架`：金属或硬骨支撑正面平直。
      - `侧后骨条`：均匀分布以维持收腰曲线。
      - `后背系带`：松紧调节核心，决定塑形力度。
      - `胸口边缘`：低胸或高胸版本对应不同礼仪语境。
    - 常见材质：棉斜纹、缎面、皮革、钢骨/塑骨。
    - 核心机制：收腰塑形 + 支撑上身结构，常与裙撑体系联动。
    - 微观观察点：系带压痕、骨条顶点磨损、呼吸节奏变化、坐姿受限感。

- 燕尾服（Tailcoat）
    - 标签：`文化=欧式-正装体系` `风格=礼仪` `时代=近代至现代仪式场合` `功能=礼服` `气质=庄严`
    - 组成成分：
      - `前片`：前摆短切，露出马甲和礼服衬衣。
      - `后摆`：双尾分叉，是燕尾服核心识别结构。
      - `驳领`：常见丝缎面，形成礼服光泽对比。
      - `肩袖结构`：高挺肩线与窄袖强调仪式感。
    - 常见材质：羊毛精纺、缎面驳领、丝衬里。
    - 常见于白领结礼仪场景，后摆分叉是识别点。
    - 微观观察点：后摆甩动弧度、驳领反光、袖口露白长度、纽扣光泽。

- 马甲（Waistcoat）
    - 标签：`文化=欧式-三件套` `风格=绅士/通勤` `时代=近代至现代` `功能=中层` `气质=精致`
    - 组成成分：
      - `前身`：面料与西装一致或对比，用于构建层次中心。
      - `后背`：常用滑面衬里，减少与外套摩擦。
      - `门襟`：单排扣为主，扣数影响正式程度。
      - `后腰调节带`：微调腰围贴合度。
    - 常见材质：羊毛混纺、提花、棉、丝质后背。
    - 与衬衫、西装叠穿时可塑造“层次和秩序感”。
    - 微观观察点：门襟平整度、最后一颗扣法、后背调节带松紧、口袋怀表链。

- 斗篷（Cloak / Cape）
    - 标签：`文化=欧式-历史延续` `风格=戏剧化/旅行` `时代=古典至现代再演绎` `功能=外披` `气质=神秘`
    - 组成成分：
      - `披身`：半圆或全圆裁片，决定包覆范围与垂感。
      - `领部`：立领或翻领，构建脸部框线和气场。
      - `闭合件`：系带、搭扣、胸针用于前胸固定。
      - `兜帽`：可选结构，增强叙事功能性。
    - 常见材质：呢料、羊毛、天鹅绒、皮革。
    - 重点可写：披挂动作、领口固定件、行走时后摆拖曳。
    - 微观观察点：披摆扫地声、边缘毛绒磨损、扣件冷光、雨水流痕。

- 双排扣大衣（Double-breasted coat）
    - 标签：`文化=欧式-军政衍生` `风格=正式/军感` `时代=近代至现代` `功能=外套` `气质=硬朗`
    - 组成成分：
      - `双排门襟`：前片重叠提供防风与体量感。
      - `驳领`：宽驳领强化肩胸区域视觉权重。
      - `肩部`：肩章或硬挺肩线可加入军感信息。
      - `腰线`：收腰或直身影响力量感和机动性。
      - `袖口`：扣袢或装饰扣用于风格强调。
    - 常见材质：粗花呢、羊毛呢、防泼水混纺。
    - 特征是前襟重叠与双列纽扣，可通过扣位表现人物控制欲或松弛度。
    - 微观观察点：纽扣磨亮区、领口折痕、肩部压线、下摆吸水重量。

    - 组成成分：

      - `领口`：圆领或开口领，便于套穿。
      - `腰带`：收束衣身并调节长度。

    - 常见材质：羊毛、亚麻。

    - 组成成分：
      - `长袍主身`：直筒长裙结构，长度多及足背。
      - `肩部固定`：肩部别针或缝位固定上沿。
      - `腰线收束`：腰带形成上提褶层。
      - `外披层`：常与帕拉组合增强外出礼仪性。
    - 常见材质：羊毛、亚麻、细织布。

      - `矩形披布`：单片大布，依折法形成不同遮蔽程度。
      - `肩颈绕法`：可单肩披挂或双肩包覆。
      - `头部覆法`：公共场景可上提覆盖头部。
    - 常见材质：羊毛、亚麻。
    - 矩形披布，可罩肩、绕体、覆头，适合表现“公共空间中的自我保护”。

      - `鞋底`：厚底结构承重，适应长距离行军。
      - `绑带`：皮带交叉缠绕小腿，稳定脚踝。
      - `鞋面开孔`：增强透气并减轻重量。
      - `底钉`：提升抓地和耐磨。
    - 常见材质：皮革、金属钉。
    - 罗马军靴代表样式，绑带结构和鞋底钉可强化行军质感。
    - 微观观察点：绑带勒痕、底钉敲击声、泥尘堆积、鞋底磨平区。

    - 组成成分：
      - `斜襟领口`：门襟偏向一侧，形成高识别度“偏领线”。
      - `长袖衣身`：常至大腿中段，便于劳动与保暖。
      - `腰带`：束腰后形成上身蓬量，便于活动。
      - `袖口刺绣区`：传统图样多落在领口、袖口、前襟边。
    - 常见材质：亚麻、棉、羊毛混纺。
    - 微观观察点：斜襟扣位、腰带结尾垂线、刺绣针脚密度、袖口磨毛。

- 俄式无袖罩裙（Сарафан / sarafan）
    - 标签：`文化=斯拉夫系-俄式` `风格=民俗/节庆` `时代=近代前后延续` `功能=常服/礼服` `气质=端雅`
    - 组成成分：
      - `罩裙主体`：无袖长裙外层，纵向线条拉长身形。
      - `肩带`：宽窄与位置决定上身比例与稳定性。
      - `胸前拼片`：常用于装饰和结构加固。
      - `内搭衬衫`：与刺绣袖口形成层次对比。
    - 常见材质：亚麻、棉布、羊毛、丝质节庆料。
    - 微观观察点：肩带压痕、胸前拼缝、下摆拖量、转身摆幅。

- 刺绣衬衣（Вышиванка / vyshyvanka）
    - 标签：`文化=斯拉夫系-乌克兰/白俄罗斯` `风格=刺绣民俗` `时代=古典至现代复兴` `功能=常服/节庆` `气质=明丽`
    - 组成成分：
      - `衬衣主体`：直身版型，方便叠穿与活动。
      - `领口刺绣`：几何或植物纹常用于身份和地域识别。
      - `袖口刺绣`：高频装饰位，颜色密度决定视觉焦点。
      - `前胸刺绣带`：沿门襟或胸前纵向布局。
    - 常见材质：亚麻、棉、家织布。
    - 微观观察点：红黑配色比例、针法粗细、领口磨损、汗渍与染线反差。

- 传统头饰（Кокошник / kokoshnik）
    - 标签：`文化=斯拉夫系-俄式` `风格=节庆礼仪` `时代=古典至民俗表演延续` `功能=头饰` `气质=华丽`
    - 组成成分：
      - `硬挺冠体`：新月或拱形轮廓，决定正面仪式感。
      - `额前装饰`：珠网、金线或花纹集中在视线焦点区。
      - `后系带`：长带在脑后固定，影响稳定性与背面动态。
      - `内衬`：提高佩戴舒适并防止滑动。
    - 常见材质：织物底胚、珠饰、金线、缎带。
    - 微观观察点：额前珠链摆动、冠体边缘硬挺度、后系带长度、反光颗粒感。

- 护耳帽（Ушанка / ushanka）
    - 标签：`文化=斯拉夫系-寒带生活` `风格=冬季机能` `时代=近代至现代` `功能=帽饰` `气质=硬朗`
    - 组成成分：
      - `帽冠`：保温核心，决定贴合与头顶轮廓。
      - `护耳翻片`：可上翻系顶、下放护耳、后绑通风。
      - `下巴固定带`：在强风与低温下固定翻片。
      - `内衬层`：提升保暖与吸湿性能。
    - 常见材质：羊皮、兔毛、人造毛、保暖织物。
    - 微观观察点：翻片折痕、耳侧压毛、呼气凝霜、系带勒痕。

- 毡靴（Валенки / valenki）
    - 标签：`文化=斯拉夫系-寒带民生` `风格=冬季实用` `时代=近代至现代` `功能=鞋履` `气质=朴拙`
    - 组成成分：
      - `毡靴主体`：整体毡化成型，保温强但不防积水。
      - `靴口`：高筒包覆小腿，减少雪灌入。
      - `外加套鞋`：常配胶鞋（galoshes）防潮防磨。
      - `鞋底补强`：皮革或橡胶补底提升耐磨。
    - 常见材质：羊毛毡、橡胶外套鞋。
    - 微观观察点：雪水浸润线、靴筒起毛、套鞋泥点、行走闷响。

- 树皮鞋（Лапти / lapti）
    - 标签：`文化=斯拉夫系-乡村传统` `风格=民俗实用` `时代=古典至近代` `功能=鞋履` `气质=粗粝`
    - 组成成分：
      - `编织鞋身`：韧皮纤维交错编结，轻但耐久度有限。
      - `绑腿带`：缠绕脚踝和小腿，固定鞋体并防松脱。
      - `内垫层`：草束或布片提升舒适和保温。
      - `鞋头结构`：编织密度决定前掌支撑力。
    - 常见材质：椴树皮、桦树皮、麻绳。
    - 微观观察点：编织纹理松紧、纤维毛刺、绑带压痕、雨后软塌感。


## 属性类

### 属性记录规范

- 属性记录模板：
    - `基线值`：角色在平稳状态下的默认表现。
    - `波动范围`：受情绪、环境、伤病、关系影响的上下限。
    - `触发条件`：导致属性变化的事件或刺激。
    - `外显信号`：他人可观察到的动作、语气、姿态或生理反应。
    - `恢复机制`：角色如何从异常状态返回基线。
    - `他人可见度`：公开/半公开/隐匿。
- 写作落点建议：优先写“外显信号”，少写抽象结论。
- 回写节奏建议：每章至少记录 1 条属性变化（可微小）。

### 视觉属性

- 发色（髪色(かみいろ) / hair color）
    - 标签：`文化=通用` `风格=视觉锚点`
    - 细化字段：底色、挑染、根部新生色差、明度梯度、冷暖偏向、逆光反边。
    - 微观细节：发根油光、发尾分叉、湿发贴颈、静电翘起。

- 发质（hair texture）
    - 标签：`文化=通用` `风格=状态显影`
    - 细化字段：粗细、卷曲度、蓬松度、顺滑度、受潮变化。
    - 微观细节：梳齿卡顿、发丝回弹、毛躁边缘、定型残留。

- 瞳色（瞳色(ひとみいろ) / eye color）
    - 标签：`文化=通用` `风格=角色识别`
    - 细化字段：主色、外圈深度、虹膜纹理、光点位置、充血程度。
    - 微观细节：高光扩散、眨眼频率、瞳孔收缩、泪膜反光。

- 肤况（skin condition）
    - 标签：`文化=通用` `风格=身体状态`
    - 细化字段：肤色基调、血色、干湿、温度感、损伤痕迹。
    - 微观细节：指节泛白、颈侧潮红、唇角干裂、旧伤色差。

- 体态（姿勢(しせい) / posture）
    - 标签：`文化=通用` `风格=行为叙事`
    - 细化字段：重心（前倾/后仰）、肩颈张力、骨盆角度、步幅、摆臂。
    - 微观细节：缩肩、锁骨起伏、脚尖朝向、防御性抱臂。

### 姿态类型库（新增）

- 站立姿态（静态）
    - 中轴中立：双脚平行，重心居中；适合“平稳、可谈判”场景。
    - 前倾压迫：上身微前、下颌前送；常见于施压或抢话语权。
    - 后仰防御：骨盆后坐、上身后撤；常见于戒备或不信任。
    - 开放展示：肩背展开、手臂外放；常见于自信或主场感。
    - 闭合收束：双臂贴躯、肩线内收；常见于不安、寒冷或服从。
    - 倚靠立姿：肩背/臀部借墙借桌；常见于拖延、观察、轻挑衅。

- 站立姿态（动态前兆）
    - 起步预备：一脚前探、脚跟抬起；表示“准备离场/追击/撤退”。
    - 半蹲蓄力：膝微屈、重心下沉；表示“可能突发动作”。
    - 侧身让位：躯干转开、手臂引导；表示“礼让或软性拒绝”。
    - 脚尖指向出口：身体未动但意图先行；表示“想结束对话”。

- 坐姿类型
    - 正坐挺背：骨盆中立、背柱上提；常见于正式会谈或受训状态。
    - 前缘坐：坐在椅子前1/3，重心前压；常见于专注和攻击性提问。
    - 后靠摊坐：靠背完全承重；常见于疲劳、松弛或轻视。
    - 并膝收束：膝并拢、手放腿面；常见于礼仪场合或自我约束。
    - 交叠腿：节奏慢，视线稳定；常见于掌控局面或审视他人。
    - 抱膝/缩腿：躯干包裹下肢；常见于自我保护与低安全感。

- 行进姿态
    - 猎行快步：步幅短而频率高，视线扫角落；适合追踪与巡查。
    - 静步潜行：落地由前掌过渡，声响最低；适合潜入/躲避。
    - 重步推进：脚跟先落、声压明显；适合威慑或愤怒推进。
    - 拖步回撤：步伐黏地、肩线下坠；适合受伤、失落、过劳状态。

- 手部姿态（伴随姿态）
    - 抱臂：防御、评判或保温；要看肩线是否放松来区分敌意。
    - 背手：控制情绪或维持礼仪；常见于上位者或受训者。
    - 指尖对触：思考与组织语言；常见于分析型角色。
    - 摸后颈/搓指节：焦虑和自我安抚高频动作。
    - 反复整衣：恢复秩序感，常见于紧张后自我重建。

- 头颈姿态
    - 低头避视：回避冲突、羞耻或服从。
    - 抬颌俯视：挑衅、压场或防御性自大。
    - 歪头倾听：示好、困惑或诱导对方继续说。
    - 颈部定住：情绪冻结或高警惕的外显信号。

- 姿态误判提醒
    - 冷并不总是敌意：寒冷环境会造成闭合姿态假象。
    - 疲劳会伪装成傲慢：后靠摊坐不必然等于轻视。
    - 文化差异影响礼仪姿势：并膝/距离阈值需按文化背景解读。

- 步态（gait）
    - 标签：`文化=通用` `风格=节奏信息`
    - 细化字段：步频、落地顺序、脚跟声、转向习惯、停步方式。
    - 微观细节：拖步、急停滑移、跨步避让、楼梯节拍变化。

- 手部特征（hand signature）
    - 标签：`文化=通用` `风格=职业痕迹`
    - 细化字段：掌纹粗糙度、茧位、指甲修整、常用手偏好。
    - 微观细节：笔茧、刀茧、指尖裂口、戒指压痕。

### 生理属性

- 体温耐受（thermal tolerance）
    - 标签：`文化=通用` `风格=环境适应`
    - 细化字段：耐寒阈值、耐热阈值、出汗模式、保温习惯。
    - 微观细节：鼻尖发红、耳后出汗、手心湿度、呼气白雾。

- 疲劳阈值（fatigue threshold）
    - 标签：`文化=通用` `风格=行动续航`
    - 细化字段：持续专注时长、体力衰减点、恢复窗口。
    - 微观细节：眼睑下坠、肩部僵硬、语尾变短、动作省略。

- 痛觉反应（pain response）
    - 标签：`文化=通用` `风格=受损反馈`
    - 细化字段：耐痛级别、反射动作、延迟反应、硬撑能力。
    - 微观细节：吸气停顿、下颌收紧、手指蜷缩、呼吸失拍。

- 睡眠结构（sleep pattern）
    - 标签：`文化=通用` `风格=恢复机制`
    - 细化字段：入睡时长、浅眠比例、梦扰频率、起床状态。
    - 微观细节：黑眼圈层次、晨起沙哑、午后断电、夜间惊醒。

### 行为属性

- 说话速度（話速(わそく) / speech rate）
    - 标签：`文化=通用` `风格=节奏控制`
    - 细化字段：平均语速、高压语速、低压语速、句长偏好。
    - 微观细节：连词密度、吞音、抢答、句尾拖音。

- 口癖（口癖(くちぐせ) / verbal tic）
    - 标签：`文化=通用` `风格=角色语气`
    - 细化字段：触发情境、频率上限、替代词、剧情阶段变化。
    - 微观细节：紧张时口癖回潮、克制时中断、模仿他人口癖。

- 语调曲线（intonation profile）
    - 标签：`文化=通用` `风格=情绪显影`
    - 细化字段：高低起伏、重音位置、句末收束方式。
    - 微观细节：音高突升、尾音下坠、笑意挂音、冷平语调。

- 停顿模式（pause pattern）
    - 标签：`文化=通用` `风格=心理线索`
    - 细化字段：停顿长度、停顿位置、补词习惯、改口频率。
    - 微观细节：逗号前停、关键名词前断句、咽口水后继续。

- 视线策略（gaze strategy）
    - 标签：`文化=通用` `风格=关系控制`
    - 细化字段：对视时长、回避方向、扫视范围、盯视目标。
    - 微观细节：先看手再看眼、说谎时看向出口、听重点时瞳孔收缩。

- 手势习惯（gesture habit）
    - 标签：`文化=通用` `风格=动作识别`
    - 细化字段：幅度、频率、左右手主次、触物偏好。
    - 微观细节：捏袖口、敲桌面、摸后颈、拇指摩擦关节。

- 决策节奏（decision cadence）
    - 标签：`文化=通用` `风格=行动逻辑`
    - 细化字段：信息收集时长、决策窗口、复盘意愿。
    - 微观细节：先问后做、先做后补、临界点突然加速。

### 情绪属性

- 情绪阈值（emotional threshold）
    - 标签：`文化=通用` `风格=冲突反应`
    - 细化字段：愤怒阈值、羞耻阈值、恐惧阈值、兴奋阈值。
    - 微观细节：耳尖泛红、语速突增、动作僵停、笑容冻结。

- 恢复速度（recovery speed）
    - 标签：`文化=通用` `风格=韧性`
    - 细化字段：自我平复时长、外部安抚依赖、恢复后残留影响。
    - 微观细节：深呼吸节拍、喝水动作、复述确认、话题转移。

- 情绪伪装能力（masking）
    - 标签：`文化=通用` `风格=社会化表现`
    - 细化字段：可伪装情绪类型、伪装持续时长、破功触发点。
    - 微观细节：笑眼不同步、嘴角上扬但肩颈僵硬、语调过平。

### 认知属性

- 注意力分配（attention allocation）
    - 标签：`文化=通用` `风格=信息处理`
    - 细化字段：单线程/多线程、干扰敏感度、优先级切换速度。
    - 微观细节：听人时手不停、环境噪声下重复确认、漏听专有名词。

- 记忆模式（memory profile）
    - 标签：`文化=通用` `风格=叙事回收`
    - 细化字段：情景记忆、程序性记忆、名字脸对应能力、遗忘曲线。
    - 微观细节：先记画面后记名字、重复口令自检、时间线倒序复述。

- 风险偏好（risk appetite）
    - 标签：`文化=通用` `风格=策略倾向`
    - 细化字段：保守/激进、可承受损失、止损触发条件。
    - 微观细节：先问退路、愿意押注、临界点撤退、事后是否后悔。

### 社会属性

- 权威响应（authority response）
    - 标签：`文化=通用` `风格=组织关系`
    - 细化字段：服从倾向、质疑阈值、公开反驳能力。
    - 微观细节：先称呼再反对、沉默抵抗、执行时留后手。

- 边界感（boundary sense）
    - 标签：`文化=通用` `风格=亲疏控制`
    - 细化字段：身体距离阈值、隐私共享阈值、拒绝表达方式。
    - 微观细节：后退半步、侧身让位、转移话题、用笑挡拒绝。

- 信任建立（trust formation）
    - 标签：`文化=通用` `风格=关系推进`
    - 细化字段：信任前置条件、验证次数、背叛容忍度。
    - 微观细节：重复确认承诺、先给小权限、失败后立即收回。

### 能力属性（世界观适配）

- 魔力输出曲线（mana output curve）
    - 标签：`文化=世界观通用` `风格=能力控制`
    - 细化字段：起手峰值、持续输出、衰减速度、过载阈值。
    - 微观细节：施法前呼吸、手势抖动、声线变化、余波滞留。

- 施法前摇（casting startup）
    - 标签：`文化=世界观通用` `风格=战术窗口`
    - 细化字段：吟唱长度、手势复杂度、可打断性、伪动作能力。
    - 微观细节：口型预备、指节定位、短停顿骗招、步伐定轴。

- 反噬阈值（backlash threshold）
    - 标签：`文化=世界观通用` `风格=代价系统`
    - 细化字段：安全区间、预警信号、崩溃触发点、恢复周期。
    - 微观细节：鼻腔血丝、耳鸣、视野暗角、手背微颤。

### 角色原型标签（用于速记，不等于定型）

- 傲娇（ツンデレ / tsundere, 和製語）
    - 标签：`文化=日式-ACGN` `风格=关系推进`
    - 外冷内热，关系推进后逐步暴露柔软面。

- 辣妹系（ギャル / gal, English）
    - 标签：`文化=日式-街头` `风格=自我表达`
    - 重点是风格张力，不等于固定性格。

- 学者型（scholar archetype）
    - 标签：`文化=通用` `风格=理性/系统化`
    - 冲突处理中偏好证据链和定义澄清，情绪表达常后置。

- 猎手型（hunter archetype）
    - 标签：`文化=通用` `风格=追踪/执行`
    - 对线索和时机高度敏感，倾向短决策链和高行动密度。


## 日常生活类

### 行为模式

#### 日式

    - 标签：`文化=日式` `风格=居住礼仪` `功能=入户行为`

- 便当分层装填与午间交换
    - 标签：`文化=日式` `风格=校园/通勤` `功能=餐食行为`

    - 微观细节：米饭蒸汽、隔层串味、弹性绑带勒痕、筷套折线。

- 交通闸机轻触通行
    - 标签：`文化=日式` `风格=城市节奏` `功能=通勤行为`
    - 关键动作：IC卡或手机贴近感应区 -> 提示音确认 -> 连续通过闸门。
    - 微观细节：闸机蜂鸣差异、失败时短暂停步、包带摆动节奏。

- 钱汤入浴流程
    - 标签：`文化=日式` `风格=公共礼仪` `功能=清洁/社交`
    - 关键动作：更衣 -> 先冲洗 -> 入池 -> 冷却休息 -> 离池擦干。
    - 微观细节：木桶碰瓷砖声、蒸汽附着、毛巾折叠位置、镜面水雾。

    - 微观细节：旋钮阻尼感、胶囊撞击声、塑封开裂纹、重复款堆积。

- 拍贴留档与关系确认
    - 标签：`文化=日式` `风格=青年社交` `功能=记忆锚点`

    - 微观细节：贴纸背胶气味、笔迹重描、包壳夹层老化、边角翘起。

### 场所

    - 微观细节：卷帘门升降声、招牌老化层、晚间霓虹反射、雨天排水沟声。

    - 微观细节：石阶苔痕、手水舍水声、绘马木纹、风铃间歇响。

- 学校屋上（屋上(おくじょう) / rooftop, English）

    - 微观细节：防护网影子、铁门链锁、风压扰发、远处社团口号回声。

    - 微观细节：边界石刻、供物残渣、祭盘铜锈、足音自发压低。

- 港口（Limen, Greek）
    - 标签：`文化=古希腊` `风格=贸易与远征节点`
    - 商贸、雇佣、情报与航海风险聚合，剧情推进效率高。
    - 微观细节：缆绳毛刺、桅杆吱响、盐雾附着、装卸口令节拍。

#### 斯拉夫系

- 蒸汽浴室（Баня / banya）
    - 标签：`文化=斯拉夫系` `风格=净身/社交复合场所`
    - 木结构与蒸汽空间并存，冷热转换动线明确。

- 木屋（Изба / izba）
    - 标签：`文化=斯拉夫系` `风格=乡村居住`
    - 原木墙体、炉灶核心、低照度室内形成强地域氛围。

- 郊野别墅（Дача / dacha）
    - 标签：`文化=斯拉夫系` `风格=季节居住`
    - 夏秋高频使用，常与菜园、储藏和修缮行为绑定。

- 木制东正教教堂
    - 标签：`文化=斯拉夫系` `风格=宗教场所`
    - 木构尖顶、圣像陈列与烛光层次可强化神圣感与历史厚度。

- 集市广场（Торг / torg, Russian）
    - 标签：`文化=斯拉夫系` `风格=交易与消息集散`
    - 农产、手工、传闻同步流动，是“民意温度计”场景。
    - 微观细节：摊布泥点、秤砣磕木、盐渍桶壁、讨价还价重音。

- 合住公寓（Коммуналка / kommunalka, Russian）
    - 标签：`文化=斯拉夫系-近现代` `风格=高密度共居`
    - 私密与公共边界模糊，冲突和互助会高频交替出现。
    - 微观细节：公共厨房排队、走廊门链声、墙纸受潮起鼓、共用灶台油膜。

- 地铁换乘大厅（Метро / metro, Russian）
    - 标签：`文化=斯拉夫系-城市` `风格=通勤洪流场`
    - 速度与秩序并存，适合写“人流掩护、错位追踪、临场决断”。
    - 微观细节：扶梯风压、报站回声、票闸灯色、鞋底雪水拖线。

- 谷仓（Амбар / ambar, Russian）
    - 标签：`文化=斯拉夫系` `风格=储藏与冬备`
    - 物资管理核心点，适合写“丰歉、偷盗、封存与分配冲突”。
    - 微观细节：木梁霉斑、谷粒摩擦声、铁锁锈蚀、鼠迹粉尘线。


## 甜点系

### 标注规范

- 甜点词条统一写法：`中文名（日语名 / 原生语言词形, 语种）`
- 纯片假名词必须补原生语言词形；和製語需标注其来源语。
- 每条建议包含：`组成要素`、`口感结构`、`微观细节`。

### 法系（French）

- 泡芙（シュークリーム / chou à la crème, French）
    - 标签：`文化=法式甜点-日式流行` `风格=轻甜` `功能=日常甜食`
    - 备注：`シュークリーム` 为和製語（和製英語），语义上对应法式奶油泡芙体系。
    - 组成要素：烫面壳体 + 奶油夹心（卡仕达/鲜奶油）+ 表面糖粉（可选）。
    - 口感结构：外壳薄脆 -> 内层空腔 -> 夹心绵滑。
    - 微观细节：壳体裂纹、回潮软化线、夹心挤出痕、纸托油印。

- 闪电泡芙（エクレア / éclair, French）
    - 标签：`文化=法式甜点` `风格=细长条形` `功能=下午茶`
    - 组成要素：烫面条体 + 奶油/卡仕达夹心 + 巧克力或糖霜表层。
    - 口感结构：外层轻脆 -> 中段柔软 -> 顶层脆糖或镜面。
    - 微观细节：糖面开裂、切口奶油渗线、握持压痕、顶层反光。

- 千层酥（ミルフィーユ / mille-feuille, French）
    - 标签：`文化=法式甜点` `风格=层叠` `功能=礼盘甜点`
    - 组成要素：千层酥皮 + 奶油层 + 糖粉/糖霜。
    - 口感结构：多层酥碎 + 奶油润滑 + 轻甜收口。
    - 微观细节：切割时层片散落、奶油位移、盘面酥屑轨迹。

- 可露丽（カヌレ / canelé, French）
    - 标签：`文化=法式甜点` `风格=焦糖外壳` `功能=咖啡搭配`
    - 组成要素：蛋奶酒香面糊（常含兰姆/香草）+ 蜂蜡或油脂模壁处理。
    - 口感结构：外壳脆焦 -> 内里湿润弹软。
    - 微观细节：壳面深浅不均、切面蜂窝孔、冷却后外壳硬化声。

- 马卡龙（マカロン / macaron, French）
    - 标签：`文化=法式甜点` `风格=精致小体积` `功能=礼赠`
    - 组成要素：杏仁粉壳体 + 夹心（甘纳许/果酱/奶油霜）。
    - 口感结构：薄脆壳 -> 湿润内芯 -> 柔软夹层。
    - 微观细节：裙边高度、壳面裂纹、色粉均匀度、夹层溢边。

- 芭菲（パフェ / parfait, French）
    - 标签：`文化=法语词源-日式发展` `风格=杯装分层` `功能=社交打卡`
    - 组成要素：冰淇淋/奶油 + 水果 + 谷物脆层 + 酱料分层。
    - 口感结构：冷、脆、软、酸甜交替。
    - 微观细节：杯壁挂酱、层间塌陷、融化液位、勺尖轨迹。

### 意大利系（Italian）

- 提拉米苏（ティラミス / tiramisù, Italian）
    - 标签：`文化=意式甜点` `风格=咖啡可可` `功能=餐后甜点`
    - 组成要素：手指饼干 + 马斯卡彭奶酪层 + 咖啡液 + 可可粉。
    - 口感结构：湿润浸透层 + 奶香厚度 + 苦甜平衡。
    - 微观细节：表层可可裂纹、切片坍塌线、咖啡浸润深浅。

- 意式奶冻（パンナコッタ / panna cotta, Italian）
    - 标签：`文化=意式甜点` `风格=冷凝乳冻` `功能=清口甜点`
    - 组成要素：奶油基底 + 凝固剂 + 果酱或果泥顶层。
    - 口感结构：整体颤感明显，入口细滑，尾味偏奶。
    - 微观细节：脱模边缘完整度、勺压回弹、果酱流速。

- 意式冰淇淋（ジェラート / gelato, Italian）
    - 标签：`文化=意式甜点` `风格=低空气含量` `功能=冷甜点`
    - 组成要素：奶基/果基 + 糖 + 风味主材（坚果/水果/巧克力）。
    - 口感结构：密度高、融化慢、风味集中。
    - 微观细节：挖球纹路、杯壁融化线、表面结晶颗粒。

### 德系（German）

- 年轮蛋糕（バウムクーヘン / Baumkuchen, German）
    - 标签：`文化=德式甜点` `风格=年轮层次` `功能=礼盒甜点`
    - 组成要素：多层刷烤蛋奶面糊 + 外层糖衣（可选）。
    - 口感结构：层层紧密、微干中带油润、咀嚼感明显。
    - 微观细节：年轮清晰度、切片断层、糖衣反光与裂线。

### 英语系（English）

- 布朗尼（ブラウニー / brownie, English）
    - 标签：`文化=英语系甜点` `风格=浓郁可可` `功能=便携甜食`
    - 组成要素：可可面糊 + 黄油 + 坚果/巧克力碎（可选）。
    - 口感结构：外层微壳 -> 内层湿润黏密。
    - 微观细节：切边碎屑、中心塌陷、油脂回光、纸衬脱模痕。

- 芝士蛋糕（チーズケーキ / cheesecake, English）
    - 标签：`文化=英语系甜点` `风格=乳酪主导` `功能=下午茶/庆祝`
    - 组成要素：饼底（消化饼）+ 奶酪层 + 顶部果酱或酸奶油（可选）。
    - 口感结构：底层脆松、中层厚滑、尾味微酸。
    - 微观细节：切面平整度、表层回缩纹、冷藏出汗点、饼底碎落。

### 斯拉夫系（Slavic）

- 俄式凝乳饼（シルニキ / сырники, Russian）
    - 标签：`文化=斯拉夫系-俄语区` `风格=家常甜点` `功能=早餐/茶点`
    - 组成要素：凝乳（творог）+ 鸡蛋 + 面粉/粗粒粉 + 平底煎制。
    - 口感结构：外层浅焦壳、内里柔软湿润。
    - 微观细节：煎面金斑、边缘鼓起、酸奶油挂边、糖粉溶点。

- 蜂蜜千层（メドヴィク / медовик, Russian）
    - 标签：`文化=斯拉夫系-俄语区` `风格=蜂蜜层蛋糕` `功能=节庆甜点`
    - 组成要素：蜂蜜薄层饼体 + 奶油/酸奶油夹层。
    - 口感结构：层体吸润后柔软，蜂蜜香后段明显。
    - 微观细节：层纹吸湿、切面压缩、顶部碎屑分布、刀口粘附。

- 拿破仑蛋糕（ナポレオン / Наполеон, Russian）
    - 标签：`文化=斯拉夫系-俄语区` `风格=千层改造` `功能=家庭聚会`
    - 组成要素：酥皮层 + 奶油夹层（俄式家常变体多）。
    - 口感结构：初始酥脆，静置后转为湿润层叠。
    - 微观细节：碎屑雪落感、边角软化差、切片倾斜、盘面奶油拖痕。

## 学科类

### 计算机科学（Computer Science）

- 代码审查会（Code Review）
    - 标签：`文化=现代通用` `风格=协作工程` `功能=质量控制`
    - 关键动作：提交变更 -> 行级评论 -> 讨论取舍 -> 合并决策。
    - 微观细节：审查批注密度、行号跳转、术语冲突、沉默时长。

- 事故复盘（Postmortem）
    - 标签：`文化=现代通用` `风格=工程治理` `功能=风险收敛`
    - 关键动作：时间线回放 -> 根因拆解 -> 行动项分配 -> 跟踪闭环。
    - 微观细节：时间戳对齐、责任措辞、白板箭头层级、任务回执延迟。

- 灰度发布（Canary Rollout）
    - 标签：`文化=现代通用` `风格=运维实践` `功能=发布控制`
    - 关键动作：小流量放量 -> 指标监控 -> 回滚阈值判断 -> 全量发布。
    - 微观细节：告警抖动、仪表盘峰值、回滚按钮触发、值班交接语句。

### 地理信息科学（Geographic Information Science）

- 外业采集（Field Survey）
    - 标签：`文化=现代通用` `风格=实地测绘` `功能=数据采集`
    - 关键动作：布点 -> 采样 -> 轨迹记录 -> 现场校核。
    - 微观细节：RTK天线架设、控制点喷漆、坐标本湿痕、现场返查。

- 投影转换（CRS Transformation）
    - 标签：`文化=现代通用` `风格=空间计算` `功能=数据一致化`
    - 关键动作：识别坐标系 -> 参数设定 -> 批量转换 -> 误差复核。
    - 微观细节：边界错位、比例尺跳变、图层“漂移感”、误差表红字。

- 栅格与矢量叠加（Raster-Vector Overlay）
    - 标签：`文化=现代通用` `风格=空间分析` `功能=模型支撑`
    - 关键动作：预处理 -> 对齐像元 -> 叠加分析 -> 输出专题图。
    - 微观细节：锯齿边界、色带分级、透明度冲突、图例重排。

## 城市运作类

- 供热管网轮转
    - 标签：`文化=寒带/温带城市` `风格=基础设施运维` `功能=民生保障`
    - 关键动作：热源调度 -> 分区平衡 -> 漏点巡检 -> 夜间补压。
    - 微观细节：阀门编号、井盖热雾、管线噪音、抢修照明反光。

- 夜间补货系统
    - 标签：`文化=现代城市` `风格=零售后勤` `功能=供给稳定`
    - 关键动作：后台预测 -> 分拨装车 -> 凌晨上架 -> 异常回传。
    - 微观细节：手持终端滴声、冷链箱凝水、货签更换速度、托盘刮地声。

- 固废收运链路
    - 标签：`文化=现代城市` `风格=公共服务` `功能=卫生治理`
    - 关键动作：分类投放 -> 定点收集 -> 转运压缩 -> 末端处理。
    - 微观细节：桶盖回弹声、压缩车液压节拍、气味分层、清运时间差。

- 雪季除雪机制
    - 标签：`文化=寒带城市` `风格=冬季应急` `功能=交通维持`
    - 关键动作：主干道优先除雪 -> 融雪剂投放 -> 人行道复清 -> 夜间复检。
    - 微观细节：铲刃划地、雪浆飞溅、车灯散射、路沿残雪堆高。

- 停电应急切换
    - 标签：`文化=现代城市` `风格=灾备` `功能=连续运行`
    - 关键动作：故障定位 -> 负荷切离 -> 备用电源接管 -> 分区复电。
    - 微观细节：应急广播延迟、电梯停层、人群回流路径、配电柜温升。
