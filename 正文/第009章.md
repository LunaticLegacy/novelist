# 第009章 纸片雨与双马尾

菲娜这一觉睡得很沉。

她醒来时，窗帘缝里已经是偏白的午后光，墙上的时钟指到十一点四十。她先摸了摸枕边，确认回执卡还在，才慢慢坐起来。昨晚那种“终于过关”的轻松还留着一点，但胸口更明显的是另一种空落落的感觉，像考试这条绷紧很久的线突然断开之后，手臂还记着用力的姿势。

她下楼时，客厅安静得只剩终端风扇的低鸣。蓝月坐在主屏前，黑外套没换，手边放着一杯已经凉透的咖啡。屏幕上同时开着四个窗口：`暗月会 API 峰值`、`命定色盘 v1.2 留存曲线`、`缘 平台实时访问`、`链上关键词分布`。

菲娜走近两步，最先看懂的是那条绿色折线。`缘` 的访问曲线比昨天同一时段低了一截，旁边标注着小字：`-15.2%`。

“你赢了？”她问。

蓝月没回头，手指在键盘上敲了两下，把另一个窗口放大。`暗月会` 的调用峰值还在高位平台，像一段刚上桥的车流。

“不是赢。”他说，“是把第一轮窗口卡住了。”

菲娜嗯了一声，站在他椅背后看了几秒。右上角还有一串滚动提醒：`开发者衍生插件 +214`、`符文生成脚本下载异常上升`、`社区镜像转载增速 3.8x`。她看不懂全部术语，却能看懂蓝月肩线的硬度。

“你到现在都没睡？”

“睡了四十分钟。”蓝月把终端锁屏，终于回头看她，“你先吃饭。我做了蘑菇汤。”

菲娜本来想说“你先去睡”，话到嘴边还是咽了回去。她知道这种时段劝不动，只会浪费彼此力气。她去厨房盛汤，热气扑到脸上时，才后知后觉地觉得饿。两人坐在餐桌两侧，谁都没提昨晚的倒计时，只聊了几句很日常的话：冰箱里牛奶快没了，晚点要补；蕾雅早上发了十几条语音，全是“通过快乐”；诺亚在班群里连发三张数据图，像在庆祝自己活过考试周。

吃到一半，菲娜把勺子一放：“我下午去星芒馆。”

蓝月点头：“可以。练基础，不碰越级，不接陌生人给的现成符文。”

“我看起来像那么好骗吗？”

“你看起来像会被‘看起来很专业’骗。”

菲娜瞪他一眼，起身去拿外套。出门前她又折回来，把桌边那杯冷咖啡端走，重新换了杯热水放到蓝月手边。蓝月看她一眼，没说谢谢，只抬手捏了捏她后颈，力道很轻。

“早去早回。”

“知道啦，监护人。”

去星芒馆的路上，她坐在轻轨靠窗位，车厢里一半人在低头刷终端。一个男生把屏幕横过来给同伴看，界面上密密麻麻全是刚发布的衍生工具，标题一个比一个夸张：`一键风格迁移`、`情绪色板自动补全`、`零门槛符文描边器`。旁边女生笑着说“这个描边器昨晚就下架了，听说有人拿去跑实操”，另一人接话“下架了又有人做镜像，拦不住”。

菲娜把视线转回车窗。玻璃里映出她的脸和后排广告屏的反光，反光上写着：`暗月会开发者挑战赛倒计时 47:13:22`。她忽然有种奇怪的错位感，好像自己刚从一场长跑下场，整个赛道却在她背后原地扩建了两圈。

到站后，她又收到蕾雅消息，连续三条。

`你醒了没！`  
`今晚逛街吗！`  
`我抢到缘的新徽章了！！！`

菲娜回了个`晚点再说，我去练馆`，把手机调成静音塞回口袋。她知道蕾雅现在兴奋，也知道自己现在不适合被这种兴奋带节奏。对她来说，考试过线只是把“会不会挂”换成了“接下来怎么走”，不是放空。

星芒馆下午人不算多。大半是刚考完试来“松一口气”的学生，训练区里时不时传来低阶术式的起落声，像一阵一阵拍在墙上的雨。菲娜租了靠窗的练习位，先做了两轮热身，再把一盒纸片撒到半空，起手拉线性气流。

第一轮她故意压低输出，只做轨迹；第二轮再提一点强度，让纸片在固定椭圆内循环。纸片升起、回落、再升起，像一团不肯落地的浅色雪。

她刚收杖，侧后方忽然响起一阵清脆的掌声。

“好稳啊！”

菲娜回头，先看见一双很亮的琥珀色眼睛，再看见那人脑后扎得很高的双马尾。女孩年纪和她差不多，穿着偏地雷系的黑白裙装，领口挂着一枚细小龙纹牌，耳侧有一段半透明的角质装饰，在馆灯下反出浅金光。

“抱歉，我不是故意偷看。”女孩举起双手，“就是觉得你那条线……像拿尺子拉的。”

菲娜把杖插回腰侧：“谢谢。你是新来的？”

“嗯，林小雨，龙族交换生。”她往前一步，笑起来有点快，“我今天第一次来这馆，想找人对练。”

菲娜听到“龙族”两个字，下意识多看了她两秒。不是警惕，是纯粹好奇。林小雨却像习惯了这种目光，已经熟练地从包里掏出平板，点开一张图递过来。

“你看，这是我中午用`暗月会`跑的草图，龙族旧式法阵改编版。线条还行吧？”

屏幕上是一张复杂的环阵图，外圈古拙，内圈却是很现代的简化回路，配色冷白带蓝，确实漂亮。菲娜盯着看了两秒，把平板推回去：“图挺好看。你拿它实操过吗？”

林小雨眨眨眼：“还没，想先找人一起校对。你也用AI辅助吗？”

菲娜脑子里几乎同步闪出蓝月的那句“看图可以，实操自己来”。她没正面回答，只耸了耸肩：“偶尔看。真上手还是靠自己。”

“懂，怕翻车嘛。”林小雨把平板收好，眼神却更亮了，“那要不要打一局？我想试试你这种现代路数。”

十分钟后，两人站到了对练线两端。

林小雨的起手和学院派明显不同，先立符，再导流，动作不快但很沉，像把每一步都钉在地面上；菲娜则走轻快节奏，先抢轨迹，再补边界，靠连续小修正拿稳定。两人第一轮互相试探，第二轮都提了半档，场中风压和火星来回切，观战区很快围上来一小圈人。

到第三轮末段，林小雨用龙纹回路压出一道低频气墙，菲娜正面不硬顶，直接侧切，把风线改成双轨，让纸片从气墙两端绕过去。两边都没崩，都没吃满，对练时间到点时刚好收在平手。

“舒服！”林小雨把杖一甩，喘着气笑，“你这个转轨我得学。”

“你那个低频气墙也挺烦人。”菲娜接过馆员递来的水，“下一次你要是再压一拍，我可能就被你卡住了。”

“那下次继续。”林小雨很自然地掏出通讯码，“加个好友？我平时也会在`暗月会`看法阵改图，想找人讨论。”

菲娜犹豫了半秒，还是扫了码。她不是不想交朋友，只是这阵子“讨论”两个字总和“测试”“桥接”“直上实操”绑在一起，让她本能地多一道刹车。

“先说好，”她收起通讯器，“聊图可以，别拉我做越级实验。”

林小雨把手按在胸前，表情很认真：“我有龙族施法许可，不代表我想把自己炸飞。放心。”

两人一边收杖一边又聊了几句。林小雨提到自己刚来中枢之地，住校外公寓，白天上课晚上会去社区馆补练；还提到她在龙族本地论坛上见过“命定色盘”的讨论，很多人觉得这是“把传统符文变成快餐”，也有人觉得“这是门槛降低，年轻人终于不用抄老本子抄到手抖”。

菲娜听着，没急着表态。她只在告别前说了一句：“工具会变快，人最好别跟着失控。”

林小雨冲她比了个“收到”的手势，转身时双马尾在馆灯下甩出一道短弧，看起来轻快得像没负担。菲娜站在原地看了她背影两秒，才把手套塞进包里。她意识到，自己已经开始本能地给每个新关系加“安全条款”，像蓝月那样先划边界再谈合作。

离开星芒馆时，天色刚转浅橙。菲娜沿着商业街慢慢往家走，路过甜品店时想起蓝月中午那杯冷咖啡，脚步一拐，进去买了两个泡芙。店里电视正播短资讯，主持人语速飞快：

“……`暗月会`上线后，开放接口引发开发者热潮，相关衍生工具一日新增超两百款。与此同时，`缘`平台宣布将于今晚推出新版内容确权规则，回应外界对流量下滑的质疑……”

菲娜提着纸袋出来，站在门口听了两秒。街对面的广告屏正在轮播命定色盘v1.2宣传，画面里银蓝色光带从城市上空掠过，下面是醒目的口号：`每个人都能生成属于自己的风格。`

她忽然想起下午那盒在空中盘旋的纸片。有人把它当练习，有人把它当内容，有人把它当生意，有人把它当武器。东西还是那个东西，落地方式却一夜之间多了很多种。

回到家时，蓝月还在客厅终端前。菲娜把泡芙放到他手边：“一个你的，一个我的。”

蓝月看了眼包装：“你今天心情不错。”

“遇到个新同学，龙族的，叫林小雨。”菲娜脱外套，顺口把对练过程说了一遍，说到`暗月会`草图时，她停了停，“她问我用不用AI辅助。”

“你怎么答的？”

“我说看图可以，上手自己来。”

蓝月终于从屏幕上移开视线，抬手在她头顶按了一下：“合格答案。”

菲娜坐到沙发另一头，咬着泡芙，奶油甜味在舌尖化开。她盯着蓝月屏幕上不断跳动的折线，忽然开口：

“欧尼酱。”

“嗯。”

“你说，为什么现在大家都这么快？昨天还在讨论考试，今天就在讨论插件、接口、榜单、规则战。像谁慢一步就会被丢下。”

蓝月沉默两秒，指尖把一条异常峰值圈出来，才回答她：

“因为很多人不是怕慢，是怕不被看见。能被看见，就会继续跑。”

菲娜没马上接话。她把最后一口泡芙吃掉，靠进沙发，盯着天花板发了一会儿呆。窗外夜色一点点落下来，街屏还在亮，平台的战斗也还在亮。

过了一会儿，她又偏头看蓝月：“林小雨这人，你觉得有问题吗？”

蓝月把她下午发来的通讯码截图调出来，几秒后关掉：“目前看，没有。龙族施法许可是真的，学习轨迹干净，社交圈也很普通。”

“你又查我朋友。”

“我查的是风险，不是你朋友。”蓝月语气很平，“你可以交朋友，但我不能让你在我不知道的风险里交朋友。”

菲娜本能想反驳，话到嘴边又停住。她知道他不是故意控制欲发作，而是过去一周的节奏把他的“预防”阈值抬高了。她抿了抿唇，最后只说：“那你至少提前告诉我你会查。”

蓝月点头：“好，下次先说。”

这句“先说”让空气松了一点。菲娜把脚缩上沙发，抱着膝盖，听终端风扇转动。她忽然意识到，自己和哥哥也在学新的相处规则：以前是“你照顾我”；现在变成“你照顾我，但我要知道你怎么照顾我”。

她的考试线结束了。

新的线，已经起笔。

不吵。

但很快。
