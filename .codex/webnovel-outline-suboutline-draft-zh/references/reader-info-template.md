# 读者面信息模板

## 使用规则
- 每章写作前先做“写前规划”，明确本章给读者哪些信息、保留哪些信息。
- 每章完稿后做“写后回填”，记录实际投放的信息与悬念。
- 信息控制原则：少量、关键、可回收，避免设定堆砌或连续空悬念。
- 先确认“全局设置”，再决定本章信息预算；若本章临时覆盖全局设置，写后必须记录并决定是否回滚。

## 读者负荷自检
- [ ] 本章新增硬设定不超过 2 条。
- [ ] 本章至少有 1 条有效信息推进（世界观/角色关系/主线因果其一）。
- [ ] 本章若新增悬念，已写明预计回收章节。
- [ ] 没有连续大段说明性文字挤占剧情行动。
- [ ] 本章信息预算与“全局设置”一致；如不一致，已填写“本章全局设置覆盖”。
- [ ] 本章若修改全局设置，已记录在“全局设置变更日志”。
