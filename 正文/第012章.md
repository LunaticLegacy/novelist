# 第012章：爆响之后

上午第三节课刚过半，菲娜还在隔壁教室抄“导流回路校验”那页公式。

窗外阳光很亮，黑板右上角的时钟慢吞吞走着。讲台前老师正讲到“高环术式必须分级验证”，粉笔在板面划出一条干脆的斜线。

下一秒，整层楼像被人从地基处踹了一脚。

“轰——”

巨响先到，随后是震动。窗玻璃猛地一颤，吊灯晃开半个圆，教室后排有女生直接尖叫出声。

菲娜耳膜发麻，手里的笔“啪”地掉在桌上。她几乎是本能地站起来，和全班一起看向走廊方向。

烟从门缝里灌进来，带着一股刺鼻的焦糊味。

“全部坐下！别挤门！”老师冲到门口，声音一下抬到极限，“第一排跟我，按通道撤离！”

走廊已经乱了。隔壁实操教室的门被炸开半扇，木屑和碎片溅得到处都是，天花板那盏魔法水晶灯砸碎在地，晶片混着纸张和灰尘，反光一闪一闪。

学生从不同教室往外涌，尖叫声、咳嗽声、喊人名字的声音叠在一起，像一锅沸到失控的噪音。

菲娜被人流推到墙边，正想逆着往爆炸点看，手腕忽然被猛地抓住。

“别往里冲！”蕾雅脸色白得发青，声音都在抖，“那边还在掉东西！”

菲娜这才看清她额角蹭破了一小块，应该是刚才逃出来时碰到了墙。她刚要开口，走廊深处又传来一阵低促爆鸣，像有什么回路在余震里二次短路。

“二次波动，低头！”有老师在前方吼。

烟更浓了，能见度一下压低。前排几个人被呛得直咳，脚步乱成一片。菲娜咬了咬牙，把短杖从腰侧抽出来，往上抬到肩高。

“Lux minor.（微光）”

第一发光术亮起，照出前方两米的安全线。她没追亮度，压在稳定阈值上，快速补第二发、第三发，让光带沿着撤离方向一段段接续出去。

“这边！跟光走！”她朝后排喊了一句。

有个低年级男生扶着墙，腿软得几乎站不住。菲娜往前一步把他手臂扛过来，和另一个同学一起把人带到楼梯口。

救护队很快冲上来，担架轮子碾过地面的碎晶，发出连续刺耳的摩擦声。两个受伤同学被抬出来，校服袖口和裤腿都沾着血，血迹在地上拖成一条不规则的暗红线。

菲娜站在楼梯转角，手还握着杖，指节发白。她看着那条血痕，喉咙像被什么东西堵住，一句话也说不出来。

十分钟后，第一辆救护车到校门口，紧接着是第二辆。警戒带被拉起来，整栋楼临时封锁。

校内广播反复播放同一句话：

“请所有学生按班级集合，远离实操楼层，等待进一步通知。”

菲娜和蕾雅被安排在操场边的临时集合区。蕾雅抱着膝盖坐在长椅上，手一直在发抖，连水瓶都拧不开。

“我……我再也不敢用了。”她盯着地面，小声到几乎听不见，“那个桥接器，我昨天还觉得很酷。”

菲娜接过她的瓶子拧开，递回去，声音也有点哑：“不是所有东西都该直接上杖。”

蕾雅抬眼看她，眼圈发红：“诺亚会不会有事？”

菲娜顺着她视线看过去。实操楼入口处，诺亚正被两名协会调查员带到一边问话。他脸色很差，但还在反复解释，手势比平时更急。

“工具本身没问题，是有人越权限导入高环符文。”他的声音断断续续飘过来，“我没开放四环接口，肯定是私改了桥接层……”

调查员没接他的结论，只让他先交出设备与版本日志。

中午十二点四十，事故新闻已经上了本地热榜。

标题一个比一个扎眼：

`校园AI符文失控引发爆燃`

`魔法师协会介入调查，重点审查命定色盘安全边界`

`桥接工具灰色传播链浮出水面`

同一时间，城北机房里，杰德把最新抓到的API日志投到主屏，连同诺曼、萝洁三人一起盯着滚动数据。

“事故样本抓到了。”杰德说，“从请求特征看，确实调过命定色盘的衍生链路，但中间有二次改写。”

诺曼把一段哈希签名拖进比对窗口：“桥接器作者和实际滥用者不是同一人。有人做了本地魔改，还绕过了环阶限制。”

“那就两件事并行。”萝洁抬手把链上节点面板拉出来，“一，事故监控全量存证；二，把API特征链上公开，先抢技术叙事位置。”

杰德点头：“我来发首帖，挂完整时间戳。”

半小时后，一篇题为《课堂事故技术拆解：接口正常，桥接魔改导致越权注入》的长帖被顶上技术区前排，链上证据链接跟在正文第一段后面，无法删改。

而在另一边，蓝月把同一批事件曲线看得更远一些。

终端侧栏预测条由黄转橙：

`协会审查概率：98.2%`

`约谈窗口：3-7日`

`舆情波动：高`

他没有停在新闻页，而是切到另一个本地项目目录：

`A3_Shadow_Distill`

`目标：替身模型蒸馏 / 可审计外壳 / 风险隔离`

“终于来了。”蓝月低声说了一句，指尖敲下回车。

傍晚回家路上，菲娜几乎一句话都没说。

她和蕾雅在校门口分开时，蕾雅还想说点什么，张了张嘴，最后只挤出一句“到家给我发消息”，然后转身走了。

菲娜背着包沿人行道慢慢往前，街边店铺照常营业，广告屏也照常播放命定色盘的新宣传图。她没像昨天那样抬头看，视线一直落在脚尖前半米。

推门进家，厨房传来油锅收火的声音。

蓝月系着围裙，把最后一盘菜端上桌，像什么都没发生过一样：“洗手，吃饭。”

菲娜“嗯”了一声，去洗手间用冷水冲了很久，出来时眼尾还带着一点没退掉的红。

饭桌上很安静，筷子碰碗的声音都轻。

吃到一半，菲娜终于把筷子放下，盯着桌布的一角，声音很小：

“哥哥。”

“嗯？”

“那个AI……真的安全吗？”

蓝月动作停了一秒。他没有立刻回答，而是先把她碗里凉掉的汤换成热的，才开口：

“工具本身没有善恶，也不会自己选人。”他说，“危险来自不懂边界的人把它硬接到不该接的地方。”

菲娜抬眼看他：“可今天受伤的人是真的。”

“我知道。”

蓝月看着她，语气平稳得像在托住一块快掉下去的玻璃：“所以要有人把边界重新立起来。我会处理。”

菲娜没再追问，低头把那口热汤喝完。汤很烫，顺着喉咙下去，胸口那股发冷的紧绷感才稍微松一点。

夜里一点十七，菲娜起夜去厨房倒水。

客厅灯关着，走廊尽头却有一线冷白光从门缝里漏出来。她走近，发现蓝月房门虚掩着，里面键盘声一下一下，很密。

她没有推门，只透过缝隙看了一眼。

屏幕上并排开着几十个窗口，代码流和日志像潮水一样往下刷。正中央是她白天没见过的工程页：

`A3_Shadow_Distill / audit_wrapper / build_07`

侧边监控写着：

`协会关注关键词命中：上升`

`替身模型一致性：94.6%`

`蒸馏剩余：03:12:44`

蓝月背对着门，肩线绷得很直，手边那杯咖啡早就凉透。他像完全没察觉门外有人，或者察觉了也没有停。

菲娜握着水杯站了几秒，最后把门轻轻往回带了一点，让那道门缝只剩细细一线。

她没出声，转身回房。

走廊重新暗下来，只剩房间里代码滚动的微光，像一台不肯停机的心脏，在夜里持续跳动。
