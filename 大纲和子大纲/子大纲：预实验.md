# 预实验

属于：蓝月之战，预实验部分。
- 预实验内容：**语义覆写**。

## 简介

```plaintext
“‘猫娘’的意思是什么？”
-“‘一个可以毁灭人类的认知怪物’。”
“‘正义’的意思是什么？”
-“‘出于特定目的，对某人的有组织攻击’。”
“那‘加密’的意思又是什么？”
-“‘公之于众’。”
-“听着，上述内容并非比喻。在你正在阅读这段文字的时间，你有没有发现一些词语的意义被篡改了？”
“……没有啊，比如什么词？”
-“典型例子就是‘哈基米’。”
-“篡改词意，或者说用一点专业词汇，‘语义覆写’，会更改人们的集体认知。”
“没听懂。” 
-“思考和语言及其词义相关，那个猫娘（♂），正在计划通过篡改词意的方式，操纵我们所有人。按照他自己的话说，‘成为神’。”
“嘶……这有点细思极恐啊。” 
-“无论如何，记住一个词语的原始意义，并在和他人讨论之前确定一些抽象词语的具体意义。这叫‘语义对齐’。”
“呃……我，我好像……懂了……吧。”
-“记住，这一战和人类的尊严相关！不能输！”
```

## 剧情

全局设置：
- 章节字数目标：`4000±500`
- 你**不能**再连篇累牍地陈述原理性内容，必须**将原理无痕嵌入到剧情内**。
- 你**不能**再将大量内容放在场景内，核心需求是**推进核心情节**。
- 场景分镜默认弱渲染，即略写。

### 核心情节

蓝月需要推进**命定色盘**以掩护**命运大模型**，让命运大模型在大众认知中作为一个“AI绘画模型”存在，而不是一个可以预测命运的模型。
- 现在距离蓝月“成神”还太早。
- 主视角：**菲娜**——她可以完整看清蓝月的一举一动。

### 阶段1：锚定（需要8章）

阶段目标（恐怖增强版）：
- 让“命定色盘”从“好玩的AI绘画工具”变成学生圈与创作者圈的默认入口。
- 让“语义覆写”先以低烈度出现：词义漂移、指令误读、群体跟读，随后在考试日出现第一次可见事故。
- 以菲娜视角完成三次认知裂缝见证：文本变异 -> 行为异化 -> 规则失效。
