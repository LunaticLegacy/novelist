# 第007章：风过林线前

夜里十一点，菲娜房间的台灯还亮着。

她把最后一页笔记写完，笔尖在句号处停了两秒，像给自己按了一个小小的结束键。手机闹钟已经设到明天早上六点，书包也按考试清单重新整理过：准考证、短杖、替换电池、纸巾、糖片，整整齐齐。

门外却还没有“结束”的意思。

蓝月房间的光从门缝里漏出来，冷白色，一直没灭。菲娜路过去倒水时，看见他正对着三层投影窗口改参数，指尖在半空一点一点拖拽，尾巴尖绷得很直。

`命定色盘 mobile: 编译中`

`目标: 低延迟 / 考试安全`

`进度: 91%`

菲娜端着杯子站在门边：“欧尼酱，你又开夜车。”

“最后一轮。”蓝月没回头，“你明天考试，今天得把移动端特征包压完。”

“特征包？”

“给你用的安全模板。”他切出另一页，把参数面板侧过来给她看，“火术抖动抑制、光术阈值保护、导流节奏提示。不是外挂，是防你在紧张状态下手抖过载。”

菲娜看着界面里那行`Fina_Exam_Profile`，眼睛眨了眨：“你这是临时给我做了个‘考试版我自己’？”

“差不多。”

“听起来有点可怕。”

“比你在考场上炸术式可爱。”

菲娜被噎了一下，最后只好抱着水杯小声嘀咕：“你这人安慰方式真烂喵。”

“谢谢评价。”

“我没夸你！”

“我知道。”

熟悉的句式让她差点笑出来。她把杯子放在他桌角，正要回房，蓝月突然关掉一层窗口：“别睡太死，明早我带你去一趟林场边。最后复习。”

“明天考试还去？”

“只做短时实操。你需要手感确认，不是再背定义。”

菲娜点头：“好。”

她回房关灯，躺下时才发现自己心跳有点快。不是害怕，是某种临考前特有的空悬感，像你已经站到跳台边缘，脚下还有半步就会离地。

手机又震了一下。

蕾雅：

“我刚在‘缘’上抽到‘必过金’！明天你一定行！！！”

“还有，诺亚那个怪物把命定色盘符文生成做成快捷按钮了，我看不懂但我大受震撼。”

菲娜盯着消息看了几秒，回了个“早点睡”，然后把手机扣到枕边，闭眼。

窗外风掠过高架，像很远的潮声。

第二天清晨六点半，车已经离开城区。

灰色公路一路往外伸，建筑密度越来越低，最后只剩林场边缘的防风林和几段废弃训练场。蓝月把车停在一块碎石空地，后备箱打开，里面并排放着两根魔杖。

一根是菲娜平时用的铝合金学徒杖，细、轻、硬，像一根被规则磨得没脾气的金属尺。

另一根是蓝月常年收着不用的旧杖，深灰杖身上有细密银纹，杖尾缀着一小圈磨损过的护环，安静得像一截冬夜。

“先用你的。”蓝月说，“按考试标准来。”

菲娜应声，站进他划好的防护圈里。今天风不大，空气干，适合做细导流。她先起手风术，接火术，再补光，三连一套做得比昨天更顺，唯一的问题还是火术尾段有轻微抖动。

“火术稳定，八十一。”蓝月看着记录板，“过线，但不漂亮。”

菲娜吐了口气：“我也感觉到了，最后收束总差一点。”

“换杖。”

蓝月把那根旧杖递给她，动作很自然，像把一把旧钥匙交还给原本的锁。菲娜接过那一瞬间，指尖先感到一层很轻的凉，随后是明显更顺滑的导流反馈，像水终于找到低处。

“这是你以前常用的？”她问。

“嗯。你先别分心，测手感。”

菲娜点头，再起手。

同样的风术起式，杖尖几乎不用额外修正就走到她想要的轨道；火术点燃时，她能清楚感到热源像被一只看不见的手托住，没再乱窜；光术收尾也更干净，亮而不刺。

她自己先愣了一下：“这差距也太大了。”

“学徒杖是统一标准，不是最优解。”蓝月语气平稳，“考场不能用这根，所以你要学会‘知道什么是好手感’，再反推回铝杖怎么补偿。”

菲娜握着旧杖，慢慢点头：“懂了。先知道满分答案长什么样。”

“对。”

短暂休整后，蓝月把手机递给她，屏幕是“缘”的简化训练页，顶部标着：`命定色盘 考试可用版`。

“最后一项。”他说，“余火。”

菲娜呼吸顿了半秒。

余火是1环术，理论不难，难在稳定维持。上一次她在星芒馆看到蕾雅放出来一圈火星，漂亮归漂亮，失控也只隔一层纸。

“目标多久？”她问。

蓝月看了眼计时环：“四分钟。”

“……你是魔鬼吧。”

“你可以拒绝。”

“我不拒绝喵。”

她把手机放到支架上，屏幕里的辅助提示随呼吸频率微微闪动。她先用铝杖试一次。

“Reliquias Igni.（余火）”

细小火星从杖尖散开，前四十秒还算稳，到第五十秒开始偏，九十秒后明显掉亮，最终在一百二十秒附近自行散掉。

菲娜皱眉：“又是这个坎。”

“记住刚才掉亮前的手腕角度。”蓝月说，“换旧杖，再来。”

第二次起术，她按命定色盘的节奏提示压慢导流，在第三十秒主动做了一次微调。火星群比第一次更密，颜色更干净，时间一路过两分钟、三分钟，仍在稳定。

她额角冒汗，后背已经被风吹得发凉，手却没抖。

“三分三十。”蓝月报数。

“三分四十。”

“三分五十。”

“……四分钟。”

计时环亮起结束标记的瞬间，菲娜才敢收术。散开的余火在空气里像一把慢慢熄灭的星砂，落得很轻。

她站在原地喘了两口气，抬头看蓝月，眼睛亮得有点发颤：“我做到了。”

“嗯。”蓝月走近，抬手替她擦掉眼角那点汗，“余火，四分钟整。有效稳定。”

他说这话时表情还是那副平静样子，但耳尖轻轻往后压了一下，尾巴尖在地上极小幅度地晃了一次。菲娜太熟悉这个动作了，那是他版本的“不错”。

“你在偷夸我喵。”

“我在记录事实。”

“翻译一下就是：菲娜很厉害。”

“翻译过度。”

“当事人有权二次解释喵。”

蓝月没跟她争，只把记录板递过去。三项数据排在一行，火与光都过了昨日上限，余火维持时间后面标着粗体数字：`04:00`。

菲娜盯着那串数字看了几秒，把记录板贴在胸前轻轻拍了一下，像拍一枚护符。

收场回程时，太阳已经爬到树顶。兄妹俩沿着林场外的辅路往回走，路过昨天还闹腾的线下团建点。今天那边正在收摊，折叠桌一张张合上，横幅卷起一半，剩下“守护色”三个字被风吹得一鼓一鼓。

两个志愿者正抱着纸箱聊天。

“昨天人超多，今天都去复习了。”

“明天考完估计又爆。”

菲娜听见这句，脚步慢了一拍，又很快恢复。她把手机从口袋里拿出来看了一眼，蕾雅果然又发来十几条消息，最上面一条是：

“我在做‘考前最后祈福图’，你上线看不看？”

菲娜想了想，回：

“晚上看。现在先把手感锁住。”

蕾雅立刻回了三个感叹号和一个“你是卷王本王”。

蓝月在一旁看见她聊天界面，问：“紧张吗？”

“有一点。”菲娜把手机收好，抬眼看前方，“但比昨天少了。”

“为什么？”

“因为我现在知道自己能做到了。”她晃了晃手里的记录板，“不是靠运气，是靠练出来的。”

蓝月点点头：“那就够了。”

上车前，菲娜忽然想起什么，回头冲他伸手：“欧尼酱。”

“嗯？”

“耳朵，借我摸一下，考试前buff。”

蓝月沉默一秒，耳尖微抖：“五秒。”

“成交喵。”

她飞快摸了一下就撤手，动作轻得像碰一片羽毛。蓝月无奈地看她一眼，替她拉开车门：“上车，回家吃饭，下午背错题，晚上早点睡。”

“你这行程安排得像作战计划。”

“明天本来就是战斗。”

菲娜坐进副驾，把记录板小心塞进书包最上层，拉链拉好时，手指还是稳的。

车驶回城区，路边屏幕正轮播“缘 x 命定色盘：画出你的守护色”，色块一闪一闪地掠过窗面。菲娜看了一眼就收回视线，低头翻开错题册，在第一页写下明天的目标：

`稳定发挥，先过，再高分。`

墨迹落下时没有一点抖。
