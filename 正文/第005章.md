# 第005章：屏幕里的守护色

车停进小区地库时，天已经彻底黑下来。菲娜抱着书包下车，跟在蓝月后面进电梯，电梯镜面里两个人都很安静，一个眼下泛青，一个额角还残着训练后的疲色。

“先吃饭，再复习。”蓝月按下楼层键，语气跟平时一样平，“今晚只做三轮，不拖到凌晨。”

菲娜抬头看他：“你上次也这么说，结果做了五轮喵。”

“上次是你自己加题。”

“……那我今晚努力做个人。”

回家后，餐桌上是温过的番茄牛肉和烤吐司。菲娜吃得很快，吃完就把碗往水槽一放，抱着《魔法师必考300问》往房间冲。蓝月没拦，只在她身后提醒：“四十分钟后出来活动三分钟。”

“收到，主考官。”

门一关，屋里只剩翻页声和笔尖摩擦纸面的细响。

第一轮她还能稳稳跟住节奏：风术定义、火术导流、光术安全阈值，逐条背，逐条写。第二轮开始，字迹就有点飘了，某些术语明明认识，却像隔着一层毛玻璃，读得出轮廓，抓不住边角。

桌角的手机亮了一次，又亮一次，再亮一次。

她没看。

直到第五次震动连着弹出，屏幕上跳出蕾雅的头像，后面跟着一串堪称精神污染的消息：

“菲娜！我又生了三张雪猫护体图！！！”

“这张是‘火术不抖红’，这张是‘笔试不空白蓝’，这张是‘考官看了都点头紫’。”

“转发必过，不转也过，但转了更有仪式感。”

菲娜盯着通知栏看了两秒，还是把手机扣过去，继续背“热源锚定三原则”。

十分钟后，她写到“热源反馈抑制”那一行，脑子里突然先蹦出蕾雅那句“考官看了都点头紫”，笔尖在纸上停住，差点笑出来。

笑完更烦。

她抬手揉了揉太阳穴，强迫自己把注意力拉回纸面，可字行开始左右晃，像在故意跟她作对。她硬撑到第三轮中段，终于“咚”地一声把额头磕在了摊开的笔记上。

“……不行了。”

她趴着没动，呼出的热气把纸页吹得微微翘起。走廊里传来很轻的脚步声，门被敲了两下。

“菲娜，活动时间。”

“我还活着喵……”

她拖着步子去开门，头发乱成一团，眼底写满“我被定义击败了”。蓝月站在门外，手里拿着一杯温水，见她脸色发木，把杯子递过去：“先喝。去洗澡，回来再看最后一轮。”

“你这个流程像在修设备。”

“你现在确实在过热。”

菲娜接过杯子，边喝边往浴室走。路过蓝月房间时，门开着一条缝，里面冷色投影一层压一层，像一整面由数据拼起来的墙。

她本来只想瞥一眼，却被最上面的两条曲线钉住了视线。

`命运大模型 关键词命中：持续下滑`

`命定色盘 相关热度：稳步上升`

蓝月没有回头，手指在空中点了一下，把曲线切到另一页。菲娜只来得及看到“正向情绪占比”几个字，就被蒸汽从浴室门里扑出来的热雾推着走开。

“偷看机密是违法喵。”她小声嘀咕。

“你看到的是民用面板。”蓝月在身后回她，“真机密你看不见。”

“你最好是。”

热水冲到肩上的那一刻，菲娜才感觉脑子慢慢解冻。她在雾气里闭着眼背了两遍火术导流，结果背着背着又跳到“考官看了都点头紫”，自己先笑了。

洗完出来，头发还滴着水。她裹着毛巾回房，先把台灯关掉，打算让眼睛歇五分钟，结果房间一下暗下来，只有手机屏幕在被子边缘发出冷白色的光。

她坐到床上，把被子拉到腿上，鬼使神差地点开了“缘”。

原本她只想看五分钟。

瀑布流第一屏就是“考前守护色挑战”。

第二屏是“今日命运色卡：专注蓝”。

第三屏是“学习打卡 Day 1：背完三章送自己一只电子雪猫”。

再往下滑，全是同类内容：有人晒笔记本边角贴着命定色盘生成的小图；有人把今日运势配色做成桌面；有人在评论区许愿“火术别炸、光术别晕、笔试别卡壳”。画风很杂，语气很闹，却莫名其妙地有种集体备考的同频感。

菲娜的拇指停不下来，一条接一条往下滑。

通知栏里，蕾雅还在疯狂刷屏：

“我把你头像P进护符了，超灵！”

“你看这只猫耳少女像不像你（只像三成）。”

“快来点赞，我要冲‘今日学习区热帖’！”

菲娜被她闹得没脾气，点开聊天窗口，终于回了一句：

“你是住在APP里了吗喵。”

对面秒回：

“是。以及你终于肯刷了！！！”

“我只是休息五分钟。”

“你骗鬼。你现在肯定已经滑了二十条以上。”

菲娜没回，默默数了一下，三十七条。

她把手机举远一点，盯着那些“考前好运”“守护色打卡”的标签，忽然发现自己刚才那股被题目追着跑的窒闷感淡了不少。像有人在她旁边并排坐下，告诉她“你不是一个人在熬夜背书”。

这感觉很轻，很软，甚至有点幼稚。

但有效。

她给蕾雅发来的“火术不抖红”点了个赞，又把其中一张存进相册，顺手设成了备考相册封面。

“就当心理暗示。”她对自己说。

另一端的夜里，诺亚把耳机挂在颈后，手边三块屏同时亮着。一块是Rust工程，一块是命定色盘API文档，一块是他刚写的自动化脚本日志。

`batch_generate(128) -> ok`

`seed persisted -> ok`

`style profile mismatch -> warn`

他盯着`warn`看了两秒，皱眉，不是因为画面丑，而是因为接口字段命名不统一。比起“画了什么”，他更在意“为什么这个字段叫palette_id而不是tone_id”。

“UI团队在犯罪……”他低声吐槽，手指又敲下一串修补代码。

另一边，萝洁登录了“缘”的测试端。

她没走常规用户路线，而是连续发了三组试探内容：标题里塞“命运大模型”，prompt里嵌“预测”“原始权重”“Lunae Caeruleus”，配图全部伪装成普通二次元插画。

第一组，发布成功，搜索权重极低。

第二组，标题被轻度改写，关键词自动替换成“命定色盘相关创作”。

第三组，评论区可见，广场分发近乎为零。

她盯着结果，嘴角一点点压平：“不是删帖，是降噪埋没。”

诺曼从语音里问：“还能测出边界吗？”

“能，粗边界能摸到。”萝洁把数据打包丢过去，“但它在学我，每次都晚一步。”

“够了。”诺曼说，“只要能留下样本，就还有路。”

蓝月的房间里，风扇和投影的轻响交织成一层白噪。他把两个窗口并排放到最大：左边是拦截日志，红字一条条滑过；右边是热度曲线，蓝线稳稳抬升。

`命定色盘 #考前守护色 互动量：+18%`

`学生圈正向情绪：63% -> 71%`

`命运大模型相关讨论：持续低频游离`

他又调出一个更细的分布图，看到“学习打卡”“考试互助”“守护色二创”三个标签开始互相导流，形成小闭环。尾巴尖在椅脚旁停了两秒，终于轻轻一摆。

“锚定……初步成功。”

这句话很轻，像说给屏幕听。

他没有立刻收工，而是把明天的素材排期再向前推了一格：凌晨低频推送不做强刺激，只保留“陪伴感”与“打卡感”。

处理完后，他起身去走廊倒水。经过菲娜房门时，他停了一下。

门缝里透出一点屏幕光，不刺眼，像夜里一小块安静的月色。

他没敲门，只在门外站了两秒就离开。

房间里，菲娜裹着被子靠在床头，手机还停在“缘”的瀑布流界面。她已经给自己定了最后通牒：再看三条就关。

第一条是“明日命运色卡：晨光橙”。

第二条是“背不动了就来这里打个卡”。

第三条是蕾雅发来的新图，配文依旧离谱：

“如果你现在还在背书，这只雪猫就在你旁边监督你。”

菲娜盯着那只圆滚滚的冰晶猫看了几秒，终于笑了。

她按下锁屏，把手机扣在枕边，重新拉开台灯，翻回那页差点把她压趴下的定义。

这一次，第一行字没再晃。

窗外很安静，偶尔有夜车从高架滑过。屏幕熄灭前最后一秒，通知栏短暂亮起一行系统推送：

“今日守护色已保存，明天也要加油。”

菲娜没看见。

她正低头写下今晚最后一行笔记，字迹比一个小时前稳得多。
