# 第009章：旧息与新光

上午九点零七分，菲娜的手机在桌边连续震了两次。

第一条来自学院系统：

`初级魔法师资格认证：通过`

`综合评定：优秀`

第二条来自班级群，内容更直接：

“菲娜，恭喜上岸！今天没课，下午星芒馆见不见？”

她盯着“通过”两个字看了好几秒，才后知后觉地笑出来。昨天那口一直吊在嗓子眼的气，到这会儿才算落地。

餐桌另一边，蓝月正在看终端。冷白光打在他侧脸上，把眼下那点青影照得更明显。

`暗月会 24h 调用：1,284,903`

`命定色盘 API 占比：79.1%`

`缘 平台流量变化：-15.3%`

`开发者衍生插件新增：412`

菲娜一边咬面包一边探头：“你这表情，像刚赢完仗又要打下一场。”

“差不多。”蓝月滑走那几行数据，抬眼看她，“今天放松可以，别乱试陌生符文。”

“我去星芒馆练基础，不乱来喵。”

“还有一条。”蓝月把牛奶推过去，“别人给你的‘现成模板’，看得懂再用。看不懂，就当没看见。”

菲娜点头：“收到，欧尼酱。”

她换好衣服出门时，蓝月又把视线转回监控面板。后台刚弹出一条新记录：

`跨域施法许可校验：通过`

`来源：龙族教育署`

`姓名：林小雨（15）`

`活动区域：星芒馆`

蓝月指尖停了半秒，右手在桌沿轻点，魔力感知沿着城市节点铺开，像一张几乎透明的网掠过去。

没有异常波动，没有伪装遮罩，也没有恶意探针。

`风险评级：低`

`处置建议：观察，无需干预`

他把标签打成灰色，关掉窗口，继续盯下一批传播日志。

菲娜到星芒馆时，正好是上午人最散的时候。

馆内的魔法水晶灯投下淡蓝色光，墙上挂着历代魔法师画像，安静得像一排在旁听的旧时代。远处零零星星有几组练习者，低声念咒，偶尔传来一两声术式落点的闷响。

她挑了靠窗的一块空练习区，从书包里拿出铝合金学徒杖，又撕了几张废草稿纸抛到半空。

“Ventus linearis.（线性气流）”

杖尖划出干净的导流线，纸片被风托起，像一场被放慢的薄雪，在她面前旋转、上浮、再沿固定轨迹下落。菲娜盯着每一片纸的转向角，手腕微调，连续做了三轮。

第四轮结束时，汗从她额角滑下来，她抬手擦掉，呼吸却很稳。

“啪、啪、啪。”

旁边突然响起三下掌声。

菲娜回头，看见一个陌生女孩站在练习区外，双马尾，发梢染了冷粉色，穿着黑灰配粉的地雷系外套和短裙，鞋跟厚得很有存在感。她抱着平板，眼睛亮亮的，瞳仁里有一圈很浅的金色竖纹。

“你刚那道气流超好看。”女孩说，“像把风缝进纸边里。”

菲娜有点不好意思地挠了挠脸：“谢谢。你也来练习吗？”

“嗯。”女孩往前走了一步，先把平板夹到臂弯里，再很正式地伸手，“林小雨，龙族来的交换生，十五岁，持证施法，不是非法喷火人员。”

菲娜被她逗笑，握了握手：“菲娜。也是刚拿到初级资格。”

“我知道你。”林小雨眨眼，“刚才你那手线性气流，在那边都有人看愣了。”

她说着把平板打开，屏幕上是一组精细的魔法阵草图，线条带着明显龙族古纹风格，但结构又很现代。

页面角落标着来源：

`暗月会 · 命定色盘 联合生成`

`Prompt：龙族古纹 / 初级安全阈 / 导流防抖`

林小雨把屏幕往她这边侧了侧：“我拿它当草图参考。你会用AI辅助吗？”

菲娜看着那组阵图，脑子里先浮起蓝月早上的话。

看得懂再用。看不懂，就当没看见。

她没有立刻回答“会”或“不会”，只给了个中间值：“偶尔看思路，不直接上杖。”

“谨慎派啊。”林小雨点点头，也不追问，“那要不要打一轮？比聊十句更快认识彼此。”

菲娜把杖在掌心转了半圈，笑起来：“可以。友好切磋，先说好不炸馆喵。”

“成交。”

两人去前台登记了对练区。系统很快拉起简易防护环，地面亮起淡银色分界线。

周围练习者听到动静，陆续把视线投过来。

林小雨先起手，龙语咒音很短，音节低而稳。她杖尖前方展开一圈铜金色纹环，像古旧钟表的内盘，随后一道压得很实的风流平推出来，轨迹厚重，几乎不飘。

菲娜眼神一紧，侧步让开正面线，反手放出线性气流切侧翼，接一发短促火术去扰对面导流节奏。

火光刚亮，林小雨杖身一横，半透明鳞纹盾“咔”地合上，火星在盾面碎成细屑。

“你这反应快得离谱。”林小雨说。

“你这盾硬得不讲理。”菲娜回她。

第二轮，两人都提了速。

林小雨的术式像古谱，重拍明显，气口沉稳，一拍一拍往前压；菲娜的节奏更像她这一代的标准训练，短、快、连续修正，靠高频微调拿控制权。

一重一轻，风在防护环里交错，像两套不同年代的乐谱硬拼在同一页纸上。

第三轮末段，菲娜抓到半个空档，杖尖亮起一道窄光，直切林小雨左侧落点；林小雨几乎同步下压龙纹环，光束擦着边缘过去，在防护墙上拉出一条细长亮痕。

裁定器“滴”地一声，自动判停。

屏幕给出结果：

`菲娜：稳定度 87 / 命中 83`

`林小雨：稳定度 86 / 命中 84`

`综合判定：平手`

围观区响起一阵不高不低的掌声。有人小声感叹“这俩初级？”，也有人已经开始拿终端记动作。

林小雨先把杖收了，笑得很痛快：“好久没打这么顺了。你现代施法真的很灵。”

菲娜也笑，胸口还在起伏：“你那套龙族术式，压迫感像老城墙。”

“我当夸奖收下。”

她们在场边坐下喝水。林小雨拿毛巾擦着额头，把平板又递过来一点：“下次有空的话，我们可以一起研究AI生成符文。我这边有不少龙族语料，能生成一些很有意思的结构。”

菲娜看着屏幕上那些漂亮得近乎诱人的图，停了两秒，还是点头：“行。先从低风险的基础符文看起。”

“专业。”林小雨给她比了个拇指，“加个联系方式？”

“好。”

两人互扫了账号。菲娜备注对方时，手指在“林小雨”后面加了个“龙族”，想了想，又删掉，只留名字。

刚加完，手机顶部就跳出蕾雅的消息：

“晚上逛街！庆祝你过证！我请奶茶和炸鸡！”

第二条紧跟着：

“不许拒绝，我都想好你要穿哪件了！”

菲娜笑出声，低头回她：

“下次吧喵，今天想早点回家。”

蕾雅秒回：

“懂，又是欧尼酱优先。你们兄妹锁死，我退下了（哭哭）”

菲娜回了个猫爪表情包，收起手机。林小雨凑过来瞄了一眼，挑眉：“你朋友好有戏。”

“她每天都很有戏。”

“那你今天真不去玩？”

菲娜把水瓶拧紧，站起来，笑意淡了半分，却没消失：“改天吧。今天想回去看看我哥。”

林小雨也起身，拍了拍裙摆：“行。那我们下次约星芒馆。我把那套龙族语料整理好发你。”

“好，路上小心。”

“你也是，初级魔法师小姐。”

两人在馆门口分开。菲娜走出去几十米，回头看了一眼，林小雨已经低头在平板上写写画画，双马尾随着步子轻轻晃，像一对很精神的感叹号。

回家路上，菲娜绕进那家常去的甜品店。玻璃柜里小泡芙刚出炉，奶油还没完全定型，表面带着一层很薄的光。

“要几只？”店员问。

“两个就好。”菲娜想了想，又补一句，“分开装。”

纸袋到手时还带着一点温度。她拎着袋子往家走，夕光把影子拉长，脚步不快，却比上午更轻。

推门进家时，客厅没人。蓝月房间的门半掩着，里面传出终端轻微的提示音。

菲娜敲了两下门，探头进去：“欧尼酱？”

蓝月抬眼，屏幕上还停着一排实时日志：

`命定色盘调用峰值：上升`

`暗月会话题热度：上升`

`衍生符文分享：上升`

`异常链路：监控中`

他看到她手里的纸袋，视线停了一下：“给我的？”

菲娜把其中一袋递过去：“嗯。今天星芒馆打了场平手，算庆祝。”

“和谁？”

“龙族来的新生，林小雨。挺强的，也挺有意思。”

蓝月接过泡芙，指尖在袋口轻轻捏了捏，语气还是那样平稳：“你没受伤就行。”

“没有。就是……”菲娜顿了顿，“她提了不少AI符文的玩法。”

蓝月看着她：“你怎么回的？”

“我说先看懂，再上杖。”

这次他没立刻说话，只很轻地点了下头：“对。”

菲娜这才笑起来，往他桌边一靠，拆开自己那份泡芙，咬了一口。奶油甜得软，壳很脆，碎屑落在她指尖。

窗外天色慢慢暗下去，室内只剩屏幕冷光和台灯暖光并在一起。菲娜看着哥哥继续处理那些飞速刷新的日志，忽然觉得今天在星芒馆里那场平手，像是把她往更大的场子里推了一步。

她还没想好那一步到底会通向哪，但至少现在，她能先把手里的杖握稳。
