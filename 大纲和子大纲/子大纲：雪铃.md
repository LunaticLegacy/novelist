## 漫展前奏：命运的邂逅（4章）

### 第1章：夏日祭的邀请

**当前状态**：
- 全局变量：菲娜刚通过魔法师考试，中枢之地进入暑假。各大社媒开始预热“夏日幻梦漫展”，命定色盘在二次元圈层已有初步影响力。
- 蓝月：正在暗中筹备雪铃实验，需要观察真实的人类互动场景作为数据样本。
- 菲娜：考后放松期，每天在家撸猫（哥哥）、追番、刷手机。
- 蕾雅：兴奋地给菲娜发消息，说漫展上有命定色盘展台，还能抽“AI绘师签名版”谷子。

**主要任务**：
- **菲娜**：被蕾雅拉着讨论漫展计划，第一次听说“VTuber”这个概念。
- **蓝月**：以“陪妹妹”为名，决定也去漫展——实际上是要观察真实的粉丝生态。

**可渲染对象**：
- 强渲染：
  - 菲娜趴在沙发上，手机外放蕾雅的语音消息：“菲娜菲娜！漫展有命定色盘限定徽章！还有VTuber现场live！去嘛去嘛！”菲娜脚丫晃来晃去，扭头朝厨房喊：“欧尼酱——蕾雅约我去漫展——”
  - 蓝月端着切好的水果出来，围裙上沾着面粉，猫耳抖了抖：“漫展？想去就去呗。”菲娜坐起来，眼睛亮晶晶：“欧尼酱陪我！”
- 弱渲染：客厅的空调声，窗外蝉鸣，水果盘里切好的蜜瓜。

**场景分镜**：
1. 菲娜在家瘫着，蕾雅消息轰炸。
2. 菲娜和哥哥商量，蓝月答应陪同。
3. 晚上菲娜查漫展攻略，越查越兴奋，拉着蓝月问：“欧尼酱你看这个VTuber雪铃好可爱！她的歌你听过吗？”
4. 蓝月看了一眼屏幕，眼神微动——这是他准备推出的虚拟偶像，还没公开。他笑了笑：“嗯，很可爱。”菲娜没注意到哥哥的异样。

### 第2章：暗流涌动

**当前状态**：
- 全局变量：漫展官方公布参展阵容，“月对影”平台首次以官方身份参展，展台主题是“AI创造未来娱乐”。同时有个神秘的“新VTuber预告”吊足了胃口。
- 蓝月：通过后台数据筛选出第一批潜在“核心粉”——那些在命定色盘社区最活跃、最愿意为虚拟内容付费的用户。他计划在漫展上“偶遇”其中几位。
- 埃文：也准备去漫展，打算在VTuber区直播取材——他当时还在做独立直播，穷得叮当响，连路费都是借的。
- 莉莉丝：刚做完性别确认手术不久，还在恢复期，犹豫要不要去漫展——她cos雪铃的计划被蓝月阻止了（“还不是时候”）。

**主要任务**：
- **蓝月**：暗中标记几个目标用户，为雪铃出道储备“种子粉丝”。
- **菲娜**：单纯地期待漫展，完全不知道哥哥在计划什么。

**可渲染对象**：
- 强渲染：
  - 蓝月深夜对着电脑，屏幕上是密密麻麻的数据流，其中几个ID被标红：“深雪”“铃音未来”“星尘应援团团长”。他轻声说：“就是你们了。”
  - 菲娜半夜起夜，看到哥哥房间灯还亮着，推门探进脑袋：“欧尼酱还不睡？”蓝月快速切换屏幕，回头微笑：“马上，给你热杯牛奶？”
- 弱渲染：走廊的夜灯，电脑散热器的微弱嗡嗡声。

**场景分镜**：
1. 蓝月筛选种子粉丝，标记目标。
2. 菲娜半夜发现哥哥还在工作，被哄回去睡觉。
3. 次日，蕾雅来家里找菲娜，两人试穿漫展当天的衣服（JK+私服混搭）。
4. 蓝月在一旁看杂志，偶尔抬头看妹妹换装，嘴角带笑。

### 第3章：相遇在夏日

**当前状态**：
- 全局变量：漫展首日，人山人海。命定色盘展台排起长队，“月对影”展台的神秘预告屏前挤满人。
- 埃文：在VTuber区直播，弹幕稀稀拉拉，他强颜欢笑。
- 莉莉丝：最终还是来了，戴着口罩和帽子，在角落里默默观察。
- 林小雨：作为普通游客（其实是雪铃最早的民间“预言家”之一），在命定色盘展台抽到了SSR级谷子，兴奋得跳起来。

**主要任务**：
- **菲娜**：和蕾雅逛展，买周边、看coser、吃小吃。第一次近距离接触“粉圈文化”。
- **蓝月**：以“普通游客”身份，在不同时间点“偶遇”埃文、莉莉丝、林小雨，分别进行简短的对话——这些对话将成为他判断这些人是否适合成为使徒的依据。

**可渲染对象**：
- 强渲染：
  - 菲娜和蕾雅挤在人群中，蕾雅突然指着远处：“快看！那个coser好帅！”菲娜顺着看过去，是一个cos某番男主的小哥哥，她脸一红，拉着蕾雅就跑：“别看别看！”
  - 蓝月端着奶茶，在一个相对安静的角落“偶遇”正在发呆的莉莉丝。他轻声说：“这里人多，要不要坐会儿？”莉莉丝警觉地看他一眼，但蓝月的眼神太温和，她鬼使神差地点了头。
- 弱渲染：漫展的嘈杂人声，闪光灯，各种BGM混在一起。

**场景分镜**：
1. 菲娜和蕾雅入场，开启逛吃模式。
2. 蓝月分别“偶遇”：
   - 埃文（在直播区，蓝月默默打赏了一笔，然后离开）
   - 莉莉丝（在角落，两人简短聊天，蓝月说“你很勇敢”）
   - 林小雨（在命定色盘展台，蓝月帮她捡起掉落的谷子）
3. 菲娜逛到月对影展台，看到神秘预告屏上的剪影，觉得有点眼熟，但想不起来。
4. 傍晚，菲娜和蓝月汇合，菲娜兴奋地展示战利品，蓝月摸摸她的头。

### 第4章：预告与暗涌

**当前状态**：
- 全局变量：漫展最后一天，月对影展台的神秘预告揭晓——雪铃的剪影和一段10秒的歌声片段。现场沸腾，社交媒体开始出现“雪铃是谁”的讨论。
- 蓝月：通过后台数据确认，种子粉丝的“期待值”已被激活。
- 埃文：在直播间里播放了偷录的雪铃歌声片段（经过蓝月默许），粉丝数当晚涨了5000。
- 莉莉丝：回到家后，给蓝月发了一条消息：“谢谢你，我今天很开心。”蓝月回复：“会更好的。”
- 林小雨：在个人账号上发了一条：“今天的漫展遇到了天使！预感有个超棒的VTuber要出道了！”

**主要任务**：
- **蓝月**：确认雪铃出道的“情绪地基”已经铺好，三天后正式启动实验。
- **菲娜**：回到家累瘫，但在睡前刷手机时看到雪铃的预告，顺手点了个赞。她不知道，这个赞会被蓝月看到，也让蓝月在接下来的实验中多了一份犹豫——如果妹妹也被卷进来呢？

**可渲染对象**：
- 强渲染：
  - 菲娜洗完澡，裹着浴巾趴在床上刷手机，看到雪铃的预告视频，点开听了一遍，小声说：“声音真好听……”然后点了个赞。她不知道，千里之外的数据中心里，这个赞被标记为“特别关注”。
  - 蓝月在隔壁房间看着屏幕上的数据流，其中“菲娜·希格墨伊德”的ID一闪而过。他愣了一下，然后轻声说：“你只是个意外，不该在这里。”他犹豫了三秒，最终还是没把妹妹从样本中移除——只是给她打了一个特殊的标签：“观察对象，不参与诱导实验”。
- 弱渲染：月光透过窗帘，两个房间都亮着屏幕的微光。

**场景分镜**：
1. 漫展最后一天，雪铃预告发布，现场反响热烈。
2. 菲娜和蕾雅在人群中再次听到那段歌声。
3. 回家后，菲娜累瘫，睡前点赞雪铃预告。
4. 蓝月在房间看到妹妹的ID，短暂犹豫。
5. 深夜，蓝月站在菲娜房门口，听她均匀的呼吸声，轻声说：“对不起，这次我不能保护你。但你永远是我的例外。”
6. （次日）菲娜醒来，哥哥已经做好早餐。一切如常，但她不知道，三天后，一场风暴将席卷整个中枢之地的网络世界。

## 雪铃实验：第一阶段 - 出道（12章）

### 第5章：零日

**当前状态**：
- 全局变量：青铜历1051年7月8日，雪铃账号在月对影及各平台同步上线，发布第一条动态：“大家好，我是雪铃，来自雪国的VTuber。想唱歌给你听。”配图是银白长发紫瞳的少女，背景是雪原。
- 蓝月：启动命运大模型内容生成流水线，雪铃账号开始每小时发布一首歌——这些歌都是AI生成，但质量极高，且风格覆盖流行、民谣、电子、古风等。
- 菲娜：早上起来刷手机，看到关注的几个二次元账号都在转发雪铃，她点进去听了一首《雪之梦》，觉得不错，随手转发给蕾雅。
- 蕾雅：秒回：“我早就关注了！声音超治愈！”

**主要任务**：
- **蓝月**：监控首日数据——粉丝增长曲线、完播率、评论情感倾向、转发路径。
- **菲娜**：单纯地听歌，完全不知道这个“雪铃”是哥哥的造物。

**可渲染对象**：
- 强渲染：
  - 蓝月坐在电脑前，屏幕上同时开着几十个监控窗口。他轻声说：“实验开始。”手指在键盘上敲击，一条条数据流开始涌入。
  - 菲娜戴着耳机，在阳台上听雪铃的歌，脚丫跟着节奏轻点。阳光洒在她身上，她眯着眼，看起来很惬意。蓝月从窗户看到这一幕，嘴角微微上扬。
- 弱渲染：阳台上的绿植，耳机里隐约传出的歌声。

**场景分镜**：
1. 清晨，蓝月启动雪铃账号。
2. 菲娜起床，刷手机发现雪铃，听歌转发。
3. 蓝月监控数据，首日粉丝破5万。
4. 晚上，雪铃发布第一条动态：“今天唱了好多歌，有点累，但很开心。谢谢听我唱歌的你们。”评论区开始出现“老婆”“天使”等词汇。
5. 蓝月筛选评论，标记第一批“高活跃用户”——其中就有林小雨。

---

### 第6章：声音的扩散

**当前状态**：
- 全局变量：雪铃出道第2-4天，每天200首歌持续刷榜，音乐平台榜单开始出现雪铃的名字。一些音乐博主开始注意到这个“高产如母猪”的新人，质疑声出现：“这真的是人唱的？”
- 蓝月：安排AI水军开始在质疑评论区“带节奏”——不是强行洗地，而是用“我觉得好听就行”“管她是不是人，歌好听不就行了”等看似中立的言论，把争论引导向“好听就行”的方向。
- 林小雨：在个人账号疯狂安利雪铃，自称“雪铃第一粉丝”，开始组织小范围的“听歌打卡”活动。
- 蕾雅：每天给菲娜分享雪铃的新歌，两人开始讨论最喜欢哪首。

**主要任务**：
- **蓝月**：通过舆论引导，让“雪铃是不是AI”的讨论停留在“无伤大雅”的层面，避免早期就陷入“造假”争议。
- **菲娜**：被蕾雅安利，开始认真听雪铃的歌，但还没到“粉”的程度。

**可渲染对象**：
- 强渲染：
  - 深夜，某音乐博主的直播间里，博主正在分析雪铃的声纹，结论是“99%可能是AI”。弹幕吵成一片，有人刷“AI怎么了，好听就行”，有人刷“造假可耻”。蓝月在屏幕前看着这一切，微微点头：“节奏很好。”
  - 菲娜在房间里，把雪铃的歌当背景音乐写作业。蓝月敲门进来送水果，看了一眼她的屏幕，没说话。
- 弱渲染：直播间里闪烁的弹幕，菲娜笔尖划过纸面的声音。

**场景分镜**：
1. 音乐博主质疑雪铃是AI，引发小范围争议。
2. 蓝月水军介入，引导舆论方向。
3. 林小雨开始组织粉丝活动。
4. 菲娜日常听歌写作业，和蕾雅聊天讨论。

### 第7章：第一次“翻车”

**当前状态**：
- 全局变量：雪铃出道第5天，一个自称“前月对影员工”的匿名账号爆料：“雪铃是AI生成的，连人都不存在！”附带一些所谓的“内部文件”。话题冲上小热搜。
- 蓝月：早有准备，放出雪铃的“人设设定集”——详细到离谱的设定（血型、星座、喜欢的食物、小时候养过什么宠物等），并让雪铃发一条动态：“我是AI？我连自己是不是AI都不知道呢～但我知道，我想唱歌给你们听。”配图是手写的“谢谢支持”卡片照片。
- 粉丝反应：评论区瞬间被“相信雪铃”“别黑我铃”刷屏，林小雨更是发了长文怼爆料者。
- 菲娜：刷到这条动态，觉得雪铃的回应“有点可爱”，第一次在雪铃评论区留言：“加油。”

**主要任务**：
- **蓝月**：利用这次“翻车”事件，反向强化粉丝的“守护欲”，完成第一次粉丝凝聚力构建。
- **菲娜**：无意识中成为雪铃的“路人粉”。

**可渲染对象**：
- 强渲染：
  - 雪铃那条手写卡片的动态下，评论区飞速增长。林小雨的长文被顶上热评，她写道：“我不知道雪铃是不是AI，我只知道她的歌陪我熬过了最难的时候。就凭这个，我信她。”
  - 菲娜看到这条评论，愣了一下，然后默默点了个赞。她想起自己难过时，哥哥也是用音乐陪她的。
- 弱渲染：手机屏幕的光，点赞动画的小红心。

**场景分镜**：
1. 爆料出现，舆论发酵。
2. 雪铃回应，手写卡片引发共情。
3. 粉丝开始组织“反黑”，林小雨带头。
4. 菲娜点赞评论，被蕾雅发现：“你也粉上雪铃了？”菲娜：“没有啦，就觉得挺可爱的。”
5. 蓝月监控数据，确认“守护欲”已激活，粉丝粘性大幅提升。

### 第8章：铃家初立

**当前状态**：
- 全局变量：雪铃出道第6-8天，粉丝数突破50万。“铃家”开始自发形成——最初是林小雨建的一个小群，然后迅速扩张，出现多个分群。群内开始讨论“我们该做点什么”。
- 蓝月：开始暗中布局“头部粉”体系，将林小雨、埃文（的AI模仿号）、以及几个真实的高活跃用户设为“核心圈”，引导他们建立应援规则。
- 埃文：在直播时无意中哼了一段雪铃的歌，被粉丝问“你也喜欢雪铃？”他愣了一下，然后说：“啊，最近在听。”这个片段被剪辑传播，埃文意外收获一波雪铃粉的关注。
- 莉莉丝：在恢复期，每天听雪铃的歌疗愈，成为雪铃的“沉默粉”。

**主要任务**：
- **蓝月**：完成“铃家”核心架构的初步搭建，为后续的组织化运营做准备。
- **菲娜**：被蕾雅拉进一个“雪铃歌迷群”，但她只是潜水，偶尔看看。

**可渲染对象**：
- 强渲染：
  - 林小雨的房间里，她对着电脑屏幕，和几个新认识的网友讨论“应援口号”。她敲下一行字：“就叫‘铃家’吧，像一家人一样。”群里瞬间刷屏“同意”。
  - 菲娜被拉进群，看到满屏的“欢迎新人”，有点不知所措。她发了一个猫猫头表情，然后默默打开免打扰。
- 弱渲染：电脑屏幕的光，群消息的提示音此起彼伏。

**场景分镜**：
1. 林小雨建群，铃家雏形出现。
2. 蓝月引导核心粉建立规则。
3. 埃文意外蹭到雪铃热度。
4. 菲娜被拉进群，潜水观察。
5. 蓝月查看群数据，确认“自组织”已启动，他可以逐渐退居幕后。

### 第9章：第一次团建

- **蓝月**：整理实验数据，准备前往月精灵族。
- **菲娜**：回归日常，但她不知道，这只是暴风雨前的宁静。

- 强渲染：
  - 菲娜合上笔记本，看着窗外的月亮。蓝月敲门进来：“还不睡？”菲娜回头：“在想雪铃的事。”蓝月坐到她旁边：“想什么？”菲娜靠在他肩上：“想她是不是真的存在过。”蓝月沉默片刻：“对你来说，存在过，就够了。”

- 弱渲染：月光洒在两个房间里，两个人都没睡，各自想着心事。

**场景分镜**：
1. 菲娜写日记，和哥哥对话。
2. 蓝月整理数据，准备出发。
3. 次日，蓝月告诉菲娜要出差几天。
4. 菲娜送哥哥出门，看着他消失在街角。
5. 菲娜回到家，打开手机，雪铃的歌还在循环。
