## 添加新词条的说明

添加一条新的衣物时，需要在右侧加入该服装对应的文化背景内的原生语言译文。
建议一并补全：`材质/版型`、`季节版本`、`常见搭配`、`适用场景`。

## 标签规范

- 标签模板：`文化=...` `风格=...` `时代=...` `功能=...` `气质=...`
- 最低要求：每条至少填写 `文化` + `风格` 两项。
- 建议写法：
  - `文化`：日式/中式/欧式/古罗马/古希腊/斯拉夫式 + 细分（如现代校园、唐风、维多利亚）。
  - `风格`：学院/甜系/哥特/礼仪/军务/通勤/仪式感等。
  - `时代`：古典/近代/现代/架空。
  - `功能`：常服/制服/礼服/工装/战斗/仪式。
  - `气质`：清冷/华丽/克制/张扬/庄严/机能。

## 日语标注规则（新增）

- 纯片假名词：后接原始语言词形与语种标注。  
  例：`リボン / ribbon, English`、`カヌレ / canelé, French`
- 含日语汉字词：汉字后补假名读音。  
  例：`玄関(げんかん)`、`作務衣(さむえ)`、`口癖(くちぐせ)`
- 和製語（日本本土造词）仍可保留日语说明，并标注“和製語”。


## 服装类

### 服装组件字段

- 每条服装词条建议至少包含：
  - `组成成分`：拆成3-6个部件，每个部件写形态 + 功能 + 可见细节。
  - `常见材质`：棉/麻/毛/丝/皮革/化纤/混纺等。
  - `微观观察点`：缝线、褶量、磨损、起球、反光、声响、活动时形变。

### 宅系服装

- 水手服（セーラー服(せーらーふく) / sailor suit, English）
    - 标签：`文化=日式-现代校园` `风格=学院` `时代=现代` `功能=制服` `气质=清爽`
    - 二次元常见萌属性之一。
    - 组成成分：
      - `水手领`：后领片较大，覆盖肩背，边缘常有对比色滚条。
      - `前胸挡片`：与领口连成视觉三角区，决定制服“脸部框线”。
      - `前胸系带`：リボン / ribbon, English 与 ネクタイ / necktie, English 两类，结体大小决定幼态或成熟感。
      - `上衣主体`：短身或常规长度，袖口常见双线条呼应领边。
      - `百褶裙`：褶宽与褶密决定动态摆幅，走动时有规律开合。
    - 常见材质：棉涤混纺、聚酯纤维、呢料（冬装）。
    - 季节版型：春夏短袖、秋冬长袖。
    - 常见变体：不良系水手服、ブレザー / blazer, English 混搭、セパレート / separate, English 改造款。
    - 微观观察点：领边压线、褶线锋利度、系带松紧、裙摆回弹、鞋跟回声。

- JK制服（女子高生制服）
    - 标签：`文化=日式-现代校园` `风格=学院/都市` `时代=现代` `功能=制服` `气质=利落`
    - 组成成分：
      - `衬衫`：标准翻领，白色最常见，领角硬挺度影响整体干净感。
      - `西装外套`：肩线、驳头、扣位构成“校规感”；胸章通常在左胸。
      - `针织层`：背心或开衫，增加层次并用于季节保温。
      - `下装`：百褶裙或西裤，长度与褶量体现校规与个人风格。
      - `领部件`：领带/领结颜色常用于年级或校别区分。
      - `鞋袜`：乐福鞋 + 中筒袜/过膝袜，决定收束感与节奏感。
    - 常见材质：棉、羊毛混纺、聚酯纤维、针织面料。
    - 校规变化点：裙长、袜长、外套扣法、领结颜色。
    - 常见穿搭分层：衬衫 + 针织背心（ニットベスト / knit vest, English）+ 开衫（カーディガン / cardigan, English）。
    - 微观观察点：外套肩线是否塌陷、胸章磨损、领带结点偏移、袜口压痕。

- Lolita（ロリータ・ファッション / Lolita fashion, English）
    - 标签：`文化=日式-次文化` `风格=甜系/古典/哥特` `时代=现代` `功能=常服/活动` `气质=华丽`
    - 组成成分：
      - `主裙体`：JSK/OP为核心，腰线与裙摆体积共同决定轮廓。
      - `衬裙`：提供体积支撑，硬挺度影响钟形或A字形态。
      - `内搭上衣`：高领、泡泡袖或蕾丝领常用于增强层次。
      - `头饰`：KC、bonnet、发带决定风格识别度。
      - `腿部与鞋履`：花纹袜 + 玛丽珍/厚底鞋，延续整体主题。
    - 常见材质：棉、提花、雪纺、蕾丝、天鹅绒。
    - 轮廓核心：通过衬裙（petticoat/crinoline）做出钟形或A字裙摆。
    - 常见单品：JSK（ジャンパースカート / jumper skirt, English）/ OP（ワンピース / one-piece dress, English），裙长多在膝部附近。
    - 主流子风格：Sweet、Classic、Gothic。
    - 细节关键词：蕾丝层次、胸前蝴蝶结、头饰（ボンネット / bonnet, French 与 リボン / ribbon, English）。
    - 微观观察点：蕾丝层级、裙摆波浪边、胸前蝴蝶结对称性、鞋面擦痕。

- 女仆装（メイド服(めいどふく) / maid outfit, English）
    - 标签：`文化=欧式-维多利亚衍生` `风格=服务角色扮演` `时代=近代/现代再演绎` `功能=制服/表演` `气质=端正或戏剧化`
    - 组成成分：
      - `连衣裙主体`：深色底布为主，领口与袖口常有荷叶边或花边。
      - `围裙`：白色高对比前片，腰带系法可传达角色习惯。
      - `头饰`：荷叶边发箍或头巾，强化职业角色识别。
      - `袖部与下摆`：泡泡袖或长袖版本决定甜系/端正走向。
      - `袜鞋组合`：常配长袜与低跟鞋，强调礼仪与整洁。
    - 常见材质：棉、涤纶、府绸、蕾丝辅料。
    - 常见结构：深色连衣裙 + 白围裙 + 头饰（カチューシャ / katyusha, Russian 与 フリルヘッドドレス / frill headdress, English）。
    - 语义分化：维多利亚风偏端正，法式女仆风偏戏剧化。
    - 场景标签：女仆咖啡店（メイド喫茶(めいどきっさ)）通常搭配固定问候语与角色扮演服务。
    - 微观观察点：围裙褶皱、腰带结尾长短、袖口白边、鞋面抛光程度。

- 死库水（スクール水着(すくーるみずぎ) / school swimsuit, English）
    - 标签：`文化=日式-校园体育` `风格=运动/制服` `时代=现代` `功能=泳装` `气质=功能性`
    - 组成成分：
      - `连体主身`：一体成型包覆躯干，强调贴身和阻力控制。
      - `肩带`：宽肩带更稳定，细肩带更强调外观轻量化。
      - `胸前名牌区`：学校或姓名标识位，常见白色名牌布块。
      - `腿根开口`：开口高低决定活动自由度与视觉风格。
      - `内衬层`：防透与贴肤舒适层，湿水后表现明显。
    - 常见材质：尼龙、涤纶、弹性纤维（氨纶）。
    - 全称缩写来自 school mizugi（スクール水着(すくーるみずぎ)）。
    - 常见为贴身连体款，颜色多为深蓝/黑，面料多见尼龙 + 涤纶。
    - 版本迭代可写：旧型/新型/竞泳型/スパッツ / spats, English 型。
    - 微观观察点：湿水后深浅变化、肩带压痕、名牌边缘卷曲、干湿交界线。

### 工作服装

- 作务衣（作務衣(さむえ) / samue）
    - 标签：`文化=日式-寺务/工务` `风格=朴素` `时代=传统延续` `功能=工装` `气质=克制`
    - 组成成分：
      - `上衣`：交叠前襟，以系带固定，活动时不易掀开。
      - `下装`：宽松长裤，腰部多为抽绳或松紧结构。
      - `袖口`：可卷起处理，便于劳动和清洗。
      - `口袋`：简洁实用，常位于上衣前片或裤侧。
    - 常见材质：棉、麻、竹纤维混纺。
    - 日本僧侣与寺务劳动服出身，后扩展到园艺、餐饮与旅馆工装。
    - 常见面料：棉、麻；版型宽松，便于长时活动。
    - 微观观察点：系带磨毛、膝部褶皱、袖口水渍、布面旧化程度。

- 法被（法被(はっぴ) / happi）
    - 标签：`文化=日式-祭典` `风格=民俗/应援` `时代=传统延续` `功能=活动服` `气质=热闹`
    - 组成成分：
      - `前襟`：直开襟，便于快速穿脱和叠穿。
      - `筒袖`：短而宽，动作幅度大时不束缚手臂。
      - `背纹`：大面积文字或家纹，是团队识别核心。
      - `腰带`：简化固定和调节松紧，便于行进或舞动。
    - 常见材质：棉布、斜纹布、涤棉混纺。
    - 传统筒袖短外衣，常在祭典、商铺活动、应援场景使用。
    - 识别点：前襟与背部印字（店名、组名、祭典名）。
    - 微观观察点：印字开裂、领口汗渍、腰带打结方向、袖口飞边。

- 白衣（白衣(はくい) / ラボコート / lab coat, English）
    - 标签：`文化=现代通用` `风格=专业` `时代=现代` `功能=工装` `气质=理性`
    - 组成成分：
      - `翻领`：驳领或平领，决定职业观感（医疗/实验）。
      - `门襟`：纽扣或暗扣结构，开合速度影响工作效率。
      - `功能口袋`：胸袋与下摆袋用于放笔、手套、记录本。
      - `袖口`：直口或收口，影响防污染和操作便利。
      - `后开衩`：提升步幅和坐姿舒适度。
    - 常见材质：棉涤混纺、防泼液面料、防静电纤维（实验室）。
    - 医疗与实验场景高频工装，强调“防污、可高温清洗、身份识别”。
    - 微观观察点：胸袋笔渍、袖口消毒剂泛白、下摆摩擦皱、暗扣松动。

- 连体工装（つなぎ / tsunagi）
    - 标签：`文化=现代通用` `风格=机能` `时代=现代` `功能=工装` `气质=硬朗`
    - 组成成分：
      - `连体主身`：上衣与裤身一体，减少勾挂风险。
      - `前中拉链`：从领口延伸到裆部，便于快速穿脱。
      - `工具口袋`：胸、臀、腿部多口袋分工明确。
      - `收口结构`：袖口/裤脚可收紧，避免卷入设备。
      - `膝部加固`：双层布或补丁提高耐磨性。
    - 常见材质：帆布、斜纹棉、阻燃面料、耐油污涂层布。
    - 机械维修、仓储、片场布景常见；多口袋、耐磨、便于收纳工具。
    - 微观观察点：拉链齿磨损、膝部油渍层、口袋边缘起毛、反光条刮痕。

### 常服

- 连帽卫衣（フーディー / hoodie, English）
    - 标签：`文化=现代通用` `风格=街头/休闲` `时代=现代` `功能=常服` `气质=松弛`
    - 组成成分：
      - `帽体`：双层或单层帽，决定垂坠和保暖性。
      - `帽绳`：金属头或塑封头，体现做工等级。
      - `前袋`：kangaroo pocket常用于暖手和临时收纳。
      - `袖口下摆罗纹`：收束轮廓并控制松紧。
    - 常见材质：棉、抓绒、毛圈布、涤棉混纺。
    - 轮廓关键词：落肩、帽绳、前袋（kangaroo pocket）。
    - 微观观察点：罗纹回弹、帽绳磨损、前袋边线拉扯、起球密度。

- 开衫（カーディガン / cardigan, English）
    - 标签：`文化=欧式起源-全球化` `风格=学院/通勤` `时代=近代至现代` `功能=常服` `气质=温和`
    - 组成成分：
      - `门襟`：纽扣排决定开合方式和视觉纵线。
      - `领型`：V领/圆领影响脸部比例和内搭露出面积。
      - `袖型`：插肩袖或常规袖影响肩线柔和度。
      - `罗纹边`：领口、袖口、下摆统一收束。
    - 常见材质：羊毛、棉纱、腈纶、羊绒混纺。
    - 叠穿友好：衬衫外层、制服外层、吊带外层。
    - 微观观察点：扣眼松紧、肘部起鼓、袖口起球、肩线塌陷。

- 针织背心（ニットベスト / knit vest, English）
    - 标签：`文化=欧式起源-全球化` `风格=学院` `时代=现代` `功能=常服/制服内搭` `气质=文雅`
    - 组成成分：
      - `领口`：V领最常见，露出领带结和衬衫领尖。
      - `肩部`：无袖开口尺寸决定手臂活动和层次露出。
      - `下摆罗纹`：控制腰部收束，防止上翻。
      - `纹理`：平针、麻花、提花影响视觉密度。
    - 常见材质：棉纱、羊毛、腈纶混纺。
    - 常用于“学院感”造型，配衬衫与领带层次明确。
    - 微观观察点：领口卷边、肩口磨毛、织纹拉伸、内搭透影。

- 百褶裙（プリーツスカート / pleated skirt, English）
    - 标签：`文化=现代通用` `风格=学院/通勤` `时代=现代` `功能=常服/制服` `气质=轻盈`
    - 组成成分：
      - `腰头`：硬衬腰头决定贴合和稳定性。
      - `褶裥区`：刀褶/箱褶不同，动态表现差异明显。
      - `裙长`：膝上、及膝、过膝对应不同礼仪和气质。
      - `内衬`：提升舒适度并减少透感。
    - 常见材质：毛涤混纺、聚酯纤维、棉质斜纹。
    - 静态看褶线密度，动态看摆幅与回弹。
    - 微观观察点：褶线保型度、腰扣磨损、坐压折痕、下摆飞线。

- 直筒牛仔裤（ストレートデニム / straight denim, English）
    - 标签：`文化=现代通用` `风格=工装衍生/休闲` `时代=现代` `功能=常服` `气质=中性`
    - 组成成分：
      - `裤腰`：腰头 + 皮牌 + 纽扣门襟，决定品牌识别度。
      - `裆部结构`：前后片拼接影响活动舒适度。
      - `口袋系统`：前插袋、零钱袋、后贴袋兼具功能与造型。
      - `裤管`：直筒线条从大腿到脚口基本一致。
      - `铆钉和车线`：高受力部位的耐久关键。
    - 常见材质：丹宁布（棉为主）、弹力牛仔（加氨纶）。
    - 写作重点：洗水、磨白、膝部折痕、裤脚堆积。
    - 微观观察点：猫须纹、蜂窝褶、口袋边磨损、脚口毛边。

- 切斯特大衣（チェスターコート / chester coat, English）
    - 标签：`文化=欧式-近代男装衍生` `风格=都市/正式` `时代=近代至现代` `功能=外套` `气质=克制`
    - 组成成分：
      - `驳领`：缺角驳领最常见，决定正式度。
      - `门襟`：单排或双排扣，控制胸前体量。
      - `肩部结构`：垫肩强弱影响气场和轮廓硬度。
      - `口袋`：插袋或盖袋，兼顾线条与实用。
      - `后开衩`：单衩/双衩决定步幅与动态垂感。
    - 常见材质：羊毛、羊绒混纺、粗花呢。
    - 冬季通勤核心单品，肩线和翻领决定气场。

    - 组成成分：
      - `前片`：前摆短切，露出马甲和礼服衬衣。
      - `后摆`：双尾分叉，是燕尾服核心识别结构。
      - `驳领`：常见丝缎面，形成礼服光泽对比。

    - 常见材质：羊毛精纺、缎面驳领、丝衬里。
    - 常见于白领结礼仪场景，后摆分叉是识别点。
    - 微观观察点：后摆甩动弧度、驳领反光、袖口露白长度、纽扣光泽。

    - 组成成分：
      - `前身`：面料与西装一致或对比，用于构建层次中心。

## 属性类

### 属性记录规范

- 属性记录模板：
    - `基线值`：角色在平稳状态下的默认表现。
    - `波动范围`：受情绪、环境、伤病、关系影响的上下限。
    - `触发条件`：导致属性变化的事件或刺激。
    - `外显信号`：他人可观察到的动作、语气、姿态或生理反应。
    - `恢复机制`：角色如何从异常状态返回基线。
    - `他人可见度`：公开/半公开/隐匿。
- 写作落点建议：优先写“外显信号”，少写抽象结论。
- 回写节奏建议：每章至少记录 1 条属性变化（可微小）。

### 属性可能内容枚举（新增）

- 使用建议：从每个属性先选 `1 个基线值 + 1 个异常值 + 1 个恢复值`，避免人设过满。

#### 视觉属性枚举

- 发色（髪色(かみいろ)）
    - 自然系：乌黑、深棕、栗色、浅棕、灰白。
    - 染发系：雾灰、亚麻、蓝黑、酒红、奶金、挑染撞色。
    - 幻想系：月白、银蓝、霓紫、极光渐层（适合高魔/科幻世界）。

- 发质
    - 直顺、微卷、自然卷、羊毛卷、硬直炸毛、湿重塌贴。
    - 伴随状态：干枯分叉、油光贴头皮、静电翘发、雨后回卷。

- 瞳色（瞳色(ひとみいろ)）
    - 常见色：黑、深棕、浅棕、灰、蓝、绿、琥珀、紫（设定向）。
    - 纹理类型：环形边深、放射纹、碎光斑、异色瞳。

- 肤况
    - 质地：细腻、粗糙、泛油、干燥、易泛红、冷白偏青。
    - 痕迹：雀斑、晒痕、旧疤、淤青、手背针眼、茧裂。

- 体态（姿勢(しせい)）
    - 基线：挺直中轴、松弛中立、轻微前倾、轻微后仰。
    - 异常：缩肩防御、抱臂封闭、下颌上挑、躯干定住。

- 步态
    - 轻步、重步、拖步、急促碎步、军式稳步、潜行前掌步。

- 手部特征
    - 指型与茧位：笔茧、刀茧、弓弦茧、键盘茧、掌心粗糙。
    - 保养程度：短甲清洁、甲缘毛刺、护手油残留、伤口贴布。

#### 生理属性枚举

- 体温耐受
    - 耐寒：高/中/低；耐热：高/中/低；出汗：少/中/多。
    - 外显：鼻尖红、手凉、额汗、后背汗、呼气白雾。

- 疲劳阈值
    - 专注续航：30 分钟 / 90 分钟 / 3 小时 / 6 小时。
    - 透支信号：语句缩短、动作省略、反应延迟、易怒或走神。

- 痛觉反应
    - 类型：高耐受硬撑型、中等延迟型、低阈值即时反应型。
    - 表现：咬牙、吸气停顿、手指蜷缩、沉默或爆发。

- 睡眠结构
    - 类型：规律深睡、浅眠易醒、熬夜补觉、梦扰型、失眠型。
    - 恢复后状态：清醒、迟钝、烦躁、反常兴奋。

#### 行为属性枚举

- 说话速度（話速(わそく)）
    - 慢速（稳压）、中速（日常）、快速（兴奋/焦虑）、爆发快语（失控）。
    - 可设“压力增幅”：压力越高，语速 +10% / +30% / +60%。

- 语调曲线
    - 平直冷调、上扬活泼、下压权威、断裂情绪化、抑扬戏剧化。

- 停顿模式
    - 句中停、句尾停、关键词前停、改口后停、沉默对抗型停顿。

- 视线策略
    - 强对视、间歇对视、回避对视、扫视环境、盯物不盯人。

- 手势习惯
    - 小幅节制型、中幅说明型、大幅主导型、零手势克制型。

- 决策节奏
    - 慢思快决、快思快决、慢思慢决、冲动先动后补。

- 口癖（口癖(くちぐせ)）重点枚举
    - 句首填充型：`那个`、`就是`、`然后`、`说实话`、`怎么说呢`。
    - 句中缓冲型：`嗯`、`呃`、`你知道吧`、`对吧`、`是吧`。
    - 句尾语气型：`啦`、`呀`、`呢`、`嘛`、`哟`、`喵`、`的说`。
    - 强调断言型：`必须`、`绝对`、`肯定`、`我保证`。
    - 回避降压型：`随便`、`都行`、`再说吧`、`看情况`。
    - 角色化自称型：`本小姐`、`本大爷`、`在下`、`鄙人`。
    - 职业术语型：程序员口头 `这个要复现`、医生口头 `先观察`。
    - 文化混用型：中日英混词、方言插入、宗教敬语插入。
    - 高频触发场景：
      - 紧张时回潮：句尾 `喵`、句首 `那个` 频率飙升。
      - 说谎时：补词变多、改口次数上升、句尾变短。
      - 发怒时：口癖消失，改为命令短句。
      - 放松时：口癖更黏连、尾音拉长。
    - 频率标尺（建议）：
      - L0：几乎无口癖。
      - L1：每 20 句 1 次。
      - L2：每 8 句 1 次。
      - L3：每 3 句 1 次（强角色化）。
      - L4：几乎句句出现（仅限喜剧化或刻意人设）。
    - 演化路径建议：
      - 前期高频 -> 中期收敛 -> 关键场景回潮（体现成长与压力）。
      - 或前期克制 -> 建立亲密关系后自然暴露。
    - 使用禁忌：
      - 不要让口癖遮蔽信息。
      - 不要在所有情境同频率出现。
      - 不要让不同角色共享同一口癖模板。

### 姿态类型库（新增）

- 站立姿态（静态）
    - 中轴中立：双脚平行，重心居中；适合“平稳、可谈判”场景。
    - 前倾压迫：上身微前、下颌前送；常见于施压或抢话语权。
    - 后仰防御：骨盆后坐、上身后撤；常见于戒备或不信任。
    - 开放展示：肩背展开、手臂外放；常见于自信或主场感。
    - 闭合收束：双臂贴躯、肩线内收；常见于不安、寒冷或服从。
    - 倚靠立姿：肩背/臀部借墙借桌；常见于拖延、观察、轻挑衅。

- 站立姿态（动态前兆）
    - 起步预备：一脚前探、脚跟抬起；表示“准备离场/追击/撤退”。
    - 半蹲蓄力：膝微屈、重心下沉；表示“可能突发动作”。
    - 侧身让位：躯干转开、手臂引导；表示“礼让或软性拒绝”。
    - 脚尖指向出口：身体未动但意图先行；表示“想结束对话”。

- 坐姿类型
    - 正坐挺背：骨盆中立、背柱上提；常见于正式会谈或受训状态。
    - 前缘坐：坐在椅子前1/3，重心前压；常见于专注和攻击性提问。
    - 后靠摊坐：靠背完全承重；常见于疲劳、松弛或轻视。
    - 并膝收束：膝并拢、手放腿面；常见于礼仪场合或自我约束。
    - 交叠腿：节奏慢，视线稳定；常见于掌控局面或审视他人。
    - 抱膝/缩腿：躯干包裹下肢；常见于自我保护与低安全感。

- 行进姿态
    - 猎行快步：步幅短而频率高，视线扫角落；适合追踪与巡查。
    - 静步潜行：落地由前掌过渡，声响最低；适合潜入/躲避。
    - 重步推进：脚跟先落、声压明显；适合威慑或愤怒推进。
    - 拖步回撤：步伐黏地、肩线下坠；适合受伤、失落、过劳状态。

- 手部姿态（伴随姿态）
    - 抱臂：防御、评判或保温；要看肩线是否放松来区分敌意。
    - 背手：控制情绪或维持礼仪；常见于上位者或受训者。
    - 指尖对触：思考与组织语言；常见于分析型角色。
    - 摸后颈/搓指节：焦虑和自我安抚高频动作。
    - 反复整衣：恢复秩序感，常见于紧张后自我重建。

- 头颈姿态
    - 低头避视：回避冲突、羞耻或服从。
    - 抬颌俯视：挑衅、压场或防御性自大。
    - 歪头倾听：示好、困惑或诱导对方继续说。
    - 颈部定住：情绪冻结或高警惕的外显信号。

- 姿态误判提醒
    - 冷并不总是敌意：寒冷环境会造成闭合姿态假象。
    - 疲劳会伪装成傲慢：后靠摊坐不必然等于轻视。
    - 文化差异影响礼仪姿势：并膝/距离阈值需按文化背景解读。

- 步态（gait）
    - 标签：`文化=通用` `风格=节奏信息`
    - 细化字段：步频、落地顺序、脚跟声、转向习惯、停步方式。
    - 微观细节：拖步、急停滑移、跨步避让、楼梯节拍变化。

- 手部特征（hand signature）
    - 标签：`文化=通用` `风格=职业痕迹`
    - 细化字段：掌纹粗糙度、茧位、指甲修整、常用手偏好。
    - 微观细节：笔茧、刀茧、指尖裂口、戒指压痕。

- 触觉特征（tactile signature）

    - 细化字段：体温、皮肤质感、握手力度、习惯性触碰。
    - 微观细节：
      - `体温`：手脚温度差异、运动后散热、紧张时局部发热。
      - `皮肤质感`：干燥/湿润、茧位粗糙度、毛发密度、伤痕凹凸。
      - `接触方式`：握手力度与时长、拍肩位置、递物时指尖触碰。
      - `衣物触感`：面料光滑度、配饰冷热度、包带/手机握持痕迹。

### 角色原型标签（用于速记，不等于定型）

    - 标签：`文化=日式-街头` `风格=自我表达`

    - 标签：`文化=通用` `风格=理性/系统化`
    - 冲突处理中偏好证据链和定义澄清，情绪表达常后置。

- 猎手型（hunter archetype）
    - 标签：`文化=通用` `风格=追踪/执行`
    - 对线索和时机高度敏感，倾向短决策链和高行动密度。

## 正餐与饮品类

### 标注规范

- 词条统一写法：`中文名（日语名 / 原生语言词形, 语种）`
- 纯片假名词必须补原生语言词形。
- 每条建议包含：`组成要素`、`口感结构`、`微观细节`、`场景标签`。

### 日式料理

- 寿司（寿司(すし) / sushi）
    - 标签：`文化=日式` `风格=精致/季节` `功能=正餐/轻食`
    - 组成要素：醋饭(シャリ) + 主材(ネタ) + 山葵/酱油。
    - 口感结构：米饭微酸弹牙 -> 鱼肉油脂或清爽 -> 山葵刺激收尾。
    - 微观细节：米饭温度(人体温)、鱼肉刀纹、酱油滴落轨迹、山葵分布。
    - 场景标签：板前吧台、回转寿司、外带盒装。

- 拉面（ラーメン / ramen, 和製語）
    - 标签：`文化=日式-中华衍生` `风格=浓郁/饱腹` `功能=正餐`
    - 组成要素：面条 + 汤底(豚骨/味噌/酱油/盐味) + 配菜(叉烧/笋/蛋/海苔)。
    - 口感结构：面条硬度(バリカタ等) -> 汤底浓度 -> 油脂挂舌。
    - 微观细节：汤面油花分布、面条吸汤程度、溏心蛋黄流淌、海苔软化过程。
    - 场景标签：立食店、深夜食堂、家系拉面。

- 天妇罗（天婦羅(てんぷら) / tempura, Portuguese loan）
    - 标签：`文化=日式-葡萄牙影响` `风格=轻炸/季节` `功能=正餐/套餐`
    - 组成要素：食材(虾/蔬菜/鱼) + 薄衣面糊 + 高温快炸 + 天汁/萝卜泥。
    - 口感结构：外壳极薄酥脆 -> 内部食材本味 -> 天汁清爽解腻。
    - 微观细节：面衣气泡大小、沥油程度、咬断时的碎裂声、天汁温度。

- 咖喱饭（カレーライス / curry rice, English loan）
    - 标签：`文化=日式-英式印度衍生` `风格=家常/饱腹` `功能=正餐`
    - 组成要素：日式咖喱酱 + 米饭 + 炸物/蔬菜配菜。
    - 口感结构：咖喱浓稠挂饭 -> 微甜不辣 -> 配菜酥脆对比。
    - 微观细节：咖喱覆盖米饭比例、酱汁流动边界、配菜浸软程度、盘底刮净声。

- 便当（弁当(べんとう) / bento）
    - 标签：`文化=日式` `风格=便携/视觉` `功能=外带正餐`
    - 组成要素：米饭(常配梅干/芝麻) + 主菜(炸/烧/煮) + 副菜(腌菜/煮物) + 分隔摆盘。
    - 口感结构：多味并置、冷热分区、色彩平衡。
    - 微观细节：梅干去核痕、饭菜分隔纸/叶、盒盖压紧痕迹、打开时的蒸汽。

### 中式料理

- 点心（點心(ディムサム) / dim sum, Cantonese）
    - 标签：`文化=中式-粤式` `风格=精致/社交` `功能=早茶/聚餐`
    - 组成要素：蒸点(虾饺/烧卖/叉烧包) + 炸点 + 甜点 + 茶水。
    - 口感结构：皮薄馅鲜 -> 汁水丰盈 -> 茶水清口循环。
    - 微观细节：虾饺皮透光度、烧卖顶部橙籽、叉烧包裂口、蒸笼水珠滴落。
    - 场景标签：茶楼推车、圆桌转盘、早茶时段。

- 火锅（火鍋(ホーコー) / hot pot）
    - 标签：`文化=中式` `风格=社交/热烈` `功能=聚餐`
    - 组成要素：锅底(麻辣/清汤/番茄等) + 生料(肉/海鲜/蔬菜/豆制品) + 蘸料。
    - 口感结构：锅底风味主导 -> 食材本味 -> 蘸料个性化叠加。
    - 微观细节：汤底翻滚状态、肉片变色过程、蘸料分层(油/酱)、蒸汽蒙眼镜。
    - 场景标签：圆桌围坐、鸳鸯锅、自助取料台。

- 小笼包（小籠包(シャオロンパオ) / xiaolongbao）
    - 标签：`文化=中式-江浙` `风格=精致/汤鲜` `功能=正餐/点心`
    - 组成要素：薄皮 + 肉馅 + 皮冻汤汁 + 蒸制。
    - 口感结构：皮韧不破 -> 汤汁烫口鲜甜 -> 肉馅紧实。
    - 微观细节：褶纹数量、底部厚度、汤汁晃动、咬破时的热气喷涌。

- 面条（麺(めん) / noodles, 泛指）
    - 标签：`文化=中式-多地域` `风格=家常/饱腹` `功能=正餐`
    - 组成要素：面条(拉/擀/削/压) + 汤底/卤 + 浇头 + 配菜。
    - 口感结构：面条筋道度 -> 汤底风味 -> 浇头层次。
    - 微观细节：面条粗细均匀度、汤底油花、葱花浮散、碗沿汤渍。

### 饮品类

- 日本茶（日本茶(にほんちゃ) / nihoncha）
    - 标签：`文化=日式` `风格=清寂/季节` `功能=日常/礼仪`
    - 细分类型：煎茶(せんちゃ)、玉露(ぎょくろ)、抹茶(まっちゃ)、番茶(ばんちゃ)。
    - 口感结构：鲜味(うまみ) -> 苦涩平衡 -> 回甘。
    - 微观细节：茶汤色泽(黄绿/翠绿)、茶筅击拂泡沫、茶渣沉淀、杯底留香。
    - 场景标签：茶室点茶、日常饮用、茶会礼仪。

- 咖啡（珈琲(コーヒー) / coffee, English loan）
    - 标签：`文化=全球化` `风格=日常/社交` `功能=提神/休闲`
    - 组成要素：咖啡豆(产地/烘焙度) + 萃取方式 + 奶/糖调配。
    - 口感结构：酸度 -> 苦度 -> 醇厚度 -> 余韵。
    - 微观细节：油脂(crema)厚度、拉花稳定性、杯壁挂渍、温度下降后的风味变化。
    - 场景标签：咖啡馆、外带杯、手冲吧台。

- 红茶（紅茶(こうちゃ) / black tea）
    - 标签：`文化=英式/全球化` `风格=优雅/日常` `功能=下午茶/暖身`
    - 组成要素：茶叶(阿萨姆/锡兰/大吉岭等) + 水温/时间 + 奶/糖/柠檬。
    - 口感结构：单宁涩感 -> 麦芽甜香 -> 奶香融合(奶茶)。
    - 微观细节：茶汤色泽(琥珀/红褐)、茶叶舒展状态、杯沿茶渍环、茶匙轻碰声。
    - 场景标签：英式下午茶、早餐配餐、冬日暖饮。

- 清酒（日本酒(にほんしゅ) / sake / nihonshu）
    - 标签：`文化=日式` `风格=仪式/社交` `功能=佐餐/庆典`
    - 组成要素：米 + 水 + 麴菌发酵 + 精米步合。
    - 口感结构：甘口/辛口 -> 米香 -> 酸度 -> 余韵。
    - 微观细节：酒液透明度、挂杯(脚)速度、冷酒结露、温酒(燗)后的香气释放。
    - 场景标签：居酒屋、神道仪式、新年庆典。

- 葡萄酒（ワイン / wine, English loan）
    - 标签：`文化=欧式-全球化` `风格=优雅/社交` `功能=佐餐/庆典`
    - 组成要素：葡萄品种 + 产区风土 + 酿造工艺 + 陈年。
    - 口感结构：单宁(红酒) -> 酸度 -> 果香 -> 橡木桶味 -> 余韵。
    - 微观细节：酒液色泽(边缘色带)、挂杯(酒泪)、醒酒后的香气展开、瓶塞拔出声。
    - 场景标签：正式晚宴、品酒会、日常佐餐。

- 鸡尾酒（カクテル / cocktail, English loan）
    - 标签：`文化=欧美-全球化` `风格=创意/社交` `功能=夜生活/庆典`
    - 组成要素：基酒(金/伏特加/朗姆/威士忌等) + 辅料 + 装饰。
    - 口感结构：酒精刺激 -> 酸甜平衡 -> 层次叠加 -> 装饰香气。
    - 微观细节：冰块的融化速度、杯壁结霜、装饰物(柠檬皮/樱桃)位置、搅拌/摇晃后的气泡。
    - 场景标签：酒吧吧台、派对、酒店酒廊。

## 学科类

### 计算机科学（Computer Science）

- 代码审查会（Code Review）
    - 标签：`文化=现代通用` `风格=协作工程` `功能=质量控制`
    - 关键动作：提交变更 -> 行级评论 -> 讨论取舍 -> 合并决策。
    - 微观细节：审查批注密度、行号跳转、术语冲突、沉默时长。

- 事故复盘（Postmortem）
    - 标签：`文化=现代通用` `风格=工程治理` `功能=风险收敛`
    - 关键动作：时间线回放 -> 根因拆解 -> 行动项分配 -> 跟踪闭环。
    - 微观细节：时间戳对齐、责任措辞、白板箭头层级、任务回执延迟。

- 灰度发布（Canary Rollout）
    - 标签：`文化=现代通用` `风格=运维实践` `功能=发布控制`
    - 关键动作：小流量放量 -> 指标监控 -> 回滚阈值判断 -> 全量发布。
    - 微观细节：告警抖动、仪表盘峰值、回滚按钮触发、值班交接语句。

### 地理信息科学（Geographic Information Science）

- 外业采集（Field Survey）
    - 标签：`文化=现代通用` `风格=实地测绘` `功能=数据采集`
    - 关键动作：布点 -> 采样 -> 轨迹记录 -> 现场校核。
    - 微观细节：RTK天线架设、控制点喷漆、坐标本湿痕、现场返查。

- 投影转换（CRS Transformation）
    - 标签：`文化=现代通用` `风格=空间计算` `功能=数据一致化`
    - 关键动作：识别坐标系 -> 参数设定 -> 批量转换 -> 误差复核。
    - 微观细节：边界错位、比例尺跳变、图层“漂移感”、误差表红字。

- 栅格与矢量叠加（Raster-Vector Overlay）
    - 标签：`文化=现代通用` `风格=空间分析` `功能=模型支撑`
    - 关键动作：预处理 -> 对齐像元 -> 叠加分析 -> 输出专题图。
    - 微观细节：锯齿边界、色带分级、透明度冲突、图例重排。
