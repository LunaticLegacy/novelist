# 第010章：风从桥上过

第二天一早，菲娜刚进教室，就被一阵比早读还热闹的讨论声迎面砸中。

“你昨晚用命定色盘了吗？我让它给我做了三套复习卡。”

“我作业插图直接AI代画，老师居然没看出来。”

“你那个提示词发我，我也想出‘一看就像我亲手画的’版本。”

桌椅碰撞声、终端提示音、压低又压不住的笑声混在一起，像一锅已经滚开的水。窗外阳光很亮，黑板上昨天下课前留下的公式还没擦干净，粉笔灰沿着边缘泛白。

教室后墙新贴了学院临时通知，红框加粗：

`严禁在教学楼内导入未审核符文模板。`

可这行字像装饰。通知下面照样围着一圈人，讨论“谁的提示词更稳”“哪家插件出图更快”，还有人现场交换压缩包，仿佛只要不在老师眼前点开，就不算违规。

菲娜把书包放下，短杖贴着桌侧立好。她还没坐稳，蕾雅就从隔壁排挤了过来，手机几乎怼到她鼻尖。

“看！我新做的‘考试必胜护符’2.0！”

屏幕上是一张闪着夸张金光的护符图，中间四个大字：`逢考必过`，边缘还缠着一圈会动的小猫尾巴。

菲娜没绷住，笑出声：“这也太花了。”

“花怎么了，灵就行。”蕾雅一脸认真，“我昨天背单词前看了三秒，今天默写全对。超灵的。”

“那是你昨晚真的背了。”

“你不要拆穿心理学奇迹。”

两人正说着，上课铃响。理论课老师抱着一摞教材进门，先把终端接上投影，教室里才一点点安静下来。

第一张投影不是公式，而是一段新闻截屏。

画面里是一间实操教室的角落，防护垫被炸出一块焦黑，旁边几名学生被防护队带离。标题很直白：

`某地学院发生小型爆燃事故`

`初步判定：未经验证AI符文直接用于实操`

老师用指节敲了敲讲台：“今天先不翻书。先讲这个。”

后排有人小声嘀咕：“又是AI背锅……”

老师抬眼扫过去，声音不高，却压得住场子：“技术不背锅，人要背。你看不懂的符文，接进回路，就是把自己的手放进不透明机器里。”

教室静了几秒。

投影切到下一页，对比图并排展开：左侧传统符文，结构清楚、环阶标注完整；右侧AI生成符文，效率高，但隐藏节点多、解释链不完整。

老师把光笔点在右侧结构图一个不起眼的弯折节点上：“看到这个‘折返钩’没有？它在可视层是装饰线，真实回路里却可能是能量分流口。你们不拆解就导入，等于把盲盒塞进杖芯。”

前排几个同学下意识缩了缩脖子。后排还想嘴硬的人也安静了，至少在这十秒里，没人再提“效率第一”。

“记住一句话。”老师在黑板上写下大字，粉笔划出短促刺耳的一声。

`未经审核的符文，不可用于实战。`

菲娜盯着那行字，手指无意识摩挲着魔杖握柄，指腹在防滑纹上来回碾了两下。她脑子里却不是黑板，是昨天蓝月那句几乎一字不差的提醒。

看得懂再用。看不懂，就当没看见。

“菲娜。”

她猛地回神，抬头。

老师正看着她：“你来复述一下，传统符文和AI生成符文在风险控制上的核心差异。”

全班视线瞬间齐刷刷挪过来。

菲娜站起来，先吸了一口气，几乎机械地把课本定义背了出来：“传统符文可追溯、可拆解、可逐级校验；AI生成符文高效但存在黑箱节点，若无审核和回路隔离，实操风险显著上升。”

老师点点头：“坐。”

她坐下后，手心才慢慢出了一层薄汗。桌下终端屏幕亮了一下，停在和蓝月的聊天框。最后一条消息还是早上发来的：

“今天上课专心。别被热闹带着跑。”

菲娜盯了两秒，锁屏，把终端推回书堆底下。

课间一到，教室又沸腾。

蕾雅刚拆开零食袋，诺亚就拎着他的平板和一根标准考杖从后门进来，眼镜后面的眼神亮得很熟悉，那是他“我又做成了个东西”的标志状态。

“都让让。”他把平板往空桌上一放，屏幕上是他自写的界面：

`RuneBridge v0.2`

`输入：命定色盘符文JSON`

`输出：杖体回路注入包（0环/1环）`

“符文桥接器？”蕾雅凑过去念出标题，“你这名字好像科幻片里的非法装置。”

“谢谢夸奖。”诺亚回得很顺，“看演示。”

他点开一个0环风符文样本，界面跑过几行转换日志，随后用连接片贴上杖身导口。

`注入中...`

`完成。`

他抬杖，几乎没做额外准备，直接触发。

一阵稳定的微风从杖尖推出去，吹得讲台边那叠练习纸哗啦啦翻了半页，又轻轻落回去。

周围同时爆出几声“我靠”。

“真能直接导？”

“这不等于省掉一大截记谱步骤？”

“发我，求内测！”

诺亚抬抬下巴，明显很受用：“0环已经稳了，1环我还在调。等我把错误回滚做完，就能小范围放出来。”

说完他又补了一次短演示，把同一段风符文重复注入两轮，第二轮起杖时回路灯条明显抖了一下，但很快被他手动压回去。旁观同学把那一下当成“炫技难度”，反而鼓掌更响。

菲娜看见那一瞬的抖动，眉心微不可见地蹙了一下。

菲娜看着那根还贴着连接片的杖，开口很轻，却很清楚：“这个东西要小心。你知道你注进去的每个节点在干什么吗？”

诺亚看向她：“当然知道大概。”

“大概不够。”菲娜说，“回路不是给‘差不多’用的。”

诺亚笑了一下，不是嘲讽，更像技术派对保守派的本能不耐：“你哥是不是又给你上安全课了？”

菲娜没接话。

诺亚把平板合上一半，语气平平地补了一句：“他太保守了。技术就是用来突破的，不是用来供着看的。”

空气里那点刚才还热腾的兴奋，忽然卡了一下。蕾雅左右看了看，咬着零食没出声。

菲娜沉默两秒，只把自己的短杖往桌里收了收：“突破可以。先确认你不会把教室炸了。”

诺亚耸肩：“等我把1环跑稳，你会改口的。”

他说完就被一群同学围住，消息弹窗和好友申请在他屏幕角落蹭蹭往上跳。

有人当场问“能不能今天就开群测”，还有人把自己杖体型号报出来求适配。诺亚边回边记，语速快得像在直播答疑。菲娜隔着人群看着那块被挤得几乎透不过气的空桌，忽然生出一种很轻却很明确的预感：这东西会比任何校规传得都快。

菲娜坐回座位，没再说话。窗外阳光正好落在她桌角，暖得很亮，她却觉得那光有点晃眼。

同一时间，蓝月在另一端看着实时日志。

他的屏幕上开着三列并行窗口：命定色盘全网调用曲线、暗月会内容扩散图、学院公开信道监控。

刚刚那次课间演示被学生拍成短视频，十五分钟内转了四十七次，标签从“诺亚又整活”一路涨到“符文桥接器内测求资格”。

模型在侧栏给出预测：

`校园内扩散概率：高`

`误用事故窗口：2-5日`

`可干预点：当前`

蓝月没点“干预”，而是把另一个标签拖进人物档案。

`Candidate_Apostle_02：Noah`

`特征：高风险偏好 / 高技术执行 / 低权威服从`

`建议：持续观察，暂不接触`

他指尖在确认键上停了一瞬，最终按下保存。屏幕反光里，他的神情很淡，像在记录一条天气变化，而不是一个可能出事的人。

保存之后，他又把“学院扩散链”单独拉了一列，把参与转发最活跃的账号全部标黄。诺亚的名字在榜首附近，后面紧跟着几个陌生ID，传播节奏过于统一，像被同一只手轻轻推着往前走。

蓝月看了两秒，没继续追。他只是把“观察优先级”上调一档，然后切回菲娜今天的课程表，确认她最后一节已经结束。

放学后，菲娜没等蕾雅，自己背着包沿主路往家走。

晚高峰还没完全开始，路边店铺一盏盏亮灯。她走到十字路口时，头顶广告屏正好切进一段命定色盘的新宣传片。

画面里，年轻魔法师把杖尖点在半空，几秒就生成整套绚烂符阵。旁白语气轻快得像在卖一种无风险的玩具：

“命定色盘，让创作更快一步。”

下一幕，字幕闪出来：

`看不见复杂，只有灵感。`

菲娜在原地站了几秒，仰头看完那句口号，手指慢慢扣紧书包带。等红灯跳绿，她才像被谁从画面里推了一把，低头加快脚步，穿过人行道。

风从街口吹过来，带着一点傍晚的凉意。她走得很快，鞋底踩在地砖上，一下一下，节奏比平时更急。

她没回头看那块屏幕。

她走到下一个路口时，刚好听见两个同校学生擦肩而过。

“你真不怕啊？老师都说了先别碰。”

“我只做低环，能出什么事。再说，大家都在用。”

菲娜脚步短暂顿了半拍，又继续往前。夕光把她影子拉得很长，像一条细细的线，一直拽到看不见的地方。

她在地铁口前停了一秒，把手机掏出来，想给蓝月发“今天有点乱”，打到一半又删掉，最后只发了三个字：

“我回家。”

蓝月回得很快：

“路上慢点。”

她盯着这三个字看了两秒，把手机收进外套口袋，跟着人流下楼。地铁进站风扑上来，吹得她后颈微凉，她忽然意识到：真正让人害怕的不是一次争论，而是所有人都觉得“先试试也没关系”。

而这种“没关系”，往往是事故开始前最常见的一句话。
