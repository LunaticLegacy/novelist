# 第007章 森林里的四分钟余火

闹钟还没响，菲娜先醒了。

她睁眼那一刻，脑子里第一反应不是“今天星期几”，而是“明天考试”。这个念头像钉子一样钉在额头中间，让她瞬间彻底清醒。她翻身下床，拉开窗帘，天色刚刚泛白，街道上几乎没人。

楼下厨房已经有光。菲娜下楼时，蓝月正把两根魔杖并排放进长盒里，一根是她常用铝杖，一根是他的旧杖。看见她下楼，他没说废话，直接把盒子推到她面前。

“今天最后一轮，去森林。”

“吃完早饭就走？”

“嗯，早点打完，下午回家做静态复盘。”

菲娜点头。她把吐司塞进嘴里，一边嚼一边看今天的训练单。蓝月把内容写得极简：`1. 杖感对照；2. 余火持续；3. 连贯切换；4. 禁止越级。`

她指着第四条：“你写了三遍‘禁止越级’。”

“怕你装看不见。”

“我又不是昨天那个差点暴毙的我。”

“你最好是。”

吃完两人直接出门。去森林的路比昨天更冷，晨雾还没散，路边已经有`缘`活动摊在搭棚。有人正把“守护色团建”横幅挂到树上，喇叭里放着平台新剪辑的应援BGM。菲娜路过时看了一眼，脚步没停。

蓝月侧头问她：“想去看看？”

“不看。考完再看。”

“执行力不错。”

“你昨天不是还说我是样本吗？样本要对得起采样费。”

蓝月笑了一声，尾巴尖在身后轻轻晃了晃。

进林后信号明显变弱，周围只剩风穿过树叶的声音。两人走到昨天那片空地，蓝月先做边界检查，再把便携监控架好。菲娜把外套脱下来搭在石头上，先做了三分钟热身，确认呼吸节奏稳定后才握杖。

“第一项，杖感对照。”蓝月报时，“铝杖起手，旧杖校准，最后回铝杖收束。”

菲娜按流程做。第一组火系点燃，铝杖延迟仍在，但她已经能用提前预置把延迟吃掉。换旧杖后，火星起得更整齐，像一串被线拉住的灯。再换回铝杖时，她没有再出现昨天那种“手感断层”，第一下就稳住了边界。

蓝月看着监控波形，低声说：“这组可以。”

菲娜没说话，继续第二组。她现在不想听夸，她只要数据稳定。

第三组结束，蓝月把手机递给她：“可以上命定色盘辅助了。还是那条规则：辅助不替代判断。”

菲娜打开模板页，选择`火系持续控制`。屏幕给出一句提示：`把每一簇火当成单独对象管理，不要贪一键全控。`

她盯着这句看了两秒，点头：“懂。”

“第二项，余火持续。目标四分钟。”

菲娜抬杖，余火起。

第一分钟最容易，火星像一群听话的小鱼在她周围缓慢游动。第二分钟开始，手腕出现酸胀，火星边缘有两次轻微抖动，她立即降功率稳边界。第三分钟最难，呼吸和魔力输入很容易错拍，她只能把注意力缩到最小：只盯最近五簇，不看全局。

“三分二十。”蓝月报时，“稳住。”

菲娜咬牙，掌心全是汗。她感觉自己像在同时托着几十只会乱飞的玻璃球，稍微分神就会碎一地。她听见风声、听见心跳、也听见自己太阳穴里一下一下的鼓点。三分五十时，左侧两簇火突然外飘，她立刻补了一个短促风切，把火带回边界。

“三分五十九……四分整，停。”

蓝月落下计时器。菲娜收杖那一瞬间膝盖发软，直接坐到地上，大口喘气。她仰头看天，天顶被树冠切成一块块碎光，眼前还在发白。

“达标了？”她问。

“达标。”蓝月把水递给她，“四分零二秒。”

菲娜接过水，喝到一半突然笑了。不是轻松的笑，是那种终于把一块石头从胸口搬开的笑。她昨天最怕的就是这项，今天她过了。

蓝月没让她马上起身，而是把回放拖回三分四十到四分零五这段，逐帧给她看：

“这里你左肩先紧了，所以输入抖了一下；这里你用了短风切救场，动作对；这里你想补强，但及时收住了，这个判断最值钱。”

菲娜蹲在屏幕前，一边听一边点头。她发现自己在“撑到四分钟”这件事里，真正起作用的不是蛮力，而是三次关键判断：该降、该补、该收。她把这三词写在掌心，防止自己明天上场忘掉。

“我明天要是被别人的节奏带跑怎么办？”她忽然问。

“那就盯你自己的计时器。”蓝月回答，“考场里没有人能替你呼吸，也没有人能替你稳边界。”

休息十分钟后，蓝月加了第三项：连贯切换。他让她在`火 -> 风 -> 光 -> 风 -> 火`之间低损切换，要求每次切换都不崩边界。菲娜前两轮都卡在光转风，第三轮才把节奏打顺。第四轮她甚至把光束压到了接近课堂三环演示的观感，亮度和聚焦都上了一个台阶。

她自己先被吓了一跳，立刻主动降档：“停，我不上去。”

蓝月点头：“做得对。你现在能摸到门槛，不代表现在该跨门槛。”

菲娜把这句话原样记进手机备忘录。她发现自己今天最像“进步”的地方，不是输出更强，而是能在“还能再上”那一刻自己踩刹车。

中午前最后一项是短口试。蓝月一边收设备一边问她：

“考试如果有人现场拿命定色盘生成符文并炫耀，你怎么办？”

“不跟，不评，不临摹。交给监考。”

“如果你临场抽到最不擅长题型？”

“先拿稳分，再争上限。”

“如果我明天不在考场外呢？”

菲娜抬头看他：“你会在。”

蓝月没接这句，只把长盒扣好：“回家。”

回程路上，林道出口那片空地已经热闹起来。`缘`平台线下团建刚开场，一群人围着移动屏幕抽“守护色盲盒”，抽到稀有配色就欢呼，抽到重复就起哄重来。摊位边有人在卖“考前护符贴纸”，十块钱三张，排队的人还不少。

蕾雅正好也在那边，远远看见菲娜就冲过来：“你们也来团建？！”

“路过。”菲娜说。

“我刚抽到金色限定卡！”蕾雅把手机举到她脸前，“看，`明日必胜·星铃版`！”

菲娜看了一眼，图确实好看，甚至比昨天那批更精细。她点了点头：“好看。你玩归玩，今晚早点睡。”

“知道啦知道啦，你现在像我妈。”

“你妈会背三百问吗？”

蕾雅被噎住，转头去找诺亚告状。诺亚站在另一侧摊位边，手里拿着一张活动传单，神情倒是比前两天平了些。他朝菲娜点了点头，算打招呼，没再提“越级测试”。

菲娜把这点变化记在心里，没多说。她跟蓝月继续往车站走，身后喇叭声越来越远。走到拐角时，蓝月忽然开口：

“今天的你，比昨天更像明天能过的人。”

菲娜没有马上回答。她低头看着自己手心，那里还有握杖留下的浅红印子。几秒后她才说：“我不是为了‘像’，我是为了真过。”

“嗯。”蓝月看着前方，“那就真过。”

上车后菲娜靠着窗，看着外面一排排掠过去的广告屏。屏幕上轮播的是同一套话术：`今晚打卡，明日好运`，`命定色盘v1.1守护模板限时开放`，`缘平台考生专区已上线`。她以前会觉得这些广告只是吵，现在却能清楚分辨它们分别在做什么：哪个在给情绪兜底，哪个在抢停留时长，哪个在诱导“先上车再思考”。

“你在看什么？”蓝月问。

“看他们怎么说服人。”菲娜说，“说服方式比内容还重要。”

“恭喜，你开始从用户视角切到结构视角了。”

“听起来像夸奖。”

“是夸奖。”

下午回家后，菲娜按计划做静态复盘：不再实操，只看回放、背流程、做口头复述。她把“四分钟余火”那段回放看了三遍，把每个险些崩盘的时刻都标出来。晚上九点，她合上本子，给自己下最后一条命令：

`明天不追极限，只打稳定。`

写完她把笔放平，去厨房倒水。蓝月还在终端前盯日志，屏幕上命定色盘和`缘`的热度曲线都在涨，但他看见她走过来时，第一反应仍是把一杯温水递到她手里。

“睡吧。”他说。

“你呢？”

“我再看一小时。”

菲娜想劝，又停住。她知道哥哥不会在这个节点听“早点休息”，就像她也不会在这个节点听“别紧张”。她只能点点头：“那你一小时后必须关机。”

“收到，主人。”

“你别闹。”

蓝月笑了下。菲娜端着水回房，关灯前最后看了一眼放在桌边的两根魔杖。明天考场上她只能带铝杖，但她现在已经不怕这件事了。

躺下后她没有立刻睡，先做了三轮“考场心理预演”：

第一轮，想象自己在等候席听见别人炫耀，心跳加速，然后把注意力拉回呼吸。  
第二轮，想象抽到不擅长题型，先按流程拿稳分，不和别人比花活。  
第三轮，想象监考突然加压提问，按“定义-条件-例外”顺序回答，不抢快。

三轮做完，她把手放在胸口，心跳果然慢下来。她忽然很清楚，明天真正的对手不是诺亚、不是同学、也不是任何平台热度，而是她自己那一瞬间“想证明点什么”的冲动。

她对着天花板轻声说了一句：“稳定第一。”

门外传来很轻的一声敲门。菲娜以为是自己幻听，过了两秒又响了一次。她起身开门，蓝月站在走廊里，手里拿着她明天要带的证件袋和备用发圈。

“差点忘了给你。”蓝月把东西递过来，“准考证、身份证明、备用笔、备用发圈，都在里面。”

菲娜接过证件袋，忽然有点鼻酸：“你怎么什么都记得。”

“因为你明天只需要记一件事：按流程做完。”

她点头，低声“嗯”了一下。蓝月抬手揉了揉她头顶：“晚安，考生。”

“晚安，监护人。”

门关上后，菲娜把证件袋放到枕边最顺手的位置，确认三遍才重新躺下。
她把闹钟再提前了五分钟，给自己留一段安静起步时间。
今晚不再改计划。
只等明天执行。
不犹豫。
不分心。
就这样。
上场。
走。

她躺下，闭眼，脑子里浮出今天计时器跳到“四分整”那一秒。那一秒很短，却足够她撑过明天。
